# Problem 3 (Hard)

**Topic: Poisson Process — Splitting and Merging**

A core router receives packets as a **Poisson process** with rate $\lambda = 10$ packets/second. Each incoming packet is independently forwarded to **Server A** with probability $p = 0.3$, or to **Server B** with probability $1 - p = 0.7$.

**(a)** [Splitting] Show that the stream of packets arriving at Server A and the stream arriving at Server B are each Poisson processes. State the rate of each sub-process.

**(b)** [Interarrival Time after Splitting] Let $T_A$ be the interarrival time between consecutive packets at Server A. State the distribution of $T_A$, compute $E[T_A]$, and find $P(T_A > 0.5)$.

**(c)** [Combined Analysis] Let $N_B(t)$ denote the number of packets arriving at Server B in an interval of length $t = 1$ second. Find $E[N_B(1)]$, $\text{var}(N_B(1))$, and $P(N_B(1) = 0)$.
