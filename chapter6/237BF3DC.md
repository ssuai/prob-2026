# Problem 2. Poisson Process

A website receives login requests according to a Poisson process with rate

$$
\lambda = 2
$$

requests per minute.

Let $N_t$ be the number of login requests during a time interval of length $t$ minutes.

Answer the following questions.

1. Find the probability that exactly 5 login requests arrive during a 3-minute interval.

$$
P(N_3 = 5)
$$

2. Let $T$ be the time until the first login request arrives. Find the probability that the first login request takes more than 1 minute to arrive.

$$
P(T > 1)
$$

Show your work clearly.
