# 3. Hard Problem

A customer service center receives calls according to a Poisson process with rate $\lambda=12$ calls per hour. Each call is independently classified as a technical-support call with probability $0.35$, and as a general-support call with probability $0.65$.

In addition, urgent alert messages arrive independently according to another Poisson process with rate $2$ messages per hour.

Define an urgent event as either a technical-support call or an urgent alert message.

1. Using the splitting property of a Poisson process, find the distributions of the number of technical-support calls and general-support calls in a $30$-minute interval. Then compute the probability that exactly $3$ technical-support calls and exactly $4$ general-support calls arrive in that interval.
2. Using the merging property of independent Poisson processes, find the distribution of the number of urgent events in one hour. Then compute the probability that at least $8$ urgent events occur in one hour.
3. Let $Y_{10}$ be the waiting time until the $10$-th urgent event. What distribution does $Y_{10}$ follow? Find $\mathbb{E}[Y_{10}]$, $\operatorname{Var}(Y_{10})$, and write the PDF of $Y_{10}$.
