# Problem 1 (Easy Level)
## Topic: Bernoulli Merging + Geometric PMF

### Problem Statement
Two separate networks transmit data packets in discrete time slots:
- **Network A** transmits a packet with probability $p_1 = 0.2$.
- **Network B** transmits a packet with probability $p_2 = 0.3$.
- Both networks operate independently of each other.

A central hub records a "combined transfer" if at least one packet is transmitted by either network in a given slot.

### Question
Find the probability that the first combined transfer occurs exactly at the 3rd time slot.
