**Problem Statement:**
A cloud computing server receives two independent streams of data: Standard Tasks (Type A) arrive as a Poisson process with rate $\lambda_A$, and Priority Tasks (Type B) arrive as a Poisson process with rate $\lambda_B$. Independently, the server undergoes a randomized background security sweep, which occurs according to a Poisson process with rate $\mu$.
Let $X$ be the number of Type A tasks processed strictly before the first security sweep occurs.
1. Determine the probability mass function of $X$. 
2. Given that exactly $k$ Type A tasks arrived before the first security sweep, derive the conditional expected time at which the first security sweep occurred. 
3. Interpret the derivation in part 2 using the structural relationship between the minimum of competing exponentials and the identity of the minimum.
