# Problem 3 (Hard)

A smart farm uses two independent monitoring systems every minute:

- System A detects abnormal humidity with probability \(0.08\).
- System B detects abnormal temperature with probability \(0.12\).

An emergency alert is triggered if at least one system detects an abnormal condition during a minute.

Assume both systems follow independent Bernoulli processes.

1. Find the probability that an emergency alert occurs during a given minute.

2. Let \(Y_3\) be the minute at which the 3rd emergency alert occurs.
Find \(P(Y_3 = 8)\).

3. Find the expected number of minutes until the 3rd emergency alert occurs.
