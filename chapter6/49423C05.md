# Problem 2 (Intermediate)

In a smart farm, an AI sensor checks the temperature every hour.
Each check independently detects an overheating event with probability \(p = 0.15\).

Let \(Y_4\) be the hour at which the 4th overheating event is detected.

1. Find the probability that the 4th overheating event is detected exactly at the 10th hour.

2. Explain why your expression follows the Pascal PMF structure.

3. Find the expected value and variance of \(Y_4\).
