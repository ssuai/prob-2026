## Problem

A small data center has **three servers**: Server 1, Server 2, and Server 3.  
Each server independently processes jobs, and the time (in hours) until a server fails follows an **exponential distribution**:

- Server 1: failure rate **λ₁ = 1** (mean lifetime = 1 hour)
- Server 2: failure rate **λ₂ = 2** (mean lifetime = 0.5 hours)
- Server 3: failure rate **λ₃ = 3** (mean lifetime = 1/3 hours)

The data center continues operating as long as **at least one server** is running (i.e., it shuts down only when the **last** server fails).

**(a)** Let $T_{\min}$ be the time until the **first** server fails. Find the distribution of $T_{\min}$ (type and parameter) and compute $E[T_{\min}]$.

**(b)** What is the probability that **Server 1 is the first to fail**?  

> *Hint: Think about which server "wins" in the race to fail first.*

**(c)** Suppose instead that all three servers have the **same failure rate λ = 2**, and the data center shuts down only after **all 6 servers** (in a larger system with 6 i.i.d. servers) have failed.  
Let $Y$ be the total time until all 6 servers have failed.  
Use the **Central Limit Theorem** to approximate $P(Y > 4)$.

> *Hint: $Y$ is the sum of 6 i.i.d. Exponential(λ=2) random variables.*  
> *Standard normal table: $\Phi(1.63) \approx 0.9484$, $\Phi(1.00) \approx 0.8413$, $\Phi(0.82) \approx 0.7939$*
