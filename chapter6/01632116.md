# Hard Problem

Requests arrive at a server according to a Poisson process with rate
λ = 12 requests per hour.

Each request is independently classified as:
- High priority with probability 0.25
- Low priority with probability 0.75

Two backup batteries have independent exponential lifetimes:
- Battery A rate: λ_A = 0.5
- Battery B rate: λ_B = 1.0

1. Find the probability that exactly 2 high-priority requests arrive in one hour.
2. Let T be the time until the first battery failure.
   Find the distribution of T.
