Problem 3.
Three independent accelerator sensors ($A, B$, and $C$) are operating inside a vacuum chamber. The lifetime of each sensor is independent and identically distributed (i.i.d.) following an Exponential distribution with parameter $\lambda = 0.5$ per year. The system experiences degraded resilience each time a sensor fails, and completely shuts down the moment the last remaining sensor fails. Find the expected total residual time from the instant the first sensor fails until the entire system shuts down ($\mathbb{E}[T_{\text{residual}}]$).
