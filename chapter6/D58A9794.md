### 문제 3 (난이도: 어려움)

**주제: 포아송 과정의 병합 및 Erlang 분포의 기댓값**

**상황:** 데이터 센터에 독립적으로 작동하는 두 대의 메인 프레임 서버 A와 B가 있습니다. 서버 A에 발생하는 오류는 월 평균 0.2회($\lambda_a = 0.2$/월)인 포아송 과정을 따르고, 서버 B에 발생하는 오류는 월 평균 0.3회($\lambda_b = 0.3$/월)인 포아송 과정을 따릅니다.

**질문:**

1.  두 서버 중 어느 서버에서든 **처음으로** 오류가 발생할 때까지 걸리는 시간의 기댓값(개월 수)을 구하세요.
    
2.  두 서버를 합친 전체 시스템에서 **3번째** 오류가 발생할 때까지 걸리는 시간의 기댓값을 구하고, 이 시간이 따르는 확률 분포의 명칭을 쓰세요.
    

**해설:**

-   **질문 1의 답:** 독립적인 두 포아송 과정을 병합(Merging)한 전체 과정은 비율이 두 비율의 합인 포아송 과정을 따릅니다. 따라서 병합된 오류 발생률 $\lambda = \lambda_a + \lambda_b = 0.2 + 0.3 = 0.5$/월 입니다. 첫 오류까지의 시간은 비율이 0.5인 지수분포를 따르므로 , 기댓값 $E[T] = 1/\lambda = 1/0.5 = 2$개월 입니다.
    
-   **질문 2의 답:** 포아송 과정에서 $k$번째 도착(여기서는 3번째 오류)까지 걸리는 시간 $Y_k$는 차수(order) $k$인 Erlang 분포(Erlang PDF)를 따릅니다. $k=3$이고 $\lambda=0.5$일 때, Erlang 분포의 기댓값 공식은 $E[Y_k] = k/\lambda$ 이므로, $E[Y_3] = 3 / 0.5 = 6$개월 입니다. (정답: 6개월, Erlang 분포)
