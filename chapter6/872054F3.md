# Hard Problem: Pascal Distribution and Order of Success

## Scenario
A specialized machine produces components independently.

Each component is defective with probability

\[
p=0.1
\]

The machine is shut down for maintenance immediately when the 4th defective component is produced.

## Questions

1. What is the probability that the machine is shut down on exactly the 20th component produced?

2. What is the expected number of components produced before the machine is shut down?
