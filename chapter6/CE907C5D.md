# Problem 2 (Intermediate Level): Packet Processing in a Dual-Server Network

A network server handles two independent packet streams: priority packets and background packets.
* **Priority packets** arrive according to a Poisson process with a rate of $\lambda_1 = 3$ packets per second.
* **Background packets** arrive according to an independent Poisson process with a rate of $\lambda_2 = 5$ packets per second.

1. Find the probability that the server receives absolutely no packets (0 arrivals of any kind) during a specific 0.5-second window.
2. Suppose a security filter is introduced that independently and randomly discards (drops) certain background packets. If a background packet successfully passes through the filter with probability $q = 0.80$ (and is dropped with probability $1-q = 0.20$), determine the type of probability distribution and the corresponding parameter rate for the stream of background packets that successfully pass through the filter.
