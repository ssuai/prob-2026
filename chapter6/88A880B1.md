# Problem 1. Geometric Random Variable

A server checks incoming requests once every second.

At each second, a high-priority request arrives with probability $p = 0.2$.

Assume that arrivals in different seconds are independent.

Let $T$ be the number of seconds up to and including the first high-priority request.

Find the probability that the first high-priority request arrives at the 4th second.

That is, find

$$
P(T = 4).
$$

Show your work clearly.
