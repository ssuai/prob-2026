# Problem 2 (Intermediate)

**Topic: Poisson Process — Poisson Arrivals & Exponential Interarrival Time**

A web server receives HTTP requests modeled as a **Poisson process** with rate $\lambda = 3$ requests per second.

Recall that the number of arrivals $N(\tau)$ in an interval of length $\tau$ follows:

$$p_{N(\tau)}(k) = \frac{(\lambda\tau)^k e^{-\lambda\tau}}{k!}, \quad k = 0, 1, 2, \ldots$$

**(a)** [Arrivals in an Interval] Compute the probability of receiving exactly $k = 0, 1, 4$ requests in a 2-second window.

**(b)** [Mean & Variance] State $E[N(2)]$ and $\text{var}(N(2))$ for the 2-second window.

**(c)** [Interarrival Time] Let $T$ be the time (in seconds) between consecutive requests. State the distribution of $T$ and write its PDF $f_T(t)$.

**(d)** [Tail Probability] Compute the probability that the server waits more than $\dfrac{1}{3}$ second for the next request:

$$P\!\left(T > \frac{1}{3}\right)$$
