# Problem 1 (Easy)

**Topic: Bernoulli Process — Binomial Arrivals \& Geometric Interarrival Time**

A network switch monitors incoming packets every millisecond. Each 1 ms slot, a packet arrives independently with probability $p = 0.2$. This defines a **Bernoulli process** with parameter $p$.

**(a)** \[Number of Arrivals] Let $N$ be the number of packets that arrive in $n = 10$ slots. State the distribution of $N$, write its PMF, and compute $P(N = 0)$ and $P(N = 2)$.

**(b)** \[Mean \& Variance] Compute $E\[N]$ and $\\text{var}(N)$ using the properties of the Binomial distribution.
