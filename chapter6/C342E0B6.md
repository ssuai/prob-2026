## Problem 1 (Easy)
A vending machine is checked at the end of each 5-minute slot. In each slot, an item gets stuck with the same probability, independently of all previous slots.

- Let $X_i=1$ if an item gets stuck during slot $i$, and $X_i=0$ otherwise.
- The sequence $X_1,X_2,\ldots$ is a **Bernoulli process** with parameter **$p=0.12$**.
- Let $S$ be the number of stuck-item events during the first **10** slots.

Calculate the **probability that exactly 2 stuck-item events occur** during the first 10 slots.
