### [Hard Level - INTEGRATED CONCEPTS] Question 3: Distributed Load Balancer (30 Points)

User requests arrive at a cloud network's primary load balancer according to a Poisson process with a total rate of $\lambda = 100$ requests per second. To distribute the workload, the load balancer randomly routes 40% ($p = 0.4$) of the incoming requests to Server A and the remaining 60% to Server B.

**(3-a)** Identify the specific statistical arrival processes for Server A and Server B, and state their exact arrival rates ($\lambda_A$ and $\lambda_B$). Then, calculate the exact joint probability that Server A receives exactly 0 requests AND Server B receives exactly 0 requests in the next $0.1$ seconds.

**(3-b)** Let $Y_{400}$ be the total time (in seconds) it takes for Server A to receive exactly 400 requests. Calculate the exact expected value, $E[Y_{400}]$, and the exact variance, $\text{Var}(Y_{400})$. Then, using the Central Limit Theorem (CLT), approximate the probability that it takes more than 11 seconds for Server A to receive 400 requests, $P(Y_{400} > 11)$. Express your final answer strictly in terms of the standard normal Cumulative Distribution Function (CDF), $\Phi(z)$.

**(3-c)** Suppose the entire system is freshly restarted. Calculate the expected time (in seconds) until *either* Server A *or* Server B receives its very first request.
