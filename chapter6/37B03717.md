## Problem 3 (Hard)
Two independent request streams reach a campus IT help desk during the morning shift. Some requests are escalated to a senior technician.

- Hardware requests arrive according to a **Poisson process** with rate **$\lambda_H=5$ per hour**.
- Software requests arrive according to an independent **Poisson process** with rate **$\lambda_S=7$ per hour**.
- Each hardware request is escalated independently with probability **$0.30$**.
- Each software request is escalated independently with probability **$0.20$**.
- Let $Z$ be the time, in hours after opening, at which the **3rd escalated request** of either type arrives.

Determine the **probability that $Z \le 0.5$ and the 3rd escalated request is a software request**.
