# Easy Problem: Memorylessness in a Bernoulli Process

A website checks every minute whether a new user arrives.

For each minute, a new user arrives with probability

\[
p = 0.2
\]

independently of all other minutes.

Let

\[
T = \text{the time of the first user arrival}
\]

where \(T\) is counted in minutes.

Suppose that no user arrives during the first 5 minutes.

## Questions

**(a)** What is the probability that the first user arrives exactly at minute 8?

**(b)** Given that no user arrives during the first 5 minutes, what is the probability that the first user arrives 3 minutes later?

**(c)** Explain why the answer in part (b) has the same form as a geometric random variable.
