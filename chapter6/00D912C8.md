# Problem 1 — Easy | Bernoulli Process: Geometric & Binomial

## Problem

A Bernoulli process has success probability p = 0.4 per trial.

**(1)** Let T = number of trials up to and including the first success.
Find P(T = 3) and E[T].

**(2)** Let S = number of successes in 5 independent trials.
Find P(S = 2) and E[S].

---

## Solution

### Part (1) — Geometric

T ~ Geometric(p = 0.4)

PMF: P(T = t) = (1-p)^(t-1) * p

P(T = 3) = (0.6)^2 * 0.4 = 0.36 * 0.4 = **0.144**

E[T] = 1/p = 1/0.4 = **2.5**

### Part (2) — Binomial

S ~ Binomial(n = 5, p = 0.4)

PMF: P(S = k) = C(n,k) * p^k * (1-p)^(n-k)

P(S = 2) = C(5,2) * (0.4)^2 * (0.6)^3
         = 10 * 0.16 * 0.216
         = **0.3456**

E[S] = np = 5 * 0.4 = **2.0**
