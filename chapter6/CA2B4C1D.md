# Problem 2 (Intermediate)

Two machines, Machine A and Machine B, operate independently. The time until failure for Machine A is exponentially distributed with a rate of $\lambda_A = 0.2$ per year, and for Machine B with a rate of $\lambda_B = 0.3$ per year. Let $Z$ be the time until the first machine fails. 

Find the Cumulative Distribution Function (CDF) of $Z$ and state its distribution type.
