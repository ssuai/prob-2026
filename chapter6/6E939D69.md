# Problem 1 (Easy)

A small online game gives a rare reward independently after each match with probability 0.2.

Let \(T_1\) be the number of matches until the first rare reward appears, and let \(T_2\) be the number of additional matches needed to obtain the second rare reward after the first one.

Assume the game follows a Bernoulli process.

1. Find the probability that \(T_1 = 3\).

2. Find the probability that \(T_2 = 2\).

3. Find the probability that the second rare reward appears exactly on the 5th match.

4. Find the expected number of matches required to obtain the second rare reward.
