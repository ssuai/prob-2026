# Problem 2: Cross-Platform Server Matchmaking

**Problem Statement:**
A multiplayer game server receives matchmaking requests from two independent platforms: PC and Console.
* Requests from PC players arrive according to a Poisson process with a rate of $\lambda_1 = 4$ requests per minute.
* Requests from Console players arrive according to an independent Poisson process with a rate of $\lambda_2 = 6$ requests per minute.

**Your Task:**
1. **Part A:** Find the probability that the server receives exactly $8$ total matchmaking requests within a $1$-minute window.
2. **Part B:** Given that a single matchmaking request has just arrived at the server, find the conditional probability that it came from a PC player.
