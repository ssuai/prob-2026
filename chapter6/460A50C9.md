2. intermediate

A campus security system detects suspicious motion events according to a Poisson process with rate 0.4 events per minute.

1. Let N(10) be the number of suspicious motion events detected in 10 minutes. Find P(N(10) = 3).
2. Find the probability that no suspicious motion event occurs during the first 5 minutes.
3. Let T be the waiting time until the first suspicious motion event occurs. Find P(T > 5).
4. Find E[N(10)] and E[T].
