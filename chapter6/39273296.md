# 1. Easy Problem

A website receives a visitor request in each one-minute time slot with probability $p=0.3$. Assume that requests in different time slots are independent.

Let $S$ be the number of one-minute slots with at least one visitor request during the next $10$ minutes.

1. What distribution does $S$ follow?
2. Find $P(S=4)$.
3. Find $E[S]$ and $\operatorname{Var}(S)$.
