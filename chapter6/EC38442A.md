Intermediate: Server Load BalancingA primary server receives incoming web requests according to a Poisson process at a rate of $\lambda = 20$ requests per minute. A load balancer immediately evaluates each request and routes it to Server A with a probability of $p = 0.3$ or to Server B with a probability of $1 - p = 0.7$. The routing decisions are completely independent for each request.Question: What is the probability that in a specific 3-minute interval, Server A receives exactly 15 requests and Server B receives exactly 40 requests?Key to Solving:Apply the Poisson splitting property from Chapter 2. First, determine the new rates for the split processes ($\lambda_A$ and $\lambda_B$). Next, calculate the expected number of arrivals for each server over the 3-minute interval. Finally, leverage the crucial property that split Poisson processes are independent of each other, allowing you to multiply their individual Poisson probabilities together.
