# Problem 1 — Easy

## Topic: Bernoulli Process and Binomial Random Variable

A website records whether a visitor clicks an advertisement during each minute.

Assume that each minute is an independent Bernoulli trial, and the probability that a visitor clicks the advertisement in a given minute is

\[
p=0.20.
\]

Let \(S\) be the number of advertisement clicks during 10 minutes.

Answer the following questions.

1. What distribution does \(S\) follow?
2. Find \(P(S=3)\).
3. Find \(E[S]\).
4. Find \(Var(S)\).
