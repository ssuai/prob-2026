# Intermediate Problem: Time to the kth Arrival

A delivery company monitors its automated sorting machine every 5 minutes.

During each 5-minute interval, there is a probability of \(p=0.12\) that the machine produces a serious sorting error. Assume that errors in different intervals occur independently.

The machine is programmed to automatically shut down when the **4th serious sorting error** occurs. However, even if fewer than 4 serious errors occur, the machine is manually stopped after **20 intervals** for routine inspection.

Let

\[
Y_4 = 	ext{the interval number in which the 4th serious sorting error occurs}
\]

and let

\[
Z = \min\{Y_4, 20\}
\]

where \(Z\) represents the interval number when the machine stops.

## Questions

**(a)** Find the probability that the machine automatically shuts down exactly at interval 12.

**(b)** Find the probability that the machine automatically shuts down before the routine inspection.

**(c)** Find the probability that the machine stops at interval 20, and explain why this probability is not simply \(P(Y_4=20)\).
