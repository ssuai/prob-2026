\## \[Easy] Topic: Discrete MAP Inference



The prior probability that a critical bug ($\\Theta$) exists in a software program is $P(\\Theta = 1) = 0.1$, and the prior probability that it does not exist is $P(\\Theta = 0) = 0.9$. A test program designed to detect this bug has a detection probability (true positive rate) of $0.95$ when the bug is present ($X=1$). However, it has a false alarm rate of $0.05$ when the bug is not present.



If the test program detects a bug ($X=1$), determine whether the bug exists ($\\hat{\\theta}$) using the \*\*Maximum A Posteriori (MAP) decision rule\*\*.
