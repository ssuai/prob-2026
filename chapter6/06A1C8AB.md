# [Hard] Problem 3: Delivery App's Routing System & Overload Alarm

## 1. Problem Scenario
On a Friday evening (18:00–20:00), incoming order requests to a food delivery platform server in the Gangnam area follow a Poisson Process with an average rate of lambda = 50 orders per second.

The system automatically routes each order: each request is independently classified as an "Express Delivery" with probability p = 0.6, or as a "Standard Combined Delivery" with probability 1 - p = 0.4, splitting the traffic into two separate queue servers.

A system engineer decides to monitor the system performance during an arbitrarily selected 10-second interval (tau = 10).

## 2. Questions

### (a)
Let NA be the number of "Express Delivery" orders and NB be the number of "Standard Combined Delivery" orders received during this 10-second interval.

Find the Joint PMF

p_NA,NB(k,m)

and mathematically/logically argue the relationship (independence) between the two random variables based on the properties of Poisson Splitting.

### (b)
The engineer has set an automated overload alarm to ring if the number of "Express Delivery" orders (NA) during this 10-second window is 320 or more.

Find the probability that the overload alarm rings,

P(NA >= 320)

by applying the Central Limit Theorem (CLT) for a normal approximation.

Use the standard normal CDF values:

- Phi(1.15) ≈ 0.8749
- Phi(1.16) ≈ 0.8770
