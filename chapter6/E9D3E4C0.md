# **Problem 1: Distributed Caching Synchronization - Easy**



###### **\[Condition]**

###### You are currently engaged in a peer-review session examining a newly developed Java-based distributed caching service written by your classmate, Kwon Min-young. The software is being tested in the Soongsil University Information Science Center. The server cluster processes incoming memory synchronization chunks in strictly rigid, discrete time slots. Based on the current network degradation metrics, the probability of a successful cache hit in any given discrete network slot is exactly $p = 0.5$. Min-young claims during the review that because the probability is fifty percent, observing a sequence of exactly 4 network slots guarantees that the system will achieve exactly 2 successful hits. You must calculate the actual mathematical probability to verify or refute her architectural logic.

###### **\[Problem]**

###### Assuming the outcome of each discrete time slot represents an independent Bernoulli trial, what is the exact probability of observing exactly 2 successful cache hits across $n = 4$ discrete time slots?
