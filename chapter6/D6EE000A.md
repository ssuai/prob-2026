# [Intermediate] Problem 2: Course Registration Server Delay

## 1. Problem Scenario
Jiwoo is sitting in front of her computer to register for "Information Security", the most popular course of this semester.

Immediately after the registration opens, connection requests flood into the school server randomly. These incoming requests follow a Poisson Process with an average arrival rate of lambda = 2 requests per minute.

Due to the server's traffic control system, at the exact instant the 4th request hits the system, Jiwoo's screen will freeze and trigger a queue delay message.

## 2. Questions

### (a)
Let Y4 be the continuous time index at which the 4th request arrives (the time when Jiwoo's screen freezes).

Provide the exact Probability Density Function (PDF)

f_Y4(y)

for this random variable.

### (b)
Find the probability that Jiwoo's screen safely survives without freezing for the first 3 minutes after the registration opens, by mapping this event to a Poisson PMF.
