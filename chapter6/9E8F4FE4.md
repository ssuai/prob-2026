3. hard

At a large online ticketing platform, two independent types of requests arrive.

Regular ticket requests arrive according to a Poisson process with rate 3 requests per minute.
VIP ticket requests arrive according to an independent Poisson process with rate 2 requests per minute.

The platform merges both types of requests into one request queue.

Also, each incoming request is independently classified as high-priority with probability 0.3.

1. What is the rate of the merged request process?
2. Find the probability that exactly 8 total requests arrive in 2 minutes.
3. Find the rate of the high-priority request process.
4. Let Y5 be the time until the 5th high-priority request arrives. Find E[Y5] and Var(Y5).
5. Find the probability that the 5th high-priority request arrives after 4 minutes.
