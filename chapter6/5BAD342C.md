# Problem 1 (Easy)

Consider a discrete-time Bernoulli process where a server attempts to authenticate user logins. In each 1-second time slot, a successful login occurs independently with a probability of $p = 0.2$. Let $T$ be the time (in seconds) until the first successful login occurs.

**Question:** Find the expected value $\mathbb{E}[T]$ and the variance $var(T)$ of the random variable $T$. Then, calculate the exact probability that the first successful login occurs exactly at the 4th second.
