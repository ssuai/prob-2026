## Problem 2: Sample Size and Empirical Frequencies (Intermediate)

A software system undergoes repeated independent testing cycles. Let $A$ be the event that a critical memory leak occurs during a single testing cycle, with a true probability $p = P(A) = 0.20$. 

To monitor the system, engineers plan to run $n$ independent trials and measure the empirical frequency of the memory leak, denoted by:
$$M_n = \frac{X_1 + X_2 + \dots + X_n}{n}$$
where $X_i = 1$ if the event occurs in the $i$-th trial, and $0$ otherwise.

Using the Chebyshev inequality approach underlying the Weak Law of Large Numbers (WLLN), determine the minimum number of trials $n$ required so that the empirical frequency $M_n$ stays within $\epsilon = 0.04$ of the true probability $p$ with a high confidence probability of at least $0.95$. That is, satisfy:
$$P(|M_n - 0.20| < 0.04) \ge 0.95$$
