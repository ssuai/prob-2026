Topic(s): Chebyshev Inequality, Weak Law of Large Numbers, Sample Mean, Required Sample Size

Problem:

A communication system sends binary signals.
2. intermediate

For each transmission, define Xi as follows:

Xi = 1 if an error occurs in the i-th transmission.
Xi = 0 otherwise.

Assume that X1, X2, ..., Xn are i.i.d. Bernoulli random variables with

P(Xi = 1) = p = 0.04.

Let Mn be the empirical error rate:

Mn = (X1 + X2 + ... + Xn) / n.

Answer the following questions.

1. Find E[Xi] and Var(Xi).

2. Find E[Mn] and Var(Mn).

3. Use Chebyshev inequality to find an upper bound for

P(|Mn - 0.04| >= 0.01)

when n = 1000.

4. Find the minimum value of n such that

P(|Mn - 0.04| >= 0.01) <= 0.05.

5. Explain how this result supports the Weak Law of Large Numbers.
