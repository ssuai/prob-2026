## Problem 1 (Easy)
A server room monitors the amount of backup power used during one short outage. Let $X$ be the number of battery units used.
- The random variable $X$ is **nonnegative**.
- The expected value is $E[X]=3.2$.
- Use **Markov's inequality**: $P(X\ge a)\le \frac{E[X]}{a}$ for $a>0$.

Calculate the **Markov upper bound** for $P(X\ge 8)$.
