# Problem 2 (Intermediate)

The `insert` operation in a `VArray` has a high variability in execution time because it requires shifting existing elements. Let $Y$ be the execution time of the `insert` operation, with a mean of $50\mu s$ and a standard deviation of $5\mu s$.

To ensure system stability, you want to set a system threshold $c$ such that the probability of the execution time $Y$ deviating from the mean by $c\mu s$ or more is at most $4\%$. Find the minimum value of $c$ that satisfies this condition using Chebyshev's Inequality.
