# Problem 1 (Easy)

A factory produces items whose weight X (in kg) is a non-negative random variable
with E[X] = 6 kg.

(a) Using the Markov inequality, find an upper bound for P(X ≥ 18).

(b) It is additionally known that var(X) = 4.
    Using the Chebyshev inequality, find an upper bound for P(|X - 6| ≥ 4).

(c) Compare the two bounds from (a) and (b).
    Rewrite the event {X ≥ 18} in the form {|X - 6| ≥ c} for some c,
    and apply the Chebyshev inequality to that same event.
    Which bound is tighter?
