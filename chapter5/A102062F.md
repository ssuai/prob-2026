# Problem 1 — Easy

## Topic
Markov Inequality

## Problem
Let \(X\) be a nonnegative random variable representing the daily number of defective products found in a small factory. Suppose that

\[
E[X] = 3.
\]

Using Markov inequality, find an upper bound for each probability below.

1. \(P(X \ge 6)\)
2. \(P(X \ge 10)\)
3. \(P(X \ge 15)\)

Then briefly explain why the upper bound becomes smaller as the threshold increases.
