# Problem 3 (Hard): CLT + MGF + Real-World Probability Approximation

## Problem Statement

A server processes incoming requests. The processing time for each request (in seconds) is an i.i.d. random variable $X_i$ with mean $\mu = 2$ seconds and variance $\sigma^2 = 1$ second².

The server must process $n = 144$ requests in a session.

Let $S_n = X_1 + X_2 + \cdots + X_n$ be the total processing time.

**(a)** Compute $E[S_n]$ and $\text{var}(S_n)$.

**(b)** Define the standardized sum:
$$Z_n = \frac{S_n - n\mu}{\sigma\sqrt{n}}.$$
Show that $E[Z_n] = 0$ and $\text{var}(Z_n) = 1$.

**(c)** Suppose the session has a time limit of **300 seconds**. Using the **Central Limit Theorem**, approximate
$$P(S_{144} > 300).$$
Express your answer in terms of the standard normal CDF $\Phi(\cdot)$, then evaluate numerically using $\Phi(1) \approx 0.8413$.

**(d)** The moment generating function of $X_i$ (after centering and scaling, $\tilde{X} = \frac{X_i - \mu}{\sigma}$) satisfies the Taylor expansion:
$$M_{\tilde{X}}(s) = 1 + \frac{1}{2}s^2 + o(s^2).$$
Using this, explain **in your own words** why the MGF of $Z_n$ converges to $e^{s^2/2}$ as $n \to \infty$, and state what this implies about the distribution of $Z_n$.

**(e)** Now suppose the time limit is reduced to **T seconds**, and the manager wants:
$$P(S_{144} > T) \leq 0.10.$$
Find the minimum value of $T$ using the CLT approximation. Use $\Phi^{-1}(0.90) \approx 1.28$.
