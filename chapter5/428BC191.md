Hard: Data Center Bandwidth Design (CLT - Sum of Random Variables)Focus: Using the CLT for the sum of random variables to design a threshold.Problem:A server in a data center is simultaneously handling $n = 400$ independent client connections. The bandwidth used by each individual connection has a mean of 2 Mbps and a standard deviation of 0.5 Mbps.Let $C\text{ Mbps}$ be the maximum total bandwidth capacity the server can reliably handle. Using the Central Limit Theorem, find the minimum required capacity $C$ so that the probability of the total bandwidth demand exceeding $C$ is exactly 2.5% (or less).(For your calculation, assume that for the standard normal distribution, $P(Z \le 1.96) \approx 0.975$.)
