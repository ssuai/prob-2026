# Intermediate Problem 

A company monitors the daily data usage \(X\) (in GB) of users on a cloud storage service. Assume that \(X \ge 0\), and suppose the following information is known:

\[
E[X] = 20, \qquad \mathrm{Var}(X) = 25
\]

1. Use the **Markov inequality** to find an upper bound for

\[
P(X \ge 50)
\]

2. Use the **Chebyshev inequality** to find an upper bound for

\[
P(|X-20| \ge 30)
\]

3. Compare the two bounds obtained above and explain which inequality provides a tighter estimate in this situation.
