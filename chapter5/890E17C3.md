## 문제 3: 약대수의 법칙과 부등식, 중심극한정리의 비교 (난이도: 어려움)

**문제:** 개발 중인 게임 서버에서 클라이언트의 핑(Ping) 요청을 처리하는 데 걸리는 응답 시간을 확률변수 $X$라고 합시다. 각 요청의 응답 시간은 평균 $\mu = 50\text{ms}$, 분산 $\sigma^2 = 100\text{ms}^2$인 임의의 분포를 따르며, 모든 요청은 서로 독립적이고 동일한 분포를 따릅니다(i.i.d.).

서버가 총 400개의 요청을 처리했을 때, 표본평균(Sample mean)인 $M_{400} = \frac{X_1 + X_2 + \dots + X_{400}}{400}$이 49ms에서 51ms 사이일 확률을 다음 두 가지 방법으로 구하고 결괏값을 비교해 보세요. 1) 체비쇼프 부등식(Chebyshev inequality)을 사용하여 이 확률의 하한선(최솟값)을 구하세요. 2) 중심극한정리(CLT)를 사용하여 이 확률을 근사적으로 구하세요. _(단, 표준정규분포표에서 $\Phi(2.0) \approx 0.9772$로 계산합니다.)_

표본평균 $M_{400}$의 기댓값은 원래 분포의 평균과 같은 $E[M_{400}] = \mu = 50$이며 , 표본평균의 분산은 $var(M_{400}) = \frac{\sigma^2}{n} = \frac{100}{400} = 0.25$가 됩니다.

**1) 체비쇼프 부등식 활용** 구하고자 하는 범위는 표본평균이 평균 50을 기준으로 $\pm 1$ 이내에 있을 확률, 즉 $P(|M_{400} - 50| < 1)$입니다. 체비쇼프 부등식은 변수가 특정 범위를 벗어날 확률의 상한선을 제공합니다.

$$P(|X - E[X]| \ge c) \le \frac{var(X)}{c^2}$$

확률변수 자리에 표본평균 $M_{400}$을 대입하고 거리 $c=1$을 적용합니다.

$$P(|M_{400} - 50| \ge 1) \le \frac{0.25}{1^2} = 0.25$$

범위 내에 존재할 확률은 이 여사건의 확률을 전체 1에서 빼서 구하므로 $1 - 0.25 = 0.75$가 됩니다. 즉, 체비쇼프 부등식에 따르면 분포의 형태와 무관하게 최소 75%의 확률로 표본평균이 49ms ~ 51ms 구간에 위치합니다.

**2) 중심극한정리(CLT) 활용** 중심극한정리에 의해 표본 크기 $n=400$은 충분히 크므로 표본평균 $M_n$의 분포는 정규분포에 매우 가까워집니다. 따라서 $M_{400}$은 근사적으로 평균이 50이고 분산이 0.25(표준편차가 $\sqrt{0.25} = 0.5$)인 정규분포를 따릅니다. 이를 표준화 변수 $Z$로 변환합니다.

$$P(49 < M_{400} < 51) = P\left(\frac{49 - 50}{0.5} < Z < \frac{51 - 50}{0.5}\right) = P(-2 < Z < 2)$$

대칭성에 의해 $P(-2 < Z < 2) = \Phi(2) - \Phi(-2) = \Phi(2) - (1 - \Phi(2))$ 입니다.

주어진 표준정규분포 조건 값인 $\Phi(2.0) \approx 0.9772$를 대입합니다.

$$0.9772 - (1 - 0.9772) = 0.9772 - 0.0228 = 0.9544$$

따라서 중심극한정리를 통해 예측한 근사 확률은 약 95.44%입니다.

**비교 결론:** 체비쇼프 부등식은 확률분포의 구체적인 형태를 모를 때 최악의 상황까지 고려하므로 75%라는 다소 보수적이고 느슨한 최솟값을 제공합니다. 반면, 표본의 크기가 충분히 커서 약대수의 법칙과 중심극한정리가 신뢰성 있게 작동할 때는 정규분포 특성을 활용해 95.44%라는 훨씬 더 정밀하고 높은 확신의 확률값을 도출할 수 있습니다.
