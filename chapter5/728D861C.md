## Problem 3 

A communication system transmits $n$ bits. Each bit is independently corrupted with probability $p = 0.1$. Let $S_n$ denote the total number of corrupted bits.

**(a)** Find $E[S_n]$ and $\text{var}(S_n)$.

**(b)** For $n = 1000$, find upper bounds on $P(S_{1000} \geq 150)$ using both the Markov inequality and the Chebyshev inequality. Compare the two results.

**(c)** For error correction to function properly, we require $P(S_n \geq k) < 0.01$. Using the Chebyshev inequality, find the condition on $k$ as a function of $n$ that guarantees this requirement.
