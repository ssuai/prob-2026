### [Hard Level - INTEGRATED CONCEPTS] Question 3: AI Model Concurrent Inference (30 Points)

A cloud server processes $n = 100$ independent AI inference tasks. Processing time $X_i$ (in seconds) for each task follows an Exponential distribution with $\lambda = 0.5$ ($\mu = 2$, $\sigma^2 = 4$). 

**(3-a)** Let $S_{100} = \sum_{i=1}^{100} X_i$. Calculate the exact expected value $E[S_{100}]$ and the exact variance $\text{Var}(S_{100})$.

**(3-b)** The cloud provider calculates a "Complexity Fee" $C = 5 \cdot S_{100}$. A "Performance Alert" is triggered if the cost $C > 1250$ units. Using the Central Limit Theorem (CLT), approximate the probability of triggering this alert, $P(C > 1250)$.

**(3-c)** Now consider a scenario where the server's mode is unknown. With probability $0.4$, the server operates in Mode A ($\mu_A=2, \sigma_A^2=4$), and with probability $0.6$ in Mode B ($\mu_B=3, \sigma_B^2=9$). 
1. Calculate the *total (unconditional) mean and variance* for a single task $X_i$. 
2. Using these updated parameters, re-approximate the probability that the total time $S_{100}$ exceeds 260 seconds using the CLT. Express your answer strictly in terms of the standard normal CDF, $\Phi(z)$.
