# Easy Problem

A nonnegative random variable $X$ represents the daily amount of data traffic, measured in gigabytes. Suppose:

$$E[X] = 6$$

1. Use Markov's inequality to find an upper bound for $P(X \ge 18)$.
2. Now suppose additionally that $E[X] = 6$ and $\operatorname{Var}(X) = 9$. Use Chebyshev's inequality to find an upper bound for:

$$P(|X - 6| \ge 6)$$
