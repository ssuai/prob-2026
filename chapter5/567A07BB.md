# Problem 1 (Easy) — Markov and Chebyshev Inequalities

## Background

A university cafeteria tracks the daily number of meal trays returned by students. Let $X$ denote the number of trays returned on a randomly chosen day. It is known that $X \geq 0$, $E[X] = 120$, and $\text{var}(X) = 900$.

---

## Questions

**(a)** Using **Markov's inequality**, find an upper bound for the probability that more than 600 trays are returned on a given day:

$$P(X \geq 600) \leq \; ?$$

**(b)** Using **Chebyshev's inequality**, find an upper bound for the probability that the number of trays returned deviates from the mean by more than 90:

$$P(|X - 120| \geq 90) \leq \; ?$$

**(c)** Express the Chebyshev bound from part **(b)** using the **alternative form** $P(|X - \mu| \geq k\sigma) \leq \dfrac{1}{k^2}$.  
Identify $\sigma$, $k$, and verify the result matches part **(b)**.

---

*Hint: Recall that $\sigma = \sqrt{\text{var}(X)}$.*
