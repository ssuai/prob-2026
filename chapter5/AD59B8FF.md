# Problem 1 (Easy): Markov and Chebyshev Inequalities

## Problem Statement

Let $X$ be a non-negative random variable with mean $E[X] = 6$ and variance $\text{var}(X) = 4$.

**(a)** Using the **Markov inequality**, find an upper bound for $P(X \geq 10)$.

**(b)** Using the **Chebyshev inequality**, find an upper bound for $P(|X - 6| \geq 4)$.

**(c)** Suppose $X \sim \text{Uniform}[0, 12]$. Compute the **exact** values of the probabilities in (a) and (b), and compare them to the bounds you found.
