# Problem 3 (Hard)

**Topic: Central Limit Theorem \& Comparison with Chebyshev**

A web server handles $n = 100$ independent requests per second. The processing time of each request $X\_i$ is i.i.d. with:

$$E\[X\_i] = 5 \\text{ ms}, \\qquad \\text{var}(X\_i) = 4$$

Let $T = X\_1 + X\_2 + \\cdots + X\_{100}$ be the total processing time per second.

**(a)** \[Mean \& Variance of $T$] Compute $E\[T]$ and $\\text{var}(T)$.

**(b)** \[CLT Approximation] By the Central Limit Theorem, $T$ is approximately normal for large $n$. Standardize $T$ to define:

$$Z = \\frac{T - E\[T]}{\\sqrt{\\text{var}(T)}}$$

Use the CLT to approximate $P(T \\leq 520)$. Express your answer in terms of $\\Phi$ (the standard normal CDF).

**(c)** \[Two-sided probability] Approximate $P(460 \\leq T \\leq 540)$ using the CLT. Express in terms of $\\Phi$.

**(d)** \[Chebyshev Bound] Using the Chebyshev inequality, find an upper bound on $P(|T - E\[T]| \\geq 40)$.
