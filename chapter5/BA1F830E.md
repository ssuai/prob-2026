# Problem 3 (Hard) — Central Limit Theorem

## Background

A logistics company loads packages onto delivery trucks. The weight of each package is **independent** and follows the same distribution with mean $\mu = 10$ kg and variance $\sigma^2 = 4$ kg².  

A truck is loaded with $n = 100$ packages.

---

## Questions

**(a)** Define the total weight $S_{100} = X_1 + X_2 + \cdots + X_{100}$.  
Using the **Central Limit Theorem**, find the approximate probability that the total weight **exceeds 1030 kg**:

$$P(S_{100} > 1030) \approx \; ?$$

Express your answer using the standard normal CDF $\Phi(\cdot)$, and evaluate numerically using $\Phi(1.5) \approx 0.9332$.

**(b)** The truck has a **maximum capacity of $C$ kg**. The company wants to ensure:

$$P(S_{100} > C) \leq 0.05$$

Using the CLT, find the **minimum integer value of $C$** (in kg) that satisfies this requirement.

Use $\Phi(1.645) \approx 0.95$.


---

*Reference values:*  
$\Phi(0) = 0.5,\quad \Phi(1.5) = 0.9332,\quad \Phi(1.645) = 0.95,\quad \Phi(2.0) = 0.9772$
