## Scenario

A non-negative random variable $X \geq 0$ has expected value $E[X] = 6$.

Using only the **Markov Inequality** (slide 6):

$$P(X \geq a) \leq \frac{E[X]}{a}, \quad \text{for all } a > 0$$

answer the following questions.

---

## Questions

1. Find an upper bound for $P(X \geq 9)$.

$$\text{(a)} \quad \underline{\hspace{4cm}}$$

2. Find an upper bound for $P(X \geq 18)$.

$$\text{(b)} \quad \underline{\hspace{4cm}}$$

3. Without any additional information about the distribution of $X$, can you determine whether the Markov bounds in (a) and (b) are tight? Select the correct statement.

   - (i) Yes, because the Markov inequality is always an equality.
   - (ii) No, the Markov inequality gives an upper bound and may be loose, as illustrated by **Example 5.1**.
   - (iii) Yes, because $X \geq 0$ guarantees the bound is exact.

$$\text{(c)} \quad \underline{\hspace{4cm}}$$

---

## Answer

**(a)**

$$P(X \geq 9) \leq \frac{E[X]}{9} = \frac{6}{9} = \frac{2}{3}$$

**(b)**

$$P(X \geq 18) \leq \frac{E[X]}{18} = \frac{6}{18} = \frac{1}{3}$$

**(c)** Option **(ii)**

---

## Explanation

**Step 1.**  
Apply the Markov Inequality. The only requirements are $X \geq 0$ and knowledge of $E[X] = 6$.

**Step 2.**  
For (a): set $a = 9$ → $6/9 = 2/3$.  
For (b): set $a = 18$ → $6/18 = 1/3$.

**Step 3.**  
As shown in **Example 5.1** (slides 8–9), when $X \sim \text{Uniform}[0, 4]$ with $E[X] = 2$, the exact probability $P(X \geq 3) = 0.25$ is strictly below the Markov bound of $2/3 \approx 0.67$. The inequality provides only a rough upper bound — it can be quite loose.
