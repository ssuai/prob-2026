# Problem 1 — Easy | Markov Inequality

## Problem

Let $X$ be a non-negative random variable with $E[X] = 6$.

**(a)** Using the Markov inequality, find an upper bound for $P(X \geq 9)$.

**(b)** Using the Markov inequality, find an upper bound for $P(X \geq 15)$.

**(c)** Suppose additionally that $X \sim \text{Uniform}[0, 12]$.
Compute the **exact** probabilities $P(X \geq 9)$ and $P(X \geq 15)$, and compare them with the bounds from (a) and (b).
What does this tell you about the tightness of the Markov bound?

---

## Solution

### Part (a)

By the **Markov inequality**: for $X \geq 0$ and any $a > 0$,

$$P(X \geq a) \leq \frac{E[X]}{a}$$

With $E[X] = 6$ and $a = 9$:

$$P(X \geq 9) \leq \frac{6}{9} = \frac{2}{3} \approx 0.667$$

### Part (b)

With $E[X] = 6$ and $a = 15$:

$$P(X \geq 15) \leq \frac{6}{15} = \frac{2}{5} = 0.4$$

### Part (c)

For $X \sim \text{Uniform}[0, 12]$, the PDF is $f_X(x) = \frac{1}{12}$ for $0 \leq x \leq 12$.

**Exact value for (a):**

$$P(X \geq 9) = \int_{9}^{12} \frac{1}{12}\,dx = \frac{12 - 9}{12} = \frac{3}{12} = 0.25$$

Markov bound: $0.667$ &nbsp;→&nbsp; **actual probability is much smaller than the bound.**

**Exact value for (b):**

$$P(X \geq 15) = 0$$

since $X \leq 12$ always. Markov bound: $0.4$ &nbsp;→&nbsp; **the bound is not tight at all.**

**Conclusion:** The Markov inequality gives a valid (correct) upper bound in both cases,
but it can be very loose because it uses only the mean and ignores all other information
about the distribution. When more information is available (e.g., the variance or the full PDF),
tighter bounds or exact calculations are possible.
