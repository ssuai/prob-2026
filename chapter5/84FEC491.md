# Problem 1: Bounds on AI Inference Server Processing Time

**Problem Statement:**
An AI inference server processes batches of images. Let $X$ be a non-negative continuous random variable ($X \ge 0$) representing the processing time (in milliseconds) for a single batch. Through system logs, it is observed that the expected processing time is $E[X] = 20$ ms, and the variance is $\text{Var}(X) = 4 \text{ ms}^2$.

**Your Task:**
1. **Part A:** To set up a system alert for unusually long processing times, use **Markov's Inequality** to find an upper bound on the probability that a batch takes $30$ ms or more to process, i.e., $P(X \ge 30)$.
2. **Part B:** To analyze the consistency of the user experience, use **Chebyshev's Inequality** to find a lower bound on the probability that a batch's processing time is strictly between $16$ ms and $24$ ms, i.e., $P(16 < X < 24)$.
