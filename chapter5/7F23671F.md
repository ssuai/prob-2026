# Problem 2 — Intermediate | Chebyshev Inequality & Weak Law of Large Numbers

## Problem

Let $X_1, X_2, \ldots, X_n$ be i.i.d. random variables, each uniformly distributed on $[2, 8]$.

Recall: for $X \sim \text{Uniform}[a,b]$, $\quad E[X] = \dfrac{a+b}{2}$, $\quad \text{Var}(X) = \dfrac{(b-a)^2}{12}$.

**(a)** Compute $\mu = E[X_i]$ and $\sigma^2 = \text{Var}(X_i)$.

**(b)** Using the **Chebyshev inequality**, find an upper bound for $P(|X_1 - \mu| \geq 2)$ for a single observation.

**(c)** Let $M_n = \dfrac{X_1 + X_2 + \cdots + X_n}{n}$ be the sample mean.
Using the **Chebyshev inequality**, find an upper bound for $P(|M_n - \mu| \geq 0.5)$ when $n = 50$.

**(d)** How large must $n$ be so that the Chebyshev bound guarantees
$P(|M_n - \mu| \geq 0.5) \leq 0.05$?
