# Exercise 1: Probability Inequalities
**Level:** Easy

## Problem
A non-negative random variable $X$ is known to have an expected value $E[X] = 10$ and a variance $var(X) = 4$.

1. Using the **Markov Inequality**, find an upper bound for the probability $P(X \ge 20)$.
2. Using the **Chebyshev Inequality**, find an upper bound for the probability $P(|X - 10| \ge 6)$.
