# Problem 2 (Intermediate) — 공분산과 상관계수

## 문제

확률변수 $X$ (하루 공부 시간, 단위: 시간)와 $Y$ (시험 점수)에 대해 다음의 통계량이 알려져 있다.

$$E[X] = 3, \quad E[Y] = 70, \quad E[X^2] = 13, \quad E[Y^2] = 5200, \quad E[XY] = 225$$

**(a)** $\text{var}(X)$와 $\text{var}(Y)$를 각각 구하시오. (공식: $\text{var}(X) = E[X^2] - (E[X])^2$)

**(b)** $\text{cov}(X, Y)$를 구하시오. (공식: $\text{cov}(X,Y) = E[XY] - E[X]E[Y]$)

**(c)** 상관계수 $\rho(X, Y)$를 구하시오.
$$\rho(X, Y) = \frac{\text{cov}(X, Y)}{\sqrt{\text{var}(X) \cdot \text{var}(Y)}}$$

**(d)** $Z = X + Y$의 분산 $\text{var}(Z)$를 구하시오.
$$\text{var}(X + Y) = \text{var}(X) + \text{var}(Y) + 2\,\text{cov}(X, Y)$$

**(e)** $W = 2X - Y + 5$의 분산 $\text{var}(W)$를 구하시오.
$$\text{var}(aX + bY + c) = a^2\,\text{var}(X) + b^2\,\text{var}(Y) + 2ab\,\text{cov}(X, Y)$$
