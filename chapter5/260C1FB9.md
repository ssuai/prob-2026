## Problem 4 — Hard — CLT Approximation + Exact Comparison
Let $X_1, X_2, \dots$ be independent and identically distributed (i.i.d.) random variables with:
$$E[X_i] = 1, \qquad \text{Var}(X_i) = 1.$$
Define their cumulative sum as $S_n = X_1 + X_2 + \cdots + X_n$.

**Questions:**

(a) Express $P(S_n \le n + \sqrt{n})$ in terms of the standard normal cumulative distribution function $\Phi$ using the CLT, and evaluate its limit as $n \to \infty$.

(b) For what value of the constant $c$ does the following statement hold?
$$P(S_n \le n + c\sqrt{n}) \to 0.95 \quad \text{as } n \to \infty$$
(Use $\Phi(1.645) \approx 0.95$).

(c) Using Chebyshev’s inequality, find an upper bound for $P(S_n \ge 2n)$. Express this bound in terms of $n$, and determine whether it converges to 0 as $n \to \infty$.
