
3. hard

A server receives requests from 2000 users.

Each user independently sends a request during a given minute with probability

p = 0.12.

Let Xi be the indicator random variable for whether the i-th user sends a request:

Xi = 1 if the i-th user sends a request.
Xi = 0 otherwise.

Let S2000 be the total number of requests during that minute:

S2000 = X1 + X2 + ... + X2000.

The server becomes overloaded if it receives at least 270 requests.

Use the Central Limit Theorem with continuity correction to approximate

P(S2000 >= 270).

You may use

Phi(2.00) = approximately 0.9772.
