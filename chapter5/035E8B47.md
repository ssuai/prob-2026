# Question 3 (Hard): Central Limit Theorem (CLT)

A web service processes $n = 100$ independent AI inference requests. Let the processing time of the $i$-th request be $X_i$, which are independent and identically distributed (i.i.d.) random variables. The mean processing time for a single request is $\mu = 20$ ms, and the standard deviation is $\sigma = 4$ ms.
Let $S_{100} = X_1 + X_2 + \dots + X_{100}$ be the total processing time for the 100 requests. Using the Central Limit Theorem (CLT), approximate the probability that the total processing time exceeds 2080 ms.
*(Express your final answer in terms of the standard normal CDF, $\Phi(z)$).*
