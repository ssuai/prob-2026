# Problem 3 (Hard)

Consider a sequence of independent and identically distributed (i.i.d.) random variables $X_1, X_2, ..., X_{100}$, where each $X_i$ has a mean $\mu = 40$ and a variance $\sigma^2 = 16$. Let $M_{100} = \frac{1}{100} \sum_{i=1}^{100} X_i$ be the sample mean. We want to evaluate the probability that this sample mean deviates from the true mean by 1 or more.

**Question:**
(a) Find an upper bound for $P(|M_{100} - 40| \ge 1)$ using Chebyshev's inequality.
(b) Approximate $P(|M_{100} - 40| \ge 1)$ using the Central Limit Theorem. (Express your answer using the standard normal CDF, $\Phi(z)$).
