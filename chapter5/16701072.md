1. easy

Let X be a nonnegative random variable representing the daily power consumption of a device in kWh.

Suppose that

E[X]=10,Var(X)=16.

Answer the following questions.

1. Use Markov inequality to find an upper bound for P(X≥25).

2. Use Chebyshev inequality to find an upper bound for P(X≥25).

3. Compare the two bounds and explain which one is tighter.
