# Question 2 (Intermediate): Chebyshev Inequality

The time it takes for a MySQL database to execute a specific query is a random variable $X$. The mean execution time is $\mu = 50$ ms, and the variance is $\sigma^2 = 16$ ms$^2$.
Using Chebyshev's Inequality, find an upper bound for the probability that a query's execution time deviates from the mean by 10 ms or more (i.e., takes less than 40 ms or more than 60 ms).
