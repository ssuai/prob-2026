In distributed deep learning, a master node aggregates weight updates from n independent worker nodes. The computation time X_i each worker takes to process a batch of data is i.i.d. with mean E[X_i] = μ and variance var(X_i) = σ^2. To analyze the asymptotic behavior of the system, the master node normalizes the total computation time using the formula Z_n = (Σ_{i=1}^n X_i - nμ) / (σ√n).

Let X̃_i = (X_i - μ) / σ. We know its Moment Generating Function (MGF) can be expanded using the Taylor series as M_{X̃}(s) = 1 + (1/2)s^2 + o(s^2).

If μ = 15 seconds, σ^2 = 16 seconds^2, and n = 100, use the Central Limit Theorem to approximate the probability that the total processing time Σ_{i=1}~{100} X_i is strictly less than 1420 seconds. Express your final approximation using the standard normal cumulative distribution function, Φ(z).
