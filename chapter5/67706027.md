# Problem 2 (Intermediate)

**Topic: Chebyshev Inequality \& Weak Law of Large Numbers**

A network engineer wants to estimate the true mean packet processing time $\\mu$ of a router. The engineer collects $n$ independent measurements $X\_1, X\_2, \\ldots, X\_n$, each i.i.d. with:

$$E\[X\_i] = \\mu = 2 \\text{ ms}, \\qquad \\text{var}(X\_i) = 1$$

Define the sample mean:

$$M\_n = \\frac{X\_1 + X\_2 + \\cdots + X\_n}{n}$$

**(a)** \[Mean \& Variance of $M\_n$] Compute $E\[M\_n]$ and $\\text{var}(M\_n)$.

**(b)** \[Chebyshev Bound] Using the Chebyshev inequality, find an upper bound on $P(|M\_n - 2| \\geq 0.1)$ as a function of $n$.

**(c)** \[Minimum Sample Size] The engineer requires that $P(|M\_n - 2| \\geq 0.1) \\leq 0.01$. Using the bound from (b), find the minimum number of measurements $n$ needed.
