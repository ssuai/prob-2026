# Hard Problem: Central Limit Theorem (CLT) Approximation

An automated system processes 100 independent tasks. The time taken for each task \(X_i\) is uniformly distributed between 0 and 10 minutes.

Use the Central Limit Theorem (CLT) to approximate:

\[
P\left(\sum_{i=1}^{100} X_i > 520\right)
\]
