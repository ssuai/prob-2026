Problem 2: Level 2 (Intermediate/Application - Subjective/Descriptive)
A web server allocates memory X (in Megabytes) for each active connection. The memory allocated is strictly non-negative (X≥0). The average memory per connection is E[X]=120 MB and the variance is var(X)=144 MB$^2$. A critical system crash occurs if a connection demands 240 MB or more.
(a) Calculate the upper bound for the probability of a system crash, P(X≥240), using Markov's inequality.

(b) Calculate the upper bound for the same probability using Chebyshev's inequality.
