# Problem 3 (Hard)

The weight X_i (in kg) of each parcel delivered by a courier per day follows
an i.i.d. Uniform[10, 50] distribution.

(a) Compute E[X_i] and var(X_i).

(b) Using the CLT, approximate the probability that the total weight of
    64 parcels exceeds 1,984 kg.
    (Given: Φ(1.5) = 0.9332)

(c) The company wants the probability that the total weight exceeds (30n + 60) kg
    to be at most 5%.
    Using the CLT, find the minimum number of parcels n.
    (Given: Φ(1.645) = 0.95)

(d) When n is large, explain which gives a more accurate result:
    the Chebyshev bound or the CLT approximation, and why.
