### [Easy Level] Question 1: Database Query Response Time (15 Points)

The processing time $X$ (measured in milliseconds) of a database query in a cloud system is known to be strictly non-negative. Extensive historical data shows that the mean processing time is $\mu = 50$ ms, with a standard deviation of $\sigma = 10$ ms (meaning the variance is $\text{Var}(X) = 100$). The exact probability distribution function of $X$ is unknown.

**(1-a)** [Markov's Inequality] Using Markov's inequality, calculate an upper bound for the probability that a randomly selected query takes 100 ms or more to process, i.e., find the bound for $P(X \ge 100)$.

**(1-b)** [Chebyshev's Inequality] Using Chebyshev's inequality, calculate an upper bound for the probability that the processing time deviates from its mean by 50 ms or more, i.e., find the bound for $P(|X - 50| \ge 50)$. Compare this numerical bound with the bound obtained in (1-a) and briefly explain the statistical reason why one inequality provides a tighter (more precise) bound than the other in this scenario.
