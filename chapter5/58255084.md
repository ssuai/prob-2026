# Problem 1 (Easy) — 정규분포와 표준화

## 문제

어느 공장에서 생산하는 나사의 길이 $X$ (단위: mm)가 정규분포를 따른다.

$$X \sim N(50,\ 9) \quad (\text{즉, } \mu = 50,\ \sigma^2 = 9,\ \sigma = 3)$$

**(a)** 표준 정규확률변수 $Y = \dfrac{X - 50}{3}$ 를 이용하여, $P(X \leq 53)$ 을 $\Phi$ 함수로 표현하시오.

**(b)** $P(47 \leq X \leq 56)$ 을 $\Phi$ 함수로 표현하시오.

**(c)** 품질 기준: 길이가 $[44, 56]$ mm 범위를 **벗어나면** 불량품으로 판정한다. 임의의 나사가 불량품일 확률을 $\Phi$ 함수로 표현하시오.

**(d)** 새로운 확률변수 $W = 2X + 10$을 정의하자. $W$가 따르는 정규분포 $W \sim N(\mu_W,\ \sigma_W^2)$의 파라미터 $\mu_W$와 $\sigma_W^2$을 구하시오.
