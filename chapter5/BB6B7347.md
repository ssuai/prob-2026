# Probability and Statistics Practice Problems: Inequalities and Limit Theorems

## Problem 1 — Medium — Markov & Chebyshev
The number of customers arriving at a store in one hour is a nonnegative random variable $X$ with:
$$E[X] = 30, \qquad \text{Var}(X) = 25.$$

**Questions:**

(a) Find an upper bound for $P(X \ge 75)$ using the Markov inequality.

(b) Find an upper bound for $P(X \ge 75)$ using the Chebyshev inequality. Which of the two bounds (from part a and part b) is tighter?

(c) Find an upper bound for $P(X \le 15)$ using the Chebyshev inequality.
