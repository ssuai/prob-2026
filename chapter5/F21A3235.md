Easy: Heap Memory Allocation (Markov & Chebyshev)Focus: Directly calculating upper bounds using inequalities.Problem:Let $X$ (in KB) be the amount of dynamic memory allocated when creating objects in a Java program. The memory allocation is never negative ($X \ge 0$). We know the average allocation size is $E[X] = 50\text{ KB}$, and the variance is $Var(X) = 64$.Using Markov's Inequality, find the upper bound for the probability that a single request allocates 200 KB or more.Using Chebyshev's Inequality, find the upper bound for the probability that the allocated memory deviates from the mean by 24 KB or more (i.e., it is either $\le 26\text{ KB}$ or $\ge 74\text{ KB}$).
