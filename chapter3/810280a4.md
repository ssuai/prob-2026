**Question 1: Continuous PDF and Expectation**
Let a continuous random variable $X$ have the probability density function $f_X(x) = c x^2$ for $0 \leq x \leq 2$, and $f_X(x) = 0$ otherwise. 
- **(a)** Determine the constant $c$ that makes this a valid PDF.
- **(b)** Calculate the expected value $E[X]$ and variance $Var(X)$.
