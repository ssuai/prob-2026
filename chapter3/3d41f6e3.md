A communication channel transmits a binary message X from the set {-1, 1}, where the prior probabilities are P(X=1) = 0.6 and P(X=-1) = 0.4. The channel adds continuous noise Z to the signal, where Z is a standard normal random variable, meaning Z follows a Normal distribution with mean 0 and variance 1. The receiver observes the continuous signal Y = X + Z.

1. Write down the conditional probability density functions f_Y|X(y|1) and f_Y|X(y|-1).

2. Use the continuous Bayes' rule to find the posterior probability P(X=1 | Y=y) as a function of the observed value y.

3. If the receiver observes Y = 0, what is the probability that the transmitted message was actually X = 1?
