### [Medium Level] Question 2: 2D Target Tracking Error (20 Points)

A radar tracking system estimates the position of a moving target. The system has measurement errors in both the horizontal and vertical directions. Let the continuous random variables $X$ and $Y$ represent the 2D position errors in meters. 

The errors are jointly distributed according to the following Joint Probability Density Function (Joint PDF):
$$f_{X,Y}(x,y) = \begin{cases} c(x + y), & \text{for } 0 \le x \le 1 \text{ and } 0 \le y \le 1 \\ 0, & \text{otherwise} \end{cases}$$

**(2-a)** [Normalization] Find the exact value of the constant $c$ that makes $f_{X,Y}(x,y)$ a valid Joint PDF. Show your step-by-step double integration.

**(2-b)** [Marginal PDFs] Calculate the Marginal PDF of the horizontal error, $f_X(x)$, for $0 \le x \le 1$. Then, calculate the Marginal PDF of the vertical error, $f_Y(y)$, for $0 \le y \le 1$.

**(2-c)** [Independence] Are the random tracking errors $X$ and $Y$ mathematically independent? Prove your answer by rigorously checking the condition for independence of continuous random variables using the joint and marginal PDFs you calculated.
