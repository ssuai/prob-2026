[Intermediate] 2. Joint PDF and Marginal PDF

Problem:
The joint PDF of random variables $X$ and $Y$ is given by:
$f_{X,Y}(x,y) = \begin{cases} c(x + 2y) & 0 \le x \le 1, 0 \le y \le 1 \\ 0 & \text{otherwise} \end{cases}$

1) Find the value of the constant $c$ that makes this a valid joint PDF.
2) Find the marginal PDF $f_X(x)$.
