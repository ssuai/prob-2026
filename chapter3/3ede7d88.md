## Problem

### Korean

두 연속 확률변수 $X$와 $Y$의 결합 확률밀도함수(joint PDF)가 다음과 같이 주어진다.

$$f_{X,Y}(x, y) = \begin{cases} c\,(x + y) & 0 \leq x \leq 1,\; 0 \leq y \leq 1 \\ 0 & \text{otherwise} \end{cases}$$

**(1)** 상수 $c$의 값을 결정하여라.

**(2)** $X$의 주변 PDF(marginal PDF) $f_X(x)$를 구하여라.

**(3)** $X = \frac{1}{2}$ 가 주어졌을 때, $Y$의 조건부 PDF $f_{Y \mid X}(y \mid \frac{1}{2})$를 구하여라.

**(4)** $P\!\left(Y > \frac{1}{2} \,\Big|\, X = \frac{1}{2}\right)$를 계산하여라.

---

### English

Two continuous random variables $X$ and $Y$ have the joint probability density function (PDF):

$$f_{X,Y}(x, y) = \begin{cases} c\,(x + y) & 0 \leq x \leq 1,\; 0 \leq y \leq 1 \\ 0 & \text{otherwise} \end{cases}$$

**(1)** Determine the constant $c$.

**(2)** Find the marginal PDF of $X$, denoted $f_X(x)$.

**(3)** Find the conditional PDF of $Y$ given $X = \frac{1}{2}$, denoted $f_{Y \mid X}\!\left(y \mid \tfrac{1}{2}\right)$.

**(4)** Compute $P\!\left(Y > \frac{1}{2} \,\Big|\, X = \frac{1}{2}\right)$.

---

## Solution

> **[Step 1] Model Setup**
>
> The sample space is $\Omega = [0,1] \times [0,1]$, the unit square. The probability of any region $A \subseteq \Omega$ is
> $$P((X,Y) \in A) = \iint_A f_{X,Y}(x,y)\, dx\, dy.$$
>
> For $f_{X,Y}$ to be a valid joint PDF, two conditions must hold:
> 1. $f_{X,Y}(x,y) \geq 0$ for all $(x,y)$ — satisfied since $c > 0$ and $x+y \geq 0$ on $[0,1]^2$.
> 2. $\displaystyle\iint_{\mathbb{R}^2} f_{X,Y}(x,y)\, dx\, dy = 1$ — this determines $c$.

---

> **[Step 2] Key Observation / Diagram**
>
> The support of $f_{X,Y}$ is the unit square. A sketch helps:
>
> ```
>   y
>   1 +-------+
>     |       |
>     | f > 0 |
>     |       |
>   0 +-------+-- x
>     0       1
> ```
>
> - **Marginal PDF:** integrating out one variable over the full support.
> - **Conditional PDF:** by definition (Bertsekas, Eq. 3.17),
>   $$f_{Y|X}(y \mid x) = \frac{f_{X,Y}(x,y)}{f_X(x)}, \quad f_X(x) > 0.$$
> - **Conditional probability:** $P(Y > \frac{1}{2} \mid X = \frac{1}{2}) = \int_{1/2}^{1} f_{Y|X}(y \mid \frac{1}{2})\, dy$.

---

> **[Step 3] Derivation**
>
> **Part (1) — Find $c$:**
>
> $$\int_0^1 \int_0^1 c(x+y)\, dy\, dx = 1.$$
>
> Inner integral ($y$):
> $$\int_0^1 (x+y)\, dy = \left[xy + \frac{y^2}{2}\right]_0^1 = x + \frac{1}{2}.$$
>
> Outer integral ($x$):
> $$\int_0^1 \left(x + \frac{1}{2}\right) dx = \left[\frac{x^2}{2} + \frac{x}{2}\right]_0^1 = \frac{1}{2} + \frac{1}{2} = 1.$$
>
> Therefore $c \cdot 1 = 1$, giving $\boxed{c = 1}$.
>
> ---
>
> **Part (2) — Marginal PDF $f_X(x)$:**
>
> For $0 \leq x \leq 1$:
> $$f_X(x) = \int_0^1 (x+y)\, dy = x + \frac{1}{2}.$$
>
> Thus
> $$f_X(x) = x + \frac{1}{2}, \quad 0 \leq x \leq 1, \quad f_X(x) = 0 \text{ otherwise}.$$
>
> ---
>
> **Part (3) — Conditional PDF $f_{Y|X}(y \mid \frac{1}{2})$:**
>
> First evaluate $f_X\!\left(\frac{1}{2}\right)$:
> $$f_X\!\left(\frac{1}{2}\right) = \frac{1}{2} + \frac{1}{2} = 1.$$
>
> Then by the definition of the conditional PDF:
> $$f_{Y|X}\!\left(y \,\Big|\, \frac{1}{2}\right) = \frac{f_{X,Y}\!\left(\frac{1}{2}, y\right)}{f_X\!\left(\frac{1}{2}\right)} = \frac{\frac{1}{2} + y}{1} = \frac{1}{2} + y, \quad 0 \leq y \leq 1.$$
>
> ---
>
> **Part (4) — Conditional Probability:**
>
> $$P\!\left(Y > \frac{1}{2} \,\Big|\, X = \frac{1}{2}\right) = \int_{1/2}^{1} \left(\frac{1}{2} + y\right) dy.$$
>
> $$= \left[\frac{y}{2} + \frac{y^2}{2}\right]_{1/2}^{1} = \left(\frac{1}{2} + \frac{1}{2}\right) - \left(\frac{1}{4} + \frac{1}{8}\right) = 1 - \frac{3}{8} = \frac{5}{8}.$$

---

> **[Step 4] Computation**
>
> | Part | Result |
> |------|--------|
> | (1) | $c = 1$ |
> | (2) | $f_X(x) = x + \tfrac{1}{2}$, $\; 0 \leq x \leq 1$ |
> | (3) | $f_{Y\|X}(y \mid \tfrac{1}{2}) = \tfrac{1}{2} + y$, $\; 0 \leq y \leq 1$ |
> | (4) | $P(Y > \tfrac{1}{2} \mid X = \tfrac{1}{2}) = \tfrac{5}{8} = 0.625$ |

---

> **[Step 5] Verification**
>
> - **Normalization of $f_X$:**
>   $\int_0^1 (x + \frac{1}{2})\, dx = \frac{1}{2} + \frac{1}{2} = 1.$ (OK)
>
> - **Normalization of $f_{Y|X}(\cdot \mid \frac{1}{2})$:**
>   $\int_0^1 (\frac{1}{2} + y)\, dy = \frac{1}{2} + \frac{1}{2} = 1.$ (OK)
>
> - **Probability bound:** $0 \leq \frac{5}{8} \leq 1.$ (OK)
>
> - **Sanity check:** Given $X = \frac{1}{2}$, the conditional PDF $\frac{1}{2} + y$ is increasing in $y$, so higher $y$ values are more likely. It is therefore expected that $P(Y > \frac{1}{2} \mid X = \frac{1}{2}) > \frac{1}{2}$, consistent with $\frac{5}{8}$. (OK)

---

**Answer:**

$$c = 1, \quad f_X(x) = x + \frac{1}{2}\;(0 \leq x \leq 1), \quad f_{Y|X}\!\left(y\,\Big|\,\tfrac{1}{2}\right) = \frac{1}{2}+y\;(0\leq y\leq 1),$$

$$P\!\left(Y > \frac{1}{2}\;\Big|\; X = \frac{1}{2}\right) = \frac{5}{8}.$$
