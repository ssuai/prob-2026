# 2. Exponential Random Variable (Intermediate)

A backup power unit is installed in a remote weather station. Let \(T\) denote the time (in days) until the unit fails for the first time. Assume that \(T\) is an exponential random variable with probability density function

\[
f_T(t)=
\begin{cases}
\lambda e^{-\lambda t}, & t\ge 0,\\
0, & \text{otherwise},
\end{cases}
\]

where \(\lambda>0\) is an unknown parameter.

The manufacturer states that the expected lifetime of the unit is 8 days.

Answer the following questions.

(a) Verify that \(f_T(t)\) is a valid probability density function.

(b) Using the given expected lifetime, determine the value of \(\lambda\).

(c) Find the cumulative distribution function \(F_T(t)\).

(d) Compute the probability that the unit fails between 3 days and 10 days, that is, find

\[
P(3<T<10).
\]

(e) Find both \(E[T]\) and \(\mathrm{Var}(T)\).
