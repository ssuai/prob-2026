A continuous random variable $X$ has a probability density function (PDF) defined as:
$$
f_X(x) = \begin{cases} c \cdot x, & \text{if } 0 \le x \le 2 \\ 0, & \text{otherwise} \end{cases}
$$
Find the value of the constant $c$ that makes this a valid PDF, and then calculate the expected value $E[X]$.
