# Easy Problem (Joint Probability)

Let X and Y be two discrete random variables with the following joint probability mass function:

| X \ Y | 0   | 1   |
|------|-----|-----|
| 0    | 0.2 | 0.3 |
| 1    | 0.1 | 0.4 |

1. Find the marginal probability P(X = 0).
2. Find the marginal probability P(Y = 1).
3. Find the probability P(X = 1, Y = 0).
4. Check whether X and Y are independent.
