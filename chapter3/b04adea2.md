# Continuous Random Variable Problem: Intermediate #1

## AI Model Inference Time
The response time $X$ (in seconds) for a complex AI model to generate a strategic analysis query has the probability density function (PDF):

$$f(x) = \frac{k}{x^2}, \quad 1 \le x \le 5$$
And $f(x) = 0$ otherwise.

1. Find the value of the constant $k$ that makes this a valid PDF.
2. Calculate the expected response time, $E[X]$.
