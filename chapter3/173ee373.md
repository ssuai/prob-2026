# **Problem 1 (Easy): Normal Distributions in Data Center Thermal Management**

###### **Topic Category: Normal Random Variables**

###### **Difficulty Level: Easy**

###### **Domain Context: Concurrent Traffic Load Balancing / Hardware Thermal Limits**



##### **Problem Context and Statement**

* A high-density hyperscale server cluster manages thousands of concurrent client connections utilizing an adaptive load balancing algorithm. As the load balancer distributes incoming network traffic across the active server pool, the individual internal CPUs generate highly variable levels of heat based on their respective localized processing loads. Due to the additive nature of tens of thousands of independent micro-processes executing concurrently across the racks, the aggregate CPU temperatures across the entire cluster closely approximate a Normal (Gaussian) Distribution.
* Live thermal telemetry indicates that the CPU temperatures ($X$), measured in degrees Celsius, maintain an operational mean ($\\mu$) of $60^\\circ C$ and a statistical variance ($\\sigma^2$) of $100$.
* Stringent hardware safety protocols dictate that if any individual CPU's core temperature exceeds $80^\\circ C$, the server will instantly initiate an aggressive thermal throttling procedure to protect the silicon. This procedure severely degrades network performance and throughput.
* Given standard normal CDF cumulative probability values:



* $\\Phi(1.0) = 0.8413$
* $\\Phi(1.5) = 0.9332$
* $\\Phi(2.0) = 0.9772$
* $\\Phi(2.5) = 0.9938$



* (Note: $\\Phi(z)$ explicitly represents the cumulative probability $P(Z \\le z)$ for a Standard Normal random variable Z).



##### **Part A:**

* Calculate and define the exact standard deviation ($\\sigma$) of the CPU temperature distribution.

##### **Part B:**

* An automated monitoring agent randomly selects one single CPU from the load-balanced data center cluster. Calculate the exact mathematical probability that this randomly selected CPU is currently exceeding $80^\\circ C$ and actively triggering thermal throttling protocols. You must fully demonstrate the mathematical standardization process utilized to convert the raw random variable $X$ into a standard normal variable $Z$ to receive credit.
