## Problem 2: Server Rendering and Query Times (Intermediate)

A cloud computing engineer monitors two continuous random variables to measure the efficiency of a web application: 
- $X$: The time (in fractions of a second) it takes to fetch data from the database.
- $Y$: The time (in fractions of a second) it takes to render the graphical user interface.

The Joint Probability Density Function (Joint PDF) of $X$ and $Y$ is modeled as:

$$
f_{X,Y}(x,y) = 
\begin{cases} 
k(x^2 + y) & \text{for } 0 \le x \le 1 \text{ and } 0 \le y \le 2 \\ 
0 & \text{otherwise} 
\end{cases}
$$

**Questions:**

**(a)** Determine the exact value of the constant $k$ that makes $f_{X,Y}(x,y)$ a valid joint probability density function.

**(b)** Find the marginal probability density functions, $f_X(x)$ and $f_Y(y)$. Using these functions, mathematically prove whether the database fetching time ($X$) and the UI rendering time ($Y$) are strictly independent variables.

**(c)** The system alerts the administrative team if the combined time of both processes takes less than 1 second. Set up and evaluate the double integral required to find $P(X + Y < 1)$.
