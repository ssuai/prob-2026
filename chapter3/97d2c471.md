## 🔴 3️⃣ Hard Problem 

The random variable ( X ) follows the distribution:

$$
X \sim \text{Uniform}(0,1)
$$

(1)

Find the CDF $F_X(x)$.

(2)

Let the random variable

$$Y = X^2$$

Find the CDF $F_Y(y)$.

(3) 

Using the result above, find the PDF $f_Y(y)$.

(4) 

Find

$$E[Y]$$

(5) 

Compute the following directly using integration:

$$E[X^2]$$

and compare it with the result from (4).

---

Answer: 

(1) $x$

(2) 
$$F_{Y}(y)=\sqrt{y} \\ (0≤y≤1)$$

(3) 
$$
f_{Y}(y) =
\begin{cases}
\tfrac{1}{2\sqrt{y}}, & 0 < y ≤ 1 \\
0, & \text{otherwise}
\end{cases}
$$

(4)
$\tfrac{1}{3}$

(5)
$\tfrac{3}{4}$
, this result should match the result in (4).
