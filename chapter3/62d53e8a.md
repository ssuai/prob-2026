# Problem 3: Parabolic Boundary and Conditional Density (Hard)

## 1. Problem Description
In a wireless communication system, assume the relationship between the transmitted signal strength $X$ and the interference level $Y$ is restricted to the following parabolic region:

$$
f_{X,Y}(x,y) = 
\begin{cases} 
c, & 0 \le x \le 1, \ 0 \le y \le x^2 \\ 
0, & \text{otherwise} 
\end{cases}
$$

## 2. Questions
1. Find the constant $c$ using the definition of the joint PDF.
2. Derive the marginal probability density function of $Y$, $f_Y(y)$. (Note: Be careful to express the integration limits of $x$ in terms of $y$).
3. Determine the conditional probability density function $f_{X|Y}(x|y)$, and describe whether this distribution represents independent or dependent variables based on the boundary conditions.
