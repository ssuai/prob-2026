# Intermediate

## Topic

Standard normal random variable

## Problem

A machine produces metal rods whose length X (in cm) follows a normal distribution with mean 50 and variance 4.

That is,

X ~ N(50, 4)

1. Standardize X using a standard normal random variable Y.
2. Express P(48 <= X <= 53) in terms of Phi(.).
3. Express P(X >= 52) in terms of Phi(.).
