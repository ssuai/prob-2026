# Hard

## Topic

Exponential distribution and Bayes' rule

## Problem

A security system generates one of two types of alerts.

- With probability 0.4, the alert is Type A. In this case, the response time T (in hours) follows an exponential distribution with mean 2.
- With probability 0.6, the alert is Type B. In this case, the response time T follows an exponential distribution with mean 5.

Now suppose an alert is observed, and its response time is longer than 3 hours.

1. Find P(T > 3 | A).
2. Find P(T > 3).
3. Given that T > 3, find the probability that the alert was Type B.
