## Problem

Let a continuous random variable $X$ have the probability density function

$$
f_X(x) =
\begin{cases}
2x, & 0 \le x \le 1 \\
0, & \text{otherwise}
\end{cases}
$$

1. Verify that $f_X(x)$ is a valid probability density function.
2. Find the cumulative distribution function $F_X(x)$ for $0 \le x \le 1$.
