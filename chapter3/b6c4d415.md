1. Easy: The Decaying SignalA continuous random variable $X$ represents the time (in milliseconds) it takes for a specific electronic signal to dissipate.
The probability density function (PDF) is given by:$$f(x) =\begin{cases}ce^{-x/2} & x \ge 0 \0 & x < 0\end{cases}$$
Task A: Determine the value of the constant $c$ such that $f(x)$ is a valid probability density function.
Task B: Calculate the probability that a signal takes longer than 4 milliseconds to dissipate, $P(X > 4)$.
Task C: Find the cumulative distribution function (CDF), $F(x)$, for all $x$.
