 # Hard
Suppose X is exponentially distributed with mean 4.
Given X = x, a measurement Y is normally distributed with mean x and variance 1.

That is,

X ~ Exponential(lambda)
Y | X = x ~ Normal(x, 1)

1. Find the value of lambda.
2. Write the PDF f_X(x).
3.Write the conditional PDF f_(Y|X)(y|x).
4. Write the joint PDF f_(X,Y)(x,y).
5. Find P(X > 2).
6. Find E[Y].
