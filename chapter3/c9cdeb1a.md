# Intermediate Problem (Joint and Conditional Probability)

Let X and Y be continuous random variables with joint probability density function:

f(x, y) = 6xy, for 0 ≤ x ≤ 1 and 0 ≤ y ≤ 1

1. Verify that f(x, y) is a valid joint probability density function.
2. Find the marginal density f_X(x).
3. Find the conditional density f(Y | X = x).
4. Determine whether X and Y are independent.
