## Problem

A dart is thrown uniformly at random onto a triangular board whose corners are at the points $(0,\,0)$, $(2,\,0)$, and $(0,\,2)$ in the $xy$-plane. Let $(X,\,Y)$ denote the impact point. The joint PDF is

$$
f_{X,Y}(x,\,y) = \begin{cases} c, & \text{if } x \ge 0,\; y \ge 0,\; x + y \le 2 \\ 0, & \text{otherwise} \end{cases}
$$

1. Find the constant $c$.  
   **(a)** $c$ = ________

2. Find the marginal PDF $f_X(x)$ for $0 \le x \le 2$.  
   **(b)** $f_X(x)$ = ________

3. Given that $X = 1$, find the conditional PDF $f_{Y|X}(y\,|\,1)$ and compute $E[Y\,|\,X = 1]$.  
   **(c)** $f_{Y|X}(y\,|\,1)$ = ________ for ________ $\le y \le$ ________  
   **(d)** $E[Y\,|\,X = 1]$ = ________

---

## Answer

| | Answer |
|---|--------|
| **(a)** | $\dfrac{1}{2}$ |
| **(b)** | $f_X(x) = \dfrac{2 - x}{2}$ for $0 \le x \le 2$ |
| **(c)** | $f_{Y|X}(y\,|\,1) = 1$ for $0 \le y \le 1$ |
| **(d)** | $E[Y\,|\,X = 1] = \dfrac{1}{2}$ |

---

## Explanation

**Step 1. Finding $c$**

By the normalization property $\displaystyle\int\!\!\int f_{X,Y}(x,y)\,dx\,dy = 1$, we need

$$
c \cdot (\text{area of triangle}) = 1
$$

The triangle has area $\dfrac{1}{2} \times 2 \times 2 = 2$, so

$$
c = \frac{1}{2}
$$

**Step 2. Marginal PDF $f_X(x)$**

Using the marginalization formula $f_X(x) = \displaystyle\int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dy$:

For a given $x \in [0,\,2]$, $y$ ranges from $0$ to $2 - x$ inside the triangle.

$$
f_X(x) = \int_0^{2-x} \frac{1}{2}\,dy = \frac{2 - x}{2}
$$

**Step 3. Conditional PDF $f_{Y|X}(y\,|\,1)$**

By the conditioning formula $f_{Y|X}(y\,|\,x) = \dfrac{f_{X,Y}(x,\,y)}{f_X(x)}$:

First compute $f_X(1) = \dfrac{2 - 1}{2} = \dfrac{1}{2}$.

$$
f_{Y|X}(y\,|\,1) = \frac{1/2}{1/2} = 1, \quad 0 \le y \le 1
$$

This is a uniform PDF on $[0,\,1]$.

**Step 4. Conditional expectation**

Since $Y\,|\,X = 1$ is uniform on $[0,\,1]$:

$$
E[Y\,|\,X = 1] = \frac{0 + 1}{2} = \frac{1}{2}
$$
