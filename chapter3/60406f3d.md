Problem 2 — Intermediate Level: Joint PDF and Cumulative Distribution Function

Problem Statement

Random variables X and Y have joint probability density function

f_{X,Y}(x,y)=
\begin{cases}
c(x+y), & 0<x<1,\;0<y<1 \\
0, & \text{otherwise}
\end{cases}

where c is a constant.
	1.	Find the value of c.
	2.	Compute the joint cumulative distribution function

F_{X,Y}(a,b)=P(X\le a,\;Y\le b)

for 0<a<1,\;0<b<1.
3. Determine whether X and Y are independent.

Given Information / Assumptions
	•	X,Y are continuous random variables.
	•	Use the formal definition of a joint CDF.
