Two continuous random variables X and Y have a joint probability density function given by:
f_X,Y(x,y) = c, if 0 <= x <= 2 and 0 <= y <= 1
f_X,Y(x,y) = 0, otherwise

1. Find the value of the constant c that makes this a valid joint PDF.

2. Find the marginal PDF of X, f_X(x), by integrating the joint PDF over y.

3. Are X and Y independent? Justify your answer using the relationship between the joint PDF and the marginal PDFs.
