# Easy Problem

A continuous random variable \(X\) has the following probability density function:

\[
f_X(x)=
\begin{cases}
cx, & 0 \le x \le 2 \\
0, & \text{otherwise}
\end{cases}
\]

where \(c\) is a constant.

1. Find the value of \(c\).  
2. Compute \(P(1 \le X \le 2)\).  
3. Compute \(P(X = 1)\).
