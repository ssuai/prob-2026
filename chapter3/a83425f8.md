# Problem 1

**Difficulty:** Easy

A continuous random variable \(X\) has pdf

\[
f_X(x)=
\begin{cases}
c(1+x), & 0\le x\le 1,\\
0, & \text{otherwise}.
\end{cases}
\]

1. Find the constant \(c\).
2. Compute \(P\left(\frac14 \le X \le \frac34\right)\).

**Main Topic:** Continuous random variables, PDF, interval probability
