# Problem 3: Joint PDF and Conditional Probability (Hard)

Two continuous random variables X and Y are uniformly distributed over a region S. 
The region S is defined as the union of two rectangles: 
A = {(x,y) | 1 ≤ x ≤ 2, 1 ≤ y ≤ 4} and B = {(x,y) | 2 ≤ x ≤ 3, 2 ≤ y ≤ 3}.

The joint PDF is given by f_X,Y(x,y) = c if (x,y) ∈ S, and 0 otherwise.
Find the conditional probability P(X + Y ≤ 4 | X ≥ 1.5).
