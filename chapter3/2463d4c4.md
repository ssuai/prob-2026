# Intermediate Problem

A discrete random variable \(X\) has the following probability mass function:

\[
p_X(k)=
\begin{cases}
ck, & k=1,2,3,4 \\
0, & \text{otherwise}
\end{cases}
\]

where \(c\) is a constant.

1. Find the value of \(c\).  
2. Find the cumulative distribution function \(F_X(x)=P(X\le x)\).  
3. Compute \(P(2 < X \le 4)\).  
4. Find the smallest value of \(x\) such that \(F_X(x)\ge 0.7\).
