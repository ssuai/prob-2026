Let X and Y be continuous random variables with joint probability density function

f_X,Y(x,y) = c, for 0 <= x <= 2 and 0 <= y <= x
f_X,Y(x,y) = 0, otherwise

Find the value of the constant c.
Find the marginal probability density functions f_X(x) and f_Y(y).
Find P(Y <= 1).
Determine whether X and Y are independent.
