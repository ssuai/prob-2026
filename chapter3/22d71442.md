# 1. Continuous Random Variable (Easy)

A small robot moves along a straight track. Let \(X\) be the distance (in meters) that the robot travels in one trial. The random variable \(X\) is continuous and has the following probability density function:

\[
f_X(x)=
\begin{cases}
\frac{x}{2}, & 0 \le x \le 2,\\
0, & \text{otherwise.}
\end{cases}
\]

Answer the following questions.

(a) Verify that \(f_X(x)\) is a valid probability density function.

(b) Compute the probability that the robot travels between \(0.5\) meters and \(1.5\) meters, that is, find \(P(0.5 \le X \le 1.5)\).

(c) Compute the probability that the robot travels at least \(1\) meter, that is, find \(P(X \ge 1)\).

(d) Find the expected value \(E[X]\).

(e) Find \(E[X^2]\).

(f) Find the variance \(\mathrm{Var}(X)\).

(g) Briefly explain why \(P(X=1)=0\) even though \(f_X(1)\neq 0\).
