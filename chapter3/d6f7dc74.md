# Hard Problem

The lifetime (in hours) of a certain sensor is modeled by a normal random variable

\[
X \sim N(1000, 100^2)
\]

A company classifies the sensors as follows:

- **Low quality**: \(X < 900\)
- **Standard quality**: \(900 \le X \le 1100\)
- **High quality**: \(X > 1100\)

1. Find the probability that a randomly selected sensor is classified as **High quality**.  
2. Find the probability that a randomly selected sensor is classified as **Standard quality**.  
3. If 25 sensors are independently produced, find the expected number of sensors classified as **Low quality**.  
4. The company wants only the top 10% longest-lasting sensors to be labeled **Premium**. Find the minimum lifetime threshold \(T\) such that

\[
P(X \ge T)=0.10
\]

(Express your answer in terms of the standard normal table.)
