2. Intermediate: Power and Transformation
Consider a continuous random variable $Y$ with the following PDF:$$f_Y(y) =\begin{cases}\frac{3}{8}y^2 & 0 \le y \le 2 \0 & \text{otherwise}\end{cases}$$
Task A: Calculate the expected value $E[Y]$ and the variance $Var(Y)$.
Task B: Let a new random variable be defined by the transformation $W = Y^3$. Derive the probability density function $f_W(w)$ for $W$.
Task C: Using $f_W(w)$ from Task B, calculate $E[W]$ and verify if it matches the result you would get using the Law of the Unconscious Statistician (LOTUS) on $Y$.
