# Problem 3 (Hard)

Let X be a normal random variable with mean = 2 and variance = 16. 
That is, X ~ N(2, 16).

* Given values from the standard normal table: 
  Phi(0.0) = 0.5000,  Phi(1.0) = 0.8413,  Phi(1.5) = 0.9332

(a) By standardizing the random variable X, express the probability P(X <= 6) in terms of the standard normal cumulative distribution function, Phi(z).
(b) Determine the exact value of P(X <= 2) using the given values above.
