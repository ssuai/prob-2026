## 🟢 1️⃣ Easy Problem


The PDF of the random variable ( X ) is given by:

$$
f_X(x) =
\begin{cases}
2x, & 0 \le x \le 1 \\
0, & \text{otherwise}
\end{cases}
$$

(1)

Verify that this function is a valid PDF.

(2)

Find the CDF $F_X(x)$

(3)

Find

$$
P\left(\tfrac{1}{2} \le X \le 1\right)
$$

(4)

Find the expectation $E[X]$.

---


Answer: 

(1) valid pdf
    
(2) $x^2$

(3) 
$\tfrac{3}{4}$

(4)
$\tfrac{2}{3}$
