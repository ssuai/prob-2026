## Problem

A binary message $X \in \{-1,\, 1\}$ is transmitted with equal probability, $P(X = 1) = P(X = -1) = \dfrac{1}{2}$. The communication channel adds noise $Z \sim \mathcal{N}(0,\,1)$, so the received signal is

$$
Y = X + Z
$$

The receiver uses the decision rule:

$$
\hat{X} = \begin{cases} 1, & \text{if } Y \ge 0 \\ -1, & \text{if } Y < 0 \end{cases}
$$

Use the standard normal table value $\Phi(1.0) = 0.8413$.

1. Using the total probability approach (as in Example 3.8), derive and compute the probability of error $P(\hat{X} \ne X)$.  
   **(a)** $P(\hat{X} \ne X)$ = ________ (exact expression in terms of $\Phi$, then numerical value)

2. Using the continuous Bayes' rule (as in Example 3.20 with $p = 1/2$), write the posterior probability $P(X = 1\,|\,Y = y)$ and determine the value of $y$ at which the receiver is equally uncertain about $X = 1$ and $X = -1$. Explain why this is consistent with the decision rule.  
   **(b)** $P(X = 1\,|\,Y = y)$ = ________  
   **(c)** The value of $y$ where $P(X = 1\,|\,Y = y) = \dfrac{1}{2}$ is $y$ = ________

---

## Answer

| | Answer |
|---|--------|
| **(a)** | $1 - \Phi(1) = 0.1587$ |
| **(b)** | $\dfrac{e^y}{e^y + e^{-y}}$ |
| **(c)** | $y = 0$ |

---

## Explanation

**Step 1. Decompose the error event**

$$
P(\hat{X} \ne X) = P(\hat{X} = 1,\, X = -1) + P(\hat{X} = -1,\, X = 1)
$$

$$
= P(\hat{X} = 1\,|\,X = -1)\,P(X = -1) + P(\hat{X} = -1\,|\,X = 1)\,P(X = 1)
$$

**Step 2. When $X = -1$**

$Y = -1 + Z$. The error $\hat{X} = 1$ occurs when $Y \ge 0$, i.e., $Z \ge 1$.

$$
P(\hat{X} = 1\,|\,X = -1) = P(Z \ge 1) = 1 - \Phi(1)
$$

**Step 3. When $X = 1$**

$Y = 1 + Z$. The error $\hat{X} = -1$ occurs when $Y < 0$, i.e., $Z < -1$.

By the symmetry of the standard normal:

$$
P(Z < -1) = 1 - \Phi(1)
$$

**Step 4. Combine**

Both error terms are identical:

$$
P(\hat{X} \ne X) = (1 - \Phi(1)) \cdot \frac{1}{2} + (1 - \Phi(1)) \cdot \frac{1}{2} = 1 - \Phi(1) = 1 - 0.8413 = \mathbf{0.1587}
$$

**Step 5. Posterior via Bayes' rule**

Applying the continuous Bayes' rule (Example 3.20) with $p = \dfrac{1}{2}$:

$$
P(X = 1\,|\,Y = y) = \frac{p \cdot e^y}{p \cdot e^y + (1-p) \cdot e^{-y}} = \frac{\frac{1}{2}\,e^y}{\frac{1}{2}\,e^y + \frac{1}{2}\,e^{-y}} = \frac{e^y}{e^y + e^{-y}}
$$

**Step 6. Decision boundary**

Setting $P(X = 1\,|\,Y = y) = \dfrac{1}{2}$:

$$
\frac{e^y}{e^y + e^{-y}} = \frac{1}{2}
$$

$$
2e^y = e^y + e^{-y} \implies e^y = e^{-y} \implies y = 0
$$

This is exactly the decision boundary used by the receiver ($\hat{X} = 1$ when $Y \ge 0$), confirming that the rule is optimal when $p = 0.5$: the receiver decides $X = 1$ precisely when the posterior probability $P(X = 1\,|\,Y = y) \ge \dfrac{1}{2}$.
