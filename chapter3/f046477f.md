# Problem 2: AI Model Resource Allocation

**Problem Statement:**
While developing a Retrieval-Augmented Generation (RAG) pipeline, you are monitoring the system's resource usage during inference. Let the continuous random variable $X$ represent the proportion of allocated GPU memory utilized during the retrieval phase, and the continuous random variable $Y$ represent the proportion utilized during the generation phase.

The joint probability density function (PDF) of $X$ and $Y$ is modeled as:
$$f_{X,Y}(x,y) = \begin{cases} c(x^2 + y) & \text{for } 0 \le x \le 1 \text{ and } 0 \le y \le 1 \\ 0 & \text{otherwise} \end{cases}$$

**Your Task:**
1. **Part A:** Determine the value of the constant $c$ that makes this a valid joint PDF.
2. **Part B:** Find the marginal probability density function of the retrieval phase memory usage, $f_X(x)$.
3. **Part C:** Calculate the probability that the pipeline's combined memory utilization across both phases exceeds $100\%$ of a single allocation unit (i.e., find $P(X + Y > 1)$).
