# Continuous Random Variable Problem: Easy #4

## The Background Process
The percentage of CPU utilization $X$ (expressed as a decimal from 0 to 1) for a Java background process has the probability density function (PDF):

$$f(x) = 3x^2, \quad 0 \le x \le 1$$

Find the probability that the CPU utilization is at least 50% ($X \ge 0.5$).
