## Problem

Let $X$ be a continuous uniform random variable on the interval $[2,\, 8]$.

1. Find the expected value $E[X]$.  
   **(a)** $E[X]$ = ________

2. Find the variance $\text{var}(X)$.  
   **(b)** $\text{var}(X)$ = ________

3. Find $P(3 \le X \le 5)$.  
   **(c)** $P(3 \le X \le 5)$ = ________

---

## Answer

| | Answer |
|---|--------|
| **(a)** | $5$ |
| **(b)** | $3$ |
| **(c)** | $\dfrac{1}{3}$ |

---

## Explanation

**Step 1.**  
The PDF of a continuous uniform random variable on $[a,\, b]$ is

$$
f_X(x) = \begin{cases} \dfrac{1}{b-a}, & \text{if } a \le x \le b \\ 0, & \text{otherwise} \end{cases}
$$

Here $a = 2$, $b = 8$, so $f_X(x) = \dfrac{1}{6}$.

**Step 2.**  
Using the mean formula $E[X] = \dfrac{a+b}{2}$:

$$
E[X] = \frac{2 + 8}{2} = 5
$$

**Step 3.**  
Using the variance formula $\text{var}(X) = \dfrac{(b-a)^2}{12}$:

$$
\text{var}(X) = \frac{(8-2)^2}{12} = \frac{36}{12} = 3
$$

**Step 4.**  
Using the area interpretation $P(a \le X \le b) = \displaystyle\int_a^b f_X(x)\,dx$:

$$
P(3 \le X \le 5) = \int_3^5 \frac{1}{6}\,dx = \frac{2}{6} = \frac{1}{3}
$$
