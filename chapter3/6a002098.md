# Problem 1 (Easy)

A continuous random variable $X$ has the probability density function

$$
f_X(x)=
\begin{cases}
2x, & 0 \le x \le 1 \\
0, & \text{otherwise}
\end{cases}
$$

Task: Find $P\left(\frac{1}{2} \le X \le 1\right)$.

---

# Solution

We use the definition of probability for a continuous random variable:

$$
P(a \le X \le b)=\int_a^b f_X(x)\,dx
$$

Thus,

$$
P\left(\frac{1}{2} \le X \le 1\right)
=
\int_{1/2}^{1} 2x \, dx
$$

Compute the integral:

$$
\int 2x \, dx = x^2
$$

$$
P\left(\frac{1}{2} \le X \le 1\right)
=
\left[ x^2 \right]_{1/2}^{1}
=
1^2 - \left(\frac{1}{2}\right)^2
$$

$$
= 1 - \frac{1}{4}
= \frac{3}{4}
$$

Therefore,

$$
\boxed{P\left(\frac{1}{2} \le X \le 1\right)=\frac{3}{4}}
$$
