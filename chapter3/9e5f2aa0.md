# Continuous Random Variable Problem: Hard #14

## Transformation of Variables
Let $X$ be the time (in seconds) to compile a program, with the probability density function (PDF):

$$f_X(x) = 2e^{-2x}, \quad x \ge 0$$

The server compute cost $Y$ is proportional to the square of the compilation time, such that $Y = X^2$. Find the continuous probability density function of the cost, $f_Y(y)$.
