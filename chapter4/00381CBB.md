# Easy Problem

A mobile game company is testing a new “reward box” feature.  
When a player opens the app, the loading time before the reward box appears is random.

Let X denote the loading time (in seconds).  
Assume that X is uniformly distributed on the interval [0,4].

To measure user satisfaction, the company defines an excitement score Y by

Y = √X.

The company wants to understand how the excitement score is distributed among users.

Find the PDF of Y.
