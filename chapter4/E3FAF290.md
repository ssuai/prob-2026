# Problem 3 (Hard)

Let $X$ and $Y$ be two independent continuous random variables. 
$X$ is uniformly distributed over the interval $[0, 1]$, so $f_X(x) = 1$ for $0 \le x \le 1$.
$Y$ is uniformly distributed over the interval $[0, 2]$, so $f_Y(y) = 1/2$ for $0 \le y \le 2$.

Define a new random variable $Z$ as the sum of $X$ and $Y$:
$Z = X + Y$

Derive the complete Probability Density Function $f_Z(z)$ using the convolution formula:
$f_Z(z) = \int_{-\infty}^{\infty} f_X(x)f_Y(z-x)dx$

*Hint: You will need to evaluate the integral by carefully defining the overlapping regions. The final PDF will be a piecewise function with three distinct valid intervals for $z$.*
