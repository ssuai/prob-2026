Problem

A communication system sends a binary signal.
A transmitted bit is 1 with probability 0.6, and 0 with probability 0.4.
If 1 is transmitted, the receiver correctly receives 1 with probability 0.9.
If 0 is transmitted, the receiver incorrectly receives 1 with probability 0.15.

Let
A= {the transmitted bit is 1}
B= {the received bit is 1}

Find the following:
1. P(B)
2. P(A∩B)
3. P(A∣B)

Use the total probability theorem and Bayes’ rule.
