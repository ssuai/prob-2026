# Problem 2: Covariance and Correlation of Algorithm Performance

### [Scenario: Execution Time vs. Memory Usage]
An AI software engineer is analyzing the relationship between the execution time ($X$, in seconds) and the memory usage ($Y$, in GB) of a new machine learning algorithm. Based on historical log data, the following statistical properties were observed:

* $E[X] = 5, E[Y] = 10$
* $E[XY] = 52$
* $var(X) = 4, var(Y) = 9$

### [Requirements]
1. Calculate the **covariance** $cov(X, Y)$ between execution time and memory usage.
2. Find the **correlation coefficient** $\rho(X, Y)$ and interpret the strength and direction of the relationship between the two variables.
