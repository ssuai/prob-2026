# Problem 1 — Easy

**Topic:** Conditional PDF and Conditional Expectation
**Source:** `43_conditional_expectation.pdf` (Section 4-3)
**Style:** \[Short Answer\]

---

## Scenario

Let $X$ and $Y$ be continuous random variables with joint PDF:

$$f_{X,Y}(x, y) = 4xy, \quad 0 \leq x \leq 1,\ 0 \leq y \leq 1$$

(zero otherwise). It is given that the marginal PDF of $Y$ is:

$$f_Y(y) = 2y, \quad 0 \leq y \leq 1$$

---

## Questions

**(a)** Using the conditional PDF definition (slide 2):

$$f_{X \mid Y}(x \mid y) = \frac{f_{X,Y}(x,\, y)}{f_Y(y)}$$

find $f_{X \mid Y}(x \mid y)$ for $0 \leq x \leq 1$.

$$f_{X \mid Y}(x \mid y) = \underline{\hspace{80pt}}$$

---

**(b)** Using the conditional expectation definition (slide 3):

$$E[X \mid Y = y] = \int_{-\infty}^{\infty} x \cdot f_{X \mid Y}(x \mid y)\, dx$$

compute $E[X \mid Y = 1/2]$.

$$E\!\left[X \;\middle|\; Y = \tfrac{1}{2}\right] = \underline{\hspace{60pt}}$$

---

## Answer

**(a)** $f_{X \mid Y}(x \mid y) = 2x, \quad 0 \leq x \leq 1$

**(b)** $E\!\left[X \mid Y = \tfrac{1}{2}\right] = \dfrac{2}{3}$

---

## Explanation

**Step 1 — Part (a): Apply the conditional PDF definition (slide 2).**

Substitute the joint PDF and marginal PDF directly:

$$f_{X \mid Y}(x \mid y) = \frac{f_{X,Y}(x, y)}{f_Y(y)} = \frac{4xy}{2y} = \boxed{2x}, \quad 0 \leq x \leq 1$$

> Notice: the $y$ cancels completely, so $f_{X|Y}(x|y) = 2x$ does **not** depend on $y$.  
> This means the conditional distribution of $X$ given $Y = y$ is the same for all $y$ — which is the signature of **independence** between $X$ and $Y$.

**Step 2 — Part (b): Apply the conditional expectation definition (slide 3).**

Since $f_{X|Y}(x|y) = 2x$ does not depend on $y$, the value $Y = 1/2$ has no effect:

$$E\!\left[X \;\middle|\; Y = \tfrac{1}{2}\right] = \int_0^1 x \cdot 2x\, dx = 2\int_0^1 x^2\, dx = 2 \cdot \left[\frac{x^3}{3}\right]_0^1 = 2 \cdot \frac{1}{3} = \boxed{\frac{2}{3}}$$

> **Check via the Law of Total Expectation (slide 4):** Since $E[X|Y] = 2/3$ is a constant, $E[E[X|Y]] = 2/3 = E[X]$.  
> We can verify: $E[X] = \int_0^1 x \cdot 2x\, dx = 2/3$. ✓
