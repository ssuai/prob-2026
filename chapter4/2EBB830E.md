# Problem 3: Moment Generating Functions and Conditional Estimation

### [Scenario: Network Packet Loss Prediction]
In a network node, the number of lost packets $X$ follows a **Poisson distribution** with parameter $\lambda$. The Moment Generating Function (MGF) of $X$ is known to be $M_X(s) = e^{\lambda(e^s - 1)}$. In a dynamic network environment, the parameter $\lambda$ is not constant but depends on the current network load $Y$.

### [Requirements]
1. Use the given MGF, $M_X(s)$, to prove that the expected number of lost packets $E[X]$ is equal to $\lambda$. (Hint: Use the property $\frac{d}{ds}M_X(s)|_{s=0} = E[X]$)
2. Suppose the conditional expectation of $X$ given the load $Y$ is $E[X|Y] = 2Y$. If the network load $Y$ is **uniformly distributed** on the interval $[1, 3]$, calculate the total expected packet loss $E[X]$ using the **Law of Iterated Expectations**.
