# Hard Problem
 
Let $X$ and $Y$ be independent random variables such that
 
$$X \sim \text{Poisson}(2),$$
 
$$Y \sim \text{Poisson}(3).$$
 
Define
 
$$Z = 2X + Y.$$
 
Using moment generating functions, answer the following questions.
 
1. Find the moment generating function $M_Z(s)$.
2. Use $M_Z(s)$ to find $E[Z]$ and $\text{Var}(Z)$.
3. Determine whether $Z$ follows a Poisson distribution.
4. Justify your answer by comparing $M_Z(s)$ with the MGF of a Poisson random variable.
