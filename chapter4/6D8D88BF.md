# Exercise 3: Derived Distributions (Linear Transformation)
**Level:** Hard
**Objective:** Prove the distribution of a linear function of a normal random variable using the PDF transformation formula.

## Problem
Suppose $X$ is a normal random variable such that $X \sim N(\mu, \sigma^2)$. A new random variable $Y$ is defined as $Y = aX + b$, where $a \neq 0$.

Using the derived distribution formula for linear functions, prove that $Y$ is also normally distributed: $Y \sim N(a\mu + b, a^2\sigma^2)$.

*Hint: The PDF of $X$ is $f_X(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$.*

---

## Solution
1. **Transformation Formula:** For a linear function $Y = aX + b$, the PDF of $Y$ is:
   $$f_Y(y) = \frac{1}{|a|} f_X\left(\frac{y-b}{a}\right)$$
2. **Substitution:** Plug the expression $\frac{y-b}{a}$ into the PDF of $X$:
   $$f_Y(y) = \frac{1}{|a| \sqrt{2\pi}\sigma} e^{-\frac{\left(\frac{y-b}{a} - \mu\right)^2}{2\sigma^2}}$$
3. **Simplifying the Exponent:**
   $$\frac{y-b}{a} - \mu = \frac{y - b - a\mu}{a} = \frac{y - (a\mu + b)}{a}$$
   Squaring this gives:
   $$\left(\frac{y - (a\mu + b)}{a}\right)^2 = \frac{(y - (a\mu + b))^2}{a^2}$$
4. **Final PDF:** Substitute back into the function:
   $$f_Y(y) = \frac{1}{\sqrt{2\pi}(|a|\sigma)} e^{-\frac{(y - (a\mu + b))^2}{2(a^2\sigma^2)}}$$
5. **Conclusion:** This result is exactly the PDF of a Normal Distribution with mean $a\mu + b$ and variance $a^2\sigma^2$. Thus, $Y \sim N(a\mu + b, a^2\sigma^2)$.
