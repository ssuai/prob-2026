# Problem 3: MGF Derivation and Moment Calculation

**Problem Statement:**
Let $X$ be a continuous random variable representing the time until a specific event occurs, modeled by an Exponential distribution with a rate parameter $\lambda > 0$. The probability density function (PDF) of $X$ is given by:
$$f_X(x) = \begin{cases} \lambda e^{-\lambda x} & \text{for } x \ge 0 \\ 0 & \text{for } x < 0 \end{cases}$$

**Your Task:**
Calculate the Variance $\text{Var}(X)$
