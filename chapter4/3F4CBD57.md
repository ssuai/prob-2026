### [Medium Level] Question 2: Dual-Core CPU Utilization (20 Points)

Consider a dynamic computing environment where the continuous random variables $X$ and $Y$ represent the CPU utilization of Core A and Core B in a dual-core processor, respectively. Both utilizations are measured on a scale from 0 to 1. The joint probability density function (Joint PDF) of the two cores is given by:

$$f_{X,Y}(x,y) = \begin{cases} x + y, & \text{for } 0 \le x \le 1 \text{ and } 0 \le y \le 1 \\ 0, & \text{otherwise} \end{cases}$$

*(For your convenience, assume the marginal PDFs have already been calculated in a previous step: $f_X(x) = x + 0.5$ for $0 \le x \le 1$, and $f_Y(y) = y + 0.5$ for $0 \le y \le 1$.)*

**(2-a)** [Expectations] Set up and evaluate the integrals to calculate the expected utilizations $E[X]$ and $E[Y]$. Then, set up and evaluate the double integral to calculate the joint expectation $E[XY]$. 

**(2-b)** [Covariance] Using your results from (2-a), calculate the Covariance between the CPU utilizations of the two cores, $Cov(X,Y)$.

**(2-c)** [Independence vs. Correlation] Based strictly on your calculated Covariance in (2-b), are the utilizations of Core A and Core B mathematically independent? Justify your answer by explicitly explaining the statistical relationship—and the classic conceptual trap—between independence and covariance.
