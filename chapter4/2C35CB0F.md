## 1. Easy: Basics of Variable Transformation
This problem uses the **'Two-step process'** emphasized by the professor in class.
The input $X$ of a system follows a **Uniform Distribution** between $[0, 1]$. The output of the system is defined as $Y = 2X + 3$.

(a) Find the Cumulative Distribution Function (CDF) $F_Y(y)$ of $Y$ using the PDF of $X$.
(b) Derive the PDF $f_Y(y)$ of $Y$ by differentiating the CDF obtained above.
