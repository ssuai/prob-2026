## Problem 1 (Easy)
A small delivery robot records how many successful drop-offs it completes during a short test run. Let $X$ be the number of successful drop-offs in one run.
- Possible values of $X$: $0, 1, 2$
- Probability mass function: $p_X(0)=0.25$, $p_X(1)=0.50$, and $p_X(2)=0.25$
- Use the moment generating function $M_X(s)=E[e^{sX}]$.

Calculate the **moment generating function $M_X(s)$** and use it to find **$E[X]$**.
