## 3. [Hard] Non-linear Transformation, PDF, and Moment Generating Function

Let $X$ be a continuous random variable with the probability density function (PDF) $f_X(x) = 2x$ for $0 < x < 1$, and $0$ otherwise. Consider a strictly monotonic transformation $Y = -\ln(X)$.
1) Using the theorem for strictly monotonic functions ($f_Y(y) = |h'(y)|f_X(h(y))$ where $X = h(Y)$), find the exact PDF of $Y$, $f_Y(y)$.
2) Using the integral definition of the Moment Generating Function (MGF) for a continuous random variable, find $M_Y(s)$.
3) Calculate the expected value $E[Y]$ by differentiating $M_Y(s)$ and evaluating it at $s=0$.
