# Problem 3 (Hard) — MGF, Conditional Expectation, and Derived Distribution

## Setup

The number of messages $N$ a user sends in one hour follows a **Poisson** distribution with a *random* rate $\Lambda$, where $\Lambda$ is itself uniformly distributed on $[1, 3]$:

$$\Lambda \sim \text{Uniform}(1, 3), \qquad N \mid \Lambda = \lambda \sim \text{Poisson}(\lambda).$$

## Find

### Part A — Conditional MGF
Write down the MGF of $N$ given $\Lambda = \lambda$, i.e., $M_{N|\Lambda}(s \mid \lambda)$.

*(Recall: for $X \sim \text{Poisson}(\lambda)$, $M_X(s) = e^{\lambda(e^s - 1)}$.)*

### Part B — Law of Total Expectation
Using the result $E[N \mid \Lambda] = \Lambda$, compute $E[N]$.

### Part C — Variance via Conditional Variance Formula
It is known that

$$\text{var}(N) = E[\text{var}(N \mid \Lambda)] + \text{var}(E[N \mid \Lambda]).$$

Given that $\text{var}(N \mid \Lambda) = \Lambda$ (Poisson property), compute $\text{var}(N)$.

### Part D — Derived Distribution of the Observed Rate
Define the **normalized observation** $Y = N / \bar{\lambda}$, where $\bar{\lambda} = E[\Lambda] = 2$.  
Suppose we condition on $\Lambda = 2$ (so $N \mid \Lambda=2 \sim \text{Poisson}(2)$) and approximate $N$ as a **continuous** random variable with PDF

$$f_N(n) = \frac{2^n e^{-2}}{n!}, \quad n \ge 0 \quad \text{(treat as continuous for this part)}.$$

Using the monotone-function (linear) formula, find $f_Y(y)$ in terms of $f_N$.

### Part E — Reflection
In Part C, which term — $E[\text{var}(N|\Lambda)]$ or $\text{var}(E[N|\Lambda])$ — is larger, and what does this mean intuitively about the source of variability in $N$?

> **Key formulas needed:**
> - $E[\Lambda] = 2$, $\text{var}(\Lambda) = \frac{(3-1)^2}{12} = \frac{1}{3}$
> - $E[\Lambda^2] = \text{var}(\Lambda) + (E[\Lambda])^2$
> - Conditional variance law: $\text{var}(X) = E[\text{var}(X|Y)] + \text{var}(E[X|Y])$
