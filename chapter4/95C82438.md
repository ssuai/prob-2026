# Hard Problem

A smart factory uses an automated sensor system to monitor machine performance.

Let X denote the operating temperature of a machine.  
Assume

X ~ N(2,1).

The factory defines a transformed stability score by

Y = 3X + 1.

Another independent noise variable Z is defined by

Z ~ N(0,4).

The final observed signal is

W = Y + Z.

The engineering team wants to analyze the statistical properties of the system.

Answer the following questions.

1. Find the PDF of Y.
2. Compute cov(X,Y).
3. Find E[X | Y = 7].
4. Find the moment generating function M_W(s).
5. Using the MGF, identify the distribution of W.
