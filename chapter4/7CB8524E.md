# Problem 1 (Easy) — Derived Distribution: Linear Transformation

## Given

Let $X \sim \text{Exponential}(\lambda)$ with parameter $\lambda = 2$, so that

$$f_X(x) = 2e^{-2x}, \quad x \geq 0.$$

Define a new random variable

$$Y = 3X + 1.$$

## Find

1. The PDF $f_Y(y)$ of $Y$.
2. The mean $E[Y]$ and variance $\text{var}(Y)$.

> **Hint:** Use the linear-function formula for derived distributions:
> $$f_Y(y) = \frac{1}{|a|} f_X\!\left(\frac{y - b}{a}\right),$$
> and recall that for $X \sim \text{Exponential}(\lambda)$: $E[X] = \frac{1}{\lambda}$ and $\text{var}(X) = \frac{1}{\lambda^2}$.
