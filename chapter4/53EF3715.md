# Problem 3 (Hard): Moment Generating Function

A discrete random variable \(X\) has probability mass function

\[
p_X(x)=
\begin{cases}
\frac{1}{4}, & x=0,\\
\frac{1}{2}, & x=2,\\
\frac{1}{4}, & x=4,\\
0, & \text{otherwise.}
\end{cases}
\]

Let \(M_X(s)\) be the moment generating function of \(X\).

Find:

1. \(M_X(s)\)
2. \(E[X]\) using \(M_X(s)\)
3. \(E[X^2]\) using \(M_X(s)\)
4. \(\operatorname{var}(X)\)
