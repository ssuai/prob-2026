## Problem: PDF of a Transformed Random Variable

Let \(X\) be a continuous random variable uniformly distributed on \([0,1]\). Define

\[
Y = 1 - X^2.
\]

Answer the following questions.

### Questions

1. Find the support of \(Y\).
2. Find the cumulative distribution function \(F_Y(y)\) for all real numbers \(y\).
3. Find the probability density function \(f_Y(y)\).
4. Compute

\[
P\left(Y \ge \frac{3}{4}\right).
\]

---

### Notes

You may use the CDF method:

\[
F_Y(y)=P(Y\le y).
\]
