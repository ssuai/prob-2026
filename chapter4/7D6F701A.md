Intermediate Level: Conditional Expectation Calculation
Concept: Evaluating Conditional Expectation $E[X|Y=y]$ for a specific value.
Problem: In a machine learning model, the time it takes to train the model ($X$, in minutes) depends heavily on the number of hidden layers in the neural network ($Y$). The relationship is defined by the following conditional expectation formula:$E[X|Y] = 3Y^2 + 5Y + 10$If you design a neural network with exactly $Y = 4$ hidden layers, calculate the expected training time.
