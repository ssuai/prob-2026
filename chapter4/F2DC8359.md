# Problem 2 (Intermediate)

**Topic: Covariance and Correlation**

A data center's monitoring system continuously tracks two resource metrics for a single server:

- $X$ : fraction of CPU capacity in use $(\,0 \le X \le 1\,)$
- $Y$ : fraction of RAM capacity in use $(\,0 \le Y \le 1\,)$

From six months of operational logs, the joint behavior of the two metrics is modeled by the joint PDF:

$$f_{X,Y}(x,\,y) = x + y, \quad 0 \le x \le 1,\ 0 \le y \le 1$$

The infrastructure team wants to know whether CPU and RAM usage are correlated, in order to decide whether a single composite alert threshold is sufficient.

**(2-a)** [Marginal PDFs] Derive the marginal PDFs $f_X(x)$ and $f_Y(y)$.

**(2-b)** [Expectations] Compute $E[X]$, $E[Y]$, and $E[XY]$.

**(2-c)** [Covariance] Calculate $\text{Cov}(X,Y) = E[XY] - E[X]\,E[Y]$.

**(2-d)** [Correlation] Compute $\text{Var}(X)$ and $\text{Var}(Y)$, then find the correlation coefficient:
$$\rho(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\,\text{Var}(Y)}}$$

**(2-e)** [Independence] The team assumes $X$ and $Y$ are independent to simplify their alerting model. Is this assumption valid? Justify your answer using **two** distinct arguments.
