Hard Level: Derived Distribution via Transformation Formula
Concept: Finding the PDF of $Y = g(X)$ using the direct transformation formula: $f_Y(y) = f_X(x) \cdot \left| \frac{dx}{dy} \right|$
Problem: A network router receives packets. The processing time for a single packet is a random variable $X$ with an exponential distribution. Its Probability Density Function (PDF) is:$f_X(x) = 2e^{-2x}$ (for $x > 0$)To add a security encryption step, the new processing time $Y$ is a linear transformation of the original time: $Y = 3X + 1$.Using the transformation formula directly (without integrating the CDF), calculate the new PDF, $f_Y(y)$.
