# Exercise 2: Moment Generating Functions (MGF)
**Level:** Intermediate
**Objective:** Calculate the expected value of a discrete random variable using its MGF.

## Problem
A discrete random variable $Y$ has the following moment generating function:
$$M_Y(s) = \frac{1}{4}e^{-s} + \frac{1}{2} + \frac{1}{8}e^{4s} + \frac{1}{8}e^{5s}$$

Find the expected value $E[Y]$ by differentiating the moment generating function.

---

## Solution
We can find the moments of a random variable by taking derivatives of the MGF and evaluating them at $s = 0$.
1. **Relationship:** The first moment (expected value) is given by $E[Y] = \frac{d}{ds}M_Y(s) \big|_{s=0}$.
2. **Differentiation:** Compute the first derivative of $M_Y(s)$ with respect to $s$:
   $$\frac{d}{ds}M_Y(s) = -\frac{1}{4}e^{-s} + 0 + \frac{4}{8}e^{4s} + \frac{5}{8}e^{5s}$$
3. **Evaluation:** Set $s = 0$ in the derivative:
   $$E[Y] = -\frac{1}{4}e^{0} + \frac{1}{2}e^{0} + \frac{5}{8}e^{0}$$
   $$E[Y] = -\frac{1}{4} + \frac{1}{2} + \frac{5}{8}$$
4. **Final Result:** Convert to a common denominator (8):
   $$E[Y] = -\frac{2}{8} + \frac{4}{8} + \frac{5}{8} = \frac{7}{8}$$
