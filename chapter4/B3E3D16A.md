## [문제 2] 적률생성함수(MGF)의 미분과 적률 (난이도: 중간)

**주제:** 적률생성함수의 정의 및 미분을 통한 적률 도출

**문제:**

이산 확률변수 $X$의 적률생성함수가 다음과 같이 주어졌습니다.

$$M_X(s) = \frac{1}{4}e^{-s} + \frac{1}{2}e^{2s} + \frac{1}{4}e^{4s}$$

적률생성함수의 성질 $\frac{d^n}{ds^n}M_X(s)|_{s=0} = E[X^n]$을 이용하여, 이 확률변수의 기댓값 $E[X]$와 분산 $var(X)$를 구하시오.

**풀이:**

1.  **기댓값 $E[X]$ 계산:**
    
    -   $M_X(s)$를 $s$에 대해 1차 미분합니다: $M_X'(s) = -\frac{1}{4}e^{-s} + e^{2s} + e^{4s}$.
        
    -   $s=0$을 대입하여 $E[X]$를 구합니다: $E[X] = M_X'(0) = -\frac{1}{4}(1) + 1 + 1 = \frac{7}{4}$.
        
2.  **$E[X^2]$ 계산:**
    
    -   2차 미분을 수행합니다: $M_X''(s) = \frac{1}{4}e^{-s} + 2e^{2s} + 4e^{4s}$.
        
    -   $s=0$을 대입합니다: $E[X^2] = M_X''(0) = \frac{1}{4}(1) + 2 + 4 = \frac{25}{4}$.
        
3.  **분산 $var(X)$ 계산:**
    
    -   $var(X) = E[X^2] - (E[X])^2$ 공식을 적용합니다.
        
    -   $var(X) = \frac{25}{4} - (\frac{7}{4})^2 = \frac{100}{16} - \frac{49}{16} = \frac{51}{16}$.
        

**정답:** $E[X] = \frac{7}{4}$, $var(X) = \frac{51}{16}$
