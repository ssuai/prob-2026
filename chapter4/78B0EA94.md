Easy Level: Covariance Calculation
Concept: Calculating Covariance using the formula: $\text{Cov}(X,Y) = E[XY] - E[X]E[Y]$
Problem: You are monitoring a database server. Let $X$ be the average CPU utilization (%) and $Y$ be the average Memory utilization (%). From your server logs, you have calculated the following expected values:
$E[X] = 60$$E[Y] = 70$The expected value of their product is $E[XY] = 4300$
Using the covariance formula, calculate the exact covariance, $\text{Cov}(X,Y)$.
