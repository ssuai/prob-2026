**Problem**

🇰🇷
$X_1, X_2, \ldots, X_n$은 서로 독립이며 동일한 지수분포 $\text{Exp}(\lambda)$ ($\lambda > 0$)를 따른다. 즉 각각의 PDF는

$$f_{X_i}(x) = \lambda e^{-\lambda x}, \quad x \ge 0.$$

$S_n = X_1 + X_2 + \cdots + X_n$으로 정의한다.

**(a)** $X_i$ 하나의 적률생성함수(moment generating function) $M_{X_i}(s)$를 구하여라. 단, $s < \lambda$.

**(b)** $S_n$의 적률생성함수 $M_{S_n}(s)$를 구하고, 이를 이용하여 $S_n$의 분포를 밝혀라.

**(c)** $M_{S_n}(s)$를 미분하여 $E[S_n]$과 $\text{Var}(S_n)$을 구하여라. 결과가 $E[S_n] = n/\lambda$, $\text{Var}(S_n) = n/\lambda^2$임을 확인하여라.

**(d)** [함정 문제] 다음 주장을 평가하여라:
> "$X_1 \sim \text{Exp}(\lambda)$이고 $X_2 \sim \text{Exp}(\lambda)$이므로, $S_2 = X_1 + X_2$도 지수분포를 따른다."

이 주장이 옳은지 틀린지 MGF를 이용하여 판별하고, $S_2$의 정확한 분포를 밝혀라.

🇺🇸
Let $X_1, X_2, \ldots, X_n$ be independent and identically distributed with the exponential distribution $\text{Exp}(\lambda)$, $\lambda > 0$, so that each has PDF

$$f_{X_i}(x) = \lambda e^{-\lambda x}, \quad x \ge 0.$$

Define $S_n = X_1 + X_2 + \cdots + X_n$.

**(a)** Find the moment generating function $M_{X_i}(s)$ for a single $X_i$, valid for $s < \lambda$.

**(b)** Derive $M_{S_n}(s)$ and use it to identify the distribution of $S_n$.

**(c)** Differentiate $M_{S_n}(s)$ to compute $E[S_n]$ and $\text{Var}(S_n)$, and verify that $E[S_n] = n/\lambda$ and $\text{Var}(S_n) = n/\lambda^2$.

**(d)** [Trap] Evaluate the following claim:
> "Since $X_1 \sim \text{Exp}(\lambda)$ and $X_2 \sim \text{Exp}(\lambda)$, the sum $S_2 = X_1 + X_2$ also follows an exponential distribution."

Use the MGF to determine whether this claim is correct or not, and identify the exact distribution of $S_2$.

---

**Solution**

- **Step 1 [Model Setup]:**
  확률 공간: $X_i$들은 독립이며 각각 $\text{Exp}(\lambda)$를 따르는 비음(non-negative) 연속 확률변수.
  적률생성함수(MGF)는 $M_X(s) = E[e^{sX}]$로 정의되며, 독립 합산에 대해

  $$M_{X_1 + X_2 + \cdots + X_n}(s) = \prod_{i=1}^n M_{X_i}(s)$$

  가 성립한다 (독립성의 핵심 활용).

- **Step 2 [Apply Formula]:**

  **(a)** 단일 $X_i$의 MGF:

  $$M_{X_i}(s) = E[e^{sX_i}] = \int_0^\infty e^{sx} \cdot \lambda e^{-\lambda x}\, dx = \lambda \int_0^\infty e^{-(\lambda - s)x}\, dx.$$

  $s < \lambda$이면 $\lambda - s > 0$이므로 적분이 수렴:

  $$M_{X_i}(s) = \lambda \cdot \frac{1}{\lambda - s} = \frac{\lambda}{\lambda - s}.$$

  **(b)** 독립성에 의해:

  $$M_{S_n}(s) = \prod_{i=1}^n \frac{\lambda}{\lambda - s} = \left(\frac{\lambda}{\lambda - s}\right)^n, \quad s < \lambda.$$

  이 MGF는 **Erlang 분포** (또는 **감마 분포** $\text{Gamma}(n, \lambda)$)의 MGF와 일치한다. MGF의 유일성(uniqueness)에 의해

  $$S_n \sim \text{Gamma}(n, \lambda).$$

- **Step 3 [Computation]:**

  **(c)** 모멘트 추출:

  $$M_{S_n}(s) = \left(\frac{\lambda}{\lambda-s}\right)^n = \lambda^n (\lambda-s)^{-n}.$$

  1계 미분:
  $$M_{S_n}'(s) = \lambda^n \cdot n(\lambda-s)^{-n-1}, \quad E[S_n] = M_{S_n}'(0) = \lambda^n \cdot n \cdot \lambda^{-(n+1)} = \frac{n}{\lambda}. \checkmark$$

  2계 미분:
  $$M_{S_n}''(s) = \lambda^n \cdot n(n+1)(\lambda-s)^{-n-2}, \quad E[S_n^2] = M_{S_n}''(0) = \frac{n(n+1)}{\lambda^2}.$$

  $$\text{Var}(S_n) = E[S_n^2] - (E[S_n])^2 = \frac{n(n+1)}{\lambda^2} - \frac{n^2}{\lambda^2} = \frac{n}{\lambda^2}. \checkmark$$

  **(d)** 함정 분석:

  $S_2$의 MGF는

  $$M_{S_2}(s) = \left(\frac{\lambda}{\lambda-s}\right)^2 = \frac{\lambda^2}{(\lambda-s)^2}.$$

  반면 지수분포 $\text{Exp}(\lambda)$의 MGF는 $\frac{\lambda}{\lambda-s}$이다.

  $\frac{\lambda^2}{(\lambda-s)^2} \ne \frac{\lambda}{\lambda-s}$ (모든 $s \ne 0$에 대해)이므로, **$S_2$는 지수분포를 따르지 않는다.**

  정확한 분포: $S_2 \sim \text{Gamma}(2, \lambda)$, PDF는

  $$f_{S_2}(x) = \lambda^2 x e^{-\lambda x}, \quad x \ge 0.$$

  (Erlang-2 분포라고도 함.)

- **Step 4 [Verification]:**
  $\text{Gamma}(n,\lambda)$의 이론값: $E = n/\lambda$, $\text{Var} = n/\lambda^2$. 위에서 MGF 미분으로 얻은 결과와 일치. ✓
  또한 $f_{S_2}(x) \ge 0$이고 $\int_0^\infty \lambda^2 x e^{-\lambda x}dx = 1$ (감마 적분). ✓

**Answer:**

**(a)** $M_{X_i}(s) = \dfrac{\lambda}{\lambda - s}$ for $s < \lambda$

**(b)** $M_{S_n}(s) = \left(\dfrac{\lambda}{\lambda-s}\right)^n$; $\quad S_n \sim \text{Gamma}(n, \lambda)$

**(c)** $E[S_n] = \dfrac{n}{\lambda}$, $\quad \text{Var}(S_n) = \dfrac{n}{\lambda^2}$

**(d)** 주장은 **틀렸다**. $S_2 \sim \text{Gamma}(2,\lambda)$이며 지수분포가 아니다.
