# [난이도: 하] 적률생성함수의 정의와 기댓값 계산

**문제:**
[cite_start]확률변수 $X$의 확률질량함수(PMF)가 다음과 같이 주어졌을 때, $X$의 적률생성함수 $M_X(s)$를 구하고, 이를 이용하여 기댓값 $E[X]$를 계산하시오. [cite: 54, 104]

* $P(X=2) = 1/2$
* $P(X=3) = 1/6$
* $P(X=5) = 1/3$

**요구사항:**
1. [cite_start]적률생성함수의 정의 식을 적으시오. [cite: 47]
2. [cite_start]유도된 $M_X(s)$를 $s$에 대해 미분하여 기댓값을 구하는 과정을 서술하시오. [cite: 105]
