# Problem 3 (Hard)

**Topic: Conditional Expectation and Law of Total Variance**

A content delivery network (CDN) routes each incoming request to one of three server tiers based on current load. Let $N$ denote the tier assigned to a request:

| Tier $N$ | Description | $P(N)$ |
|---|---|---|
| $N = 1$ | Edge server (lightest load) | $1/2$ |
| $N = 2$ | Regional server | $1/3$ |
| $N = 3$ | Origin server (heaviest load) | $1/6$ |

Once a request is routed to Tier $N = n$, its **service time** $T$ (seconds) is modeled as:

$$T \mid N = n \;\sim\; \text{Uniform}[\,0,\; 2n\,]$$

The network SLA requires that $P(T > 2)$ stays below a target threshold. Engineers also need $E[T]$ and $\text{Var}(T)$ to dimension the system's buffer.

**(3-a)** [Conditional Moments] For each tier $n \in \{1, 2, 3\}$, compute $E[T \mid N = n]$ and $\text{Var}(T \mid N = n)$.

**(3-b)** [Law of Total Expectation] Using the results from (3-a), compute the overall expected service time:
$$E[T] = E\bigl[E[T \mid N]\bigr]$$

**(3-c)** [Law of Total Variance] Compute $\text{Var}(T)$ by decomposing it into within-tier and between-tier components:
$$\text{Var}(T) = E\bigl[\text{Var}(T \mid N)\bigr] + \text{Var}\bigl(E[T \mid N]\bigr)$$

**(3-d)** [Total Probability] Compute $P(T > 2)$, the probability that a request violates the 2-second SLA threshold, using the Law of Total Probability:
$$P(T > 2) = \sum_{n=1}^{3} P(T > 2 \mid N = n)\, P(N = n)$$
