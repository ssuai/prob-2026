### [Hard Level - INTEGRATED CONCEPTS] Question 3: Dynamic Cloud Resource Allocation (30 Points)

An advanced dynamic cloud resource allocator processes incoming data batches. The size of an incoming data batch, denoted by $X$ (in GB), is uniformly distributed between 1 and 3. That is, $X \sim \text{Uniform}(1, 3)$. 

The algorithm's processing time, $Y$ (in seconds), heavily depends on the batch size $X$. Given a specific batch size $X = x$, the processing time $Y$ follows an Exponential distribution with a mean exactly equal to that batch size. Therefore, you are given the following conditional properties:
* $E[Y|X=x] = x$
* $Var(Y|X=x) = x^2$

**(3-a)** [Law of Iterated Expectations & Total Variance] Calculate the unconditional expected processing time $E[Y]$ and the unconditional variance of the processing time $Var(Y)$. You must explicitly state the Law of Iterated Expectations and the Law of Total Variance in your derivations.

**(3-b)** [Covariance via Conditioning] Calculate the Covariance between the batch size and the processing time, $Cov(X,Y)$. 
*(Hint: To find $E[XY]$, first condition on $X$ by utilizing the property $E[XY] = E[E[XY|X]]$.)*

**(3-c)** [Derived Distribution] The cloud computing provider charges a computational fee $C$ strictly based on the squared batch size, defined by the non-linear function $C = X^2$. Using the rigorous 2-step method for derived distributions, first find the Cumulative Distribution Function (CDF) $F_C(c)$, and then differentiate it to find the Probability Density Function (PDF) of the cost, $f_C(c)$. You must specify the valid numerical domain for $c$.
