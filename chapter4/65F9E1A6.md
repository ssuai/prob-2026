# Problem 1 (Easy)

**Topic: Derived Distributions — Monotonic Function**

A wireless base station receives signal power $X$ (in mW) from a mobile device. Due to environmental interference, $X$ is modeled as a **Uniform** random variable on $[1, 5]$ mW.

The base station passes the signal through a **power amplifier**, which outputs:

$$Y = 3X + 2 \quad \text{(mW)}$$

Engineers need to characterize the distribution of the output power $Y$ to verify it stays within the safe operating range of the amplifier.

**(1-a)** [Range] Determine the minimum and maximum values of $Y$.

**(1-b)** [Derived PDF] Using the monotonic-function formula
$$f_Y(y) = \left|\frac{dh(y)}{dy}\right| f_X(h(y)), \quad h = g^{-1}$$
find the PDF $f_Y(y)$ of the output power.

**(1-c)** [Verification] Confirm that $f_Y(y)$ is a valid PDF by checking $\displaystyle\int f_Y(y)\,dy = 1$.

**(1-d)** [Expected Output] Compute $E[Y]$ directly from $f_Y(y)$, and verify the result using linearity of expectation.
