# Problem (Easy) — Finding Moments Using MGF

A random variable $X$ has the following PMF:

$$p_X(x) = \begin{cases} 1/4 & \text{if } x = 1 \\ 1/2 & \text{if } x = 2 \\ 1/4 & \text{if } x = 3 \end{cases}$$

**(a)** Find the MGF $M_X(s)$.

**(b)** Use $M_X(s)$ to find $E[X]$.

**(c)** Use $M_X(s)$ to find $E[X^2]$, and then compute $\text{var}(X)$.
