2. intermediate

Let X be a discrete random variable with the following probability mass function:

P(X = 1) = 0.3
P(X = 2) = 0.4
P(X = 3) = 0.3

Given X = x, suppose the conditional expectation of another random variable Y is

E[Y | X = x] = 2x + 1

Also suppose that

E[Y] = 5

Answer the following questions:

1. Find E[X].
2. Find E[XY].
3. Find Cov(X, Y).
4. Determine whether X and Y are independent.
