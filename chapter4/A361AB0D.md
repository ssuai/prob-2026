## Problem 1: Linear Transformation of a Uniform Random Variable

Let $X$ be a continuous random variable uniformly distributed on the interval $[0, 2]$. 
A new random variable $Y$ is defined by the linear transformation:
$$Y = 3X + 1$$

1. Find the Cumulative Distribution Function (CDF) of $Y$, $F_Y(y)$.
2. Find the Probability Density Function (PDF) of $Y$, $f_Y(y)$.
3. Calculate the expected value $E[Y]$ using the derived PDF.
