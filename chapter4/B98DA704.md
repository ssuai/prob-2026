## [문제 3] 파생 분포와 조건부 기댓값의 정리 (난이도: 어려움)

**주제:** PDF 변환(Derived PDF) 및 반복 기댓값의 법칙(Law of Iterated Expectations)

**문제:**

확률변수 $X$가 구간 $[0, 2]$에서 균등분포(Uniform distribution)를 따릅니다 ($f_X(x) = 1/2, 0 \le x \le 2$). 새로운 확률변수 $Y = X^2$에 대하여 다음을 수행하시오.

1.  $Y$의 확률밀도함수 $f_Y(y)$를 구하시오.
    
2.  조건부 기댓값 $E[X|Y]$를 구하고, $E[E[X|Y]] = E[X]$가 성립함을 증명하시오.
    

**풀이:**

1.  **$f_Y(y)$ 도출:**
    
    -   $Y$의 누적분포함수(CDF)를 먼저 구합니다: $F_Y(y) = P(Y \le y) = P(X^2 \le y) = P(X \le \sqrt{y})$.
        
    -   $F_Y(y) = \int_0^{\sqrt{y}} \frac{1}{2} dx = \frac{\sqrt{y}}{2}$ (단, $0 \le y \le 4$).
        
    -   CDF를 미분하여 PDF를 구합니다: $f_Y(y) = \frac{d}{dy}(\frac{1}{2}y^{1/2}) = \frac{1}{4\sqrt{y}}$.
        
2.  **조건부 기댓값 정리 증명:**
    
    -   $Y=X^2$이고 $X \ge 0$이므로, $Y=y$가 주어지면 $X$는 $\sqrt{y}$로 유일하게 결정됩니다. 따라서 $E[X|Y=y] = \sqrt{y}$입니다.
        
    -   $E[E[X|Y]] = E[\sqrt{Y}]$를 계산합니다: $\int_0^4 \sqrt{y} \cdot f_Y(y) dy = \int_0^4 \sqrt{y} \cdot \frac{1}{4\sqrt{y}} dy = \int_0^4 \frac{1}{4} dy = 1$.
        
    -   원래 변수의 기댓값 $E[X]$를 계산합니다: $\int_0^2 x \cdot \frac{1}{2} dx = [\frac{x^2}{4}]_0^2 = 1$.
        
    -   결과적으로 $E[E[X|Y]] = E[X] = 1$이 되어 정리가 성립합니다.
        

**정답:** 1) $f_Y(y) = \frac{1}{4\sqrt{y}}$ ($0 < y \le 4$)

2) $E[E[X|Y]] = E[X] = 1$ 임을 수치적으로 확인.
