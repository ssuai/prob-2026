# Hard Problem: Conditional Expectation and Conditional Variance

A factory machine operates in two possible modes during production:

- Normal mode (\(Y=0\)) with probability \(0.7\)
- Overloaded mode (\(Y=1\)) with probability \(0.3\)

Let \(X\) denote the number of defective products produced in one hour.

The conditional distributions of \(X\) are:

- If \(Y=0\),
\[
X|Y=0 \sim \text{Poisson}(2)
\]

- If \(Y=1\),
\[
X|Y=1 \sim \text{Poisson}(6)
\]

---

1. Find the conditional expectations

\[
E[X|Y=0], \qquad E[X|Y=1].
\]

2. Find the conditional variances

\[
\operatorname{var}(X|Y=0), \qquad \operatorname{var}(X|Y=1).
\]

3. Compute the unconditional expectation

\[
E[X].
\]

(Hint: Use the law of total expectation.)

4. Let

\[
\hat X = E[X|Y]
\]

be the conditional expectation estimator of \(X\).

Find

\[
\operatorname{var}(\hat X).
\]

5. Compute

\[
E[\operatorname{var}(X|Y)].
\]

6. Compare the following two quantities and explain their meanings:

\[
\operatorname{var}(\hat X)
\]

and

\[
E[\operatorname{var}(X|Y)].
\]

Which quantity represents the variability explained by knowing the machine mode \(Y\), and which represents the remaining uncertainty?
