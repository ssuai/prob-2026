## Scenario

An engineer transmits a signal $X$, a discrete random variable with PMF:

| $x$        | 0   | 2   |
|------------|-----|-----|
| $p_X(x)$   | 1/2 | 1/2 |

The received measurement $Y$ is **independent** of $X$ (a completely noisy channel).
Because $X \perp Y$, the conditional expectation estimator (Section 4-3, slide 7):

$$\hat{X} = E[X \mid Y]$$

reduces to $\hat{X} = E[X]$ ‚Äî knowing $Y$ yields zero information about $X$.

The **estimation error** is defined as (Section 4-3, slide 7):

$$\tilde{X} = \hat{X} - X$$

---

## Questions

**(a)** Using the discrete MGF definition (Section 4-4, slide 4):

$$M_X(s) = E\!\left[e^{sX}\right] = \sum_x e^{sx}\, p_X(x)$$

compute $M_X(s)$ in closed form.

$$M_X(s) = \underline{\hspace{160pt}}$$

---

**(b)** Using the moment-generation property (Section 4-4, slides 10‚Äì11):

$$\left.\frac{d^n}{ds^n} M_X(s)\right|_{s=0} = E[X^n]$$

compute $E[X]$, $E[X^2]$, and $var(X) = E[X^2] - (E[X])^2$.

$$E[X] = \underline{\hspace{50pt}} \qquad E[X^2] = \underline{\hspace{50pt}} \qquad var(X) = \underline{\hspace{50pt}}$$

---

**(c)** This part requires three conditions to be verified in order before applying Example 4.25.

**Condition 1 ‚Äî Justify $\hat{X} = E[X]$.**
Since $X$ and $Y$ are independent, their conditional and marginal PMFs satisfy $p_{X|Y}(x|y) = p_X(x)$ for all $y$. Using the conditional expectation definition :

$\hat{X} = E[X]$ regardless of the observed value of $Y$.

**Condition 2 ‚Äî Substitute $E[X]$ from part (b) and write $\tilde{X}$ in the form $aX + b$.**
From part (b), $E[X] = 1$. The estimation error (Section 4-3, slide 7) is:

$$\tilde{X} = \hat{X} - X = 1 - X = (-1) \cdot X + 1$$

Identify the constants:

$$a = \underline{\hspace{30pt}}, \qquad b = \underline{\hspace{30pt}}$$

**Condition 3 ‚Äî Verify that Example 4.25 (Section 4-4, slide 7) applies.**
Example 4.25 states: if $\tilde{X} = aX + b$ and $M_X(s)$ exists, then:

$$M_{\tilde{X}}(s) = E\!\left[e^{s\tilde{X}}\right] = E\!\left[e^{s(aX+b)}\right] = e^{sb}\,E\!\left[e^{saX}\right] = e^{sb}\,M_X(sa)$$

Check: $M_X(s) = \dfrac{1}{2} + \dfrac{1}{2}e^{2s}$ exists for all $s \in \mathbb{R}$ (finite discrete distribution). ‚úì

Now apply the formula with $a = -1$, $b = 1$, and $M_X(s)$ from part (a) to find $M_{\tilde{X}}(s)$:

$$M_{\tilde{X}}(s) = \underline{\hspace{200pt}}$$

---

**(d)** Compute $var(\tilde{X})$ from $M_{\tilde{X}}(s)$ using the second derivative, then verify it satisfies the conditional variance relation :

$$var(\tilde{X}) = var(X) - var(\hat{X})$$

$$var(\tilde{X}) = \underline{\hspace{60pt}}$$

---

## Answer

**(a)** $M_X(s) = \dfrac{1}{2} + \dfrac{1}{2}e^{2s}$

**(b)** $E[X] = 1$ $\quad$ $E[X^2] = 2$ $\quad$ $var(X) = 1$

**(c)** $a = -1,\quad b = 1$ $\qquad$ $M_{\tilde{X}}(s) = \dfrac{1}{2}e^{s} + \dfrac{1}{2}e^{-s}$

**(d)** $var(\tilde{X}) = 1$ ‚úì ‚Äî consistent with $var(X) - var(\hat{X}) = 1 - 0$

---

## Explanation

**Step 1 ‚Äî Part (a): Apply the discrete MGF definition (Section 4-4, slide 4).**

$$M_X(s) = e^{s \cdot 0} \cdot \frac{1}{2} + e^{s \cdot 2} \cdot \frac{1}{2} = \boxed{\frac{1}{2} + \frac{1}{2}e^{2s}}$$

---

**Step 2 ‚Äî Part (b): Extract moments via differentiation (Section 4-4, slides 10‚Äì11).**

First derivative:

$$\frac{d}{ds} M_X(s) = e^{2s}$$

$$E[X] = \left.\frac{d}{ds} M_X(s)\right|_{s=0} = e^{0} = \boxed{1}$$

Second derivative:

$$\frac{d^2}{ds^2} M_X(s) = 2e^{2s}$$

$$E[X^2] = \left.\frac{d^2}{ds^2} M_X(s)\right|_{s=0} = 2e^{0} = \boxed{2}$$

Variance:

$$var(X) = E[X^2] - (E[X])^2 = 2 - 1^2 = \boxed{1}$$

---

**Step 3 ‚Äî Part (c): Three conditions, then apply Example 4.25 (Section 4-4, slide 7).**

**Condition 1 ‚Äî Justify $\hat{X} = E[X]$ from independence.**

Because $X$ and $Y$ are independent, their conditional and marginal PMFs satisfy:

$$p_{X|Y}(x \mid y) = p_X(x) \quad \text{for all } y$$

Substituting into the conditional expectation definition (Section 4-3, slide 3):

$$\hat{X} = E[X \mid Y = y] = \sum_x x\, p_{X|Y}(x \mid y) = \sum_x x\, p_X(x) = E[X] \quad \text{for all } y$$

So $\hat{X} = E[X]$ is a constant ‚Äî it does not depend on the observed value of $Y$.

**Condition 2 ‚Äî Substitute $E[X] = 1$ and write $\tilde{X}$ explicitly in the form $aX + b$.**

From part (b), $E[X] = 1$. The estimation error (Section 4-3, slide 7) becomes:

$$\tilde{X} = \hat{X} - X = 1 - X = \underbrace{(-1)}_{a} \cdot X + \underbrace{1}_{b}$$

Therefore: $a = -1$, $b = 1$.

**Condition 3 ‚Äî Verify that Example 4.25 applies.**

Example 4.25 (Section 4-4, slide 7) requires $M_X(s)$ to exist. Since $X$ is a finite discrete random variable, $M_X(s) = \dfrac{1}{2} + \dfrac{1}{2}e^{2s}$ exists for all $s \in \mathbb{R}$. ‚úì

The formula $M_{\tilde{X}}(s) = e^{sb}\,M_X(sa)$ follows directly from:

$$M_{\tilde{X}}(s) = E\!\left[e^{s\tilde{X}}\right] = E\!\left[e^{s(-X+1)}\right] = E\!\left[e^{s \cdot 1} \cdot e^{s \cdot (-1) \cdot X}\right] = e^{s \cdot 1}\,E\!\left[e^{(-s)X}\right] = e^{s}\,M_X(-s)$$

**Apply with $a = -1$, $b = 1$:**

$$M_{\tilde{X}}(s) = e^{s \cdot 1}\,M_X(s \cdot (-1)) = e^{s}\,M_X(-s)$$

Substitute $M_X(-s) = \dfrac{1}{2} + \dfrac{1}{2}e^{-2s}$:

$$M_{\tilde{X}}(s) = e^{s}\!\left(\frac{1}{2} + \frac{1}{2}e^{-2s}\right) = \boxed{\frac{1}{2}e^{s} + \frac{1}{2}e^{-s}}$$


---

**Step 4 ‚Äî Part (d): Compute $var(\tilde{X})$ and verify the conditional variance relation.**

Second derivative of $M_{\tilde{X}}(s)$:

$$\frac{d^2}{ds^2} M_{\tilde{X}}(s) = \frac{1}{2}e^{s} + \frac{1}{2}e^{-s}$$

$$E[\tilde{X}^2] = \left.\frac{d^2}{ds^2} M_{\tilde{X}}(s)\right|_{s=0} = \frac{1}{2} + \frac{1}{2} = 1$$

$$var(\tilde{X}) = E[\tilde{X}^2] - (E[\tilde{X}])^2 = 1 - 0^2 = \boxed{1}$$

**Verification via Section 4-3, slide 9:**

Since $\hat{X} = E[X] = 1$ is a **constant**, $var(\hat{X}) = 0$. Therefore:

$$var(\tilde{X}) = var(X) - var(\hat{X}) = 1 - 0 = 1 \qquad \checkmark$$

> **Key synthesis:** The MGF (Section 4-4) served as the primary computational engine ‚Äî extracting all moments of $X$ and $\tilde{X}$ purely through differentiation. The conditional expectation estimator framework (Section 4-3) then provided two independent checks: the zero-mean property $E[\tilde{X}]=0$ and the variance decomposition $var(\tilde{X}) = var(X) - var(\hat{X})$. Both were confirmed exactly by the MGF results.
