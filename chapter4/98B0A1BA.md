Problem

Two independent random variables X and Y each follow the same discrete uniform distribution on the set
{1,2,3,4,5}.

Define a new random variable
Z=max(X,Y).

Find the following:
1.The cumulative distribution function Fz(k)=P(Z≤k)
2. The probability mass function pz(k)
3. The expectation E[Z]

Your final answers should be expressed exactly.
