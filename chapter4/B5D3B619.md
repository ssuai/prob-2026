# Problem 1 (Easy)

Let $X$ be a discrete random variable with the following probability mass function (PMF):
- $P(X = 1) = 1/4$
- $P(X = 2) = 1/2$
- $P(X = 4) = 1/4$

**Question:** Find the moment generating function (MGF) $M_X(s)$ for the random variable $X$. Using this MGF, calculate the expected value $E[X]$.
