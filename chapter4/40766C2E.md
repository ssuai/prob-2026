Question 3:
In a distributed cloud computing environment, a client sends data packets to two independent server nodes, Node A and Node B. Let X and Y be the discrete random variables representing the number of data batches processed by Node A and Node B, respectively.

The Moment Generating Functions (MGFs) for X and Y are given as:
M_X(s) = (1/4)e^(-s) + (3/4)e^(2s)
M_Y(s) = (1/2) + (1/2)e^s

The cloud provider calculates the total billing cost C using the formula: C = 3(X + Y) + 2.

Using the properties of MGFs discussed in the lecture, derive the MGF of the total cost C, denoted as M_C(s). Then, use M_C(s) to find the exact probability that the billing cost C is exactly 11. You must show all step-by-step calculations, including the multiplication of MGFs and linear transformations.
