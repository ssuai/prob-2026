# **Problem 1 (Easy): Uncorrelated but Dependent Routing Metrics** 

# **Difficulty Level: Easy**

###### **\[Condition]**

###### Consider a localized edge computing network where load distribution is directed by a primary routing variable $X$. Due to randomized round-robin load balancing, $X$ is uniformly distributed over the continuous interval $\[-1, 1]$. A secondary monitoring service continuously records the memory cache allocation on the router, denoted by the random variable $Y$. The memory cache is strictly driven by the square of the routing metric, mathematically constrained as $Y = 3X^2$.

###### \[Problem]

###### Given the uniform distribution of the primary routing metric $X \\sim U\[-1,1]$, and the deterministic quadratic relationship of the memory cache $Y=3X^2$:

###### &#x09;1. Prove mathematically that the routing metric $X$ and the memory cache $Y$ 	are completely uncorrelated (i.e., calculate $\\text{cov}(X,Y)$ and show it equals 0).

###### &#x09;2. Provide a rigorous logical argument demonstrating that $X$ and $Y$ are statistically dependent, thereby proving that correlation is an insufficient metric for discovering non-linear relationships in system telemetry.
