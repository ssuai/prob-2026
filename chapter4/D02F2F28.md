# Problem 1 (Easy)

Let $X$ be a continuous random variable that is uniformly distributed over the interval $[0, 2]$. 
A new random variable $Y$ is defined by the linear transformation:
$Y = 3X - 1$

Find the Probability Density Function (PDF) of $Y$, denoted as $f_Y(y)$. Be sure to specify the valid range (support) for $y$.
