3. hard

Let X be a continuous random variable with probability density function

f_X(x) = 2x, 0 < x < 1
f_X(x) = 0, otherwise

Given X = x, suppose Y has a uniform distribution on the interval (0, x).

That is,

Y | X = x follows Uniform(0, x)

Answer the following questions:

1. Find E[Y | X].
2. Find E[Y].
3. Find E[XY].
4. Find Cov(X, Y).
5. Let Z = Y / X. Find the distribution of Z.
