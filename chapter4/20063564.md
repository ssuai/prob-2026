# Problem 3 (Hard): Conditional Expectation

The probability of heads $p$ is not fixed but is itself a random variable, uniformly distributed on $[0, 1]$. Given $p$, a coin is tossed $n = 3$ times independently, and $X$ denotes the number of heads. Using the law of total expectation $E[X] = E[E[X \mid p]]$, find $E[X]$. Then compute $E[X^2]$ by applying $E[X^2] = E[E[X^2 \mid p]]$ together with the conditional identity $E[X^2 \mid p] = \mathrm{Var}(X \mid p) + (E[X \mid p])^2$, and use the result to find $\mathrm{Var}(X)$.
