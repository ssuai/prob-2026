# Problem 1: Linear Transformation of a Sensor Value

### [Scenario: Sensor Data Calibration]
In a robotic sensing system, a raw distance sensor value is represented by a random variable $X$. The variable $X$ is **uniformly distributed** on the interval $[0, 1]$. To calibrate this sensor for a specific navigation task, a linear transformation $Y = 3X + 2$ is applied to the raw data.

### [Requirements]
1. Find the **Probability Density Function (PDF)** $f_Y(y)$ of the calibrated sensor value $Y$ using the linear transformation method.
2. Calculate the **expected value** $E[Y]$ by applying the linearity property of expectation.
