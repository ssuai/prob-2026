# Problem Set 3 — Hard

## Problem 1

Let $X \sim \mathrm{Uniform}(0, 2)$ and, given $X = x$, let $Y \mid X = x \sim \mathrm{Uniform}(0, x)$.

(1) Find $E[Y \mid X = x]$.

(2) Use the law of total expectation to find $E[Y]$.

(3) Find $\mathrm{var}(Y \mid X = x)$.

(4) Find $E[\mathrm{var}(Y \mid X)]$.

(5) Find $\mathrm{var}(E[Y \mid X])$, then use the law of total variance
$$\mathrm{var}(Y) = E[\mathrm{var}(Y \mid X)] + \mathrm{var}(E[Y \mid X])$$
to compute $\mathrm{var}(Y)$.
