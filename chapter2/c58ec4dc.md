Problem 1 — Easy

Topic: Discrete random variable, conditioning

A university library tracks whether students return borrowed calculators on time.
 Historically, 70% of students return a calculator on time.
 Let the random variable X represent the number of calculators, out of the next 4 borrowed, that are returned on time.

1. Identify the probability distribution of X including its parameters.
2. Find the probability that exactly 3 calculators are returned on time.
3. Find the probability that at least 2 calculators are returned on time.
4. Given that at least 1 calculator is returned late, find the conditional probability that exactly 2 calculators are returned on time.

      Assume each student’s return behavior is independent of the others.
