# Problem 1: Smart Bulbs (Difficulty: Easy)

We are testing smart bulbs in the lab. Each bulb has a 10% chance of failing (p = 0.1).

**Part A**
You test exactly 1 bulb. Let X = 1 if it fails, and X = 0 if it works. This is a Bernoulli random variable. Write the PMF of X.

**Part B**
Now, you test exactly 5 bulbs. Let Y be the total number of failures. This is a Binomial random variable. Write the PMF of Y. Then, calculate the expected value (E[Y]) and the variance (var[Y]).
