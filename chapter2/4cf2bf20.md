## Problem

Let $X$ be a discrete random variable with the following PMF:

$$p_X(x) = \frac{1}{5} \quad \text{for } x \in \{-2,\ -1,\ 0,\ 1,\ 2\}, \quad \text{and } 0 \text{ otherwise.}$$

1. Compute the expectation $E[X]$.

   **(a)** ________

2. Compute the variance $\text{var}(X)$ using the formula $\text{var}(X) = E[X^2] - (E[X])^2$.

   **(b)** ________

---

## Answer

**(a)**  
$$E[X] = 0$$

**(b)**  
$$\text{var}(X) = 2$$

---

## Explanation

**Step 1.** By the definition of expectation:

$$E[X] = \sum_x x \cdot p_X(x) = \frac{1}{5}(-2 + (-1) + 0 + 1 + 2) = \frac{1}{5}(0) = 0$$

**Step 2.** Compute the second moment $E[X^2]$:

$$E[X^2] = \sum_x x^2 \cdot p_X(x) = \frac{1}{5}(4 + 1 + 0 + 1 + 4) = \frac{1}{5}(10) = 2$$

**Step 3.** Apply the variance formula:

$$\text{var}(X) = E[X^2] - (E[X])^2 = 2 - 0^2 = 2$$
