# Easy Problem: Bernoulli Random Variable

### Situation: Festival Lucky Draw
Let's consider an experiment where you participate in a lucky draw at a festival. The probability of drawing a winning lot from the box is $0.15$.

---

* **experiment:** Draw one lot from the lucky draw box.
* **sample space:** $\Omega = \{ \text{Win}, \text{Lose} \}$
* **probability law:** $P(\text{Win}) = 0.15$
* **Bernoulli random variable:** $X = \text{Winning status}$ ($1$ if Win, $0$ if Lose)
    * $P(\{X=1\}) = 0.15$

---

### Question
Based on the situation above, complete the probability mass function (PMF) for this random variable $X$.
