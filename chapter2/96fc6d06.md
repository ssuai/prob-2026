# Problem 2 (Intermediate)

A communication system transmits digital packets independently.
The probability that a single packet transmission results in an error is p = 0.1.

(a) Let X be the number of transmissions until the first error occurs. Define the PMF pX(k) for this random variable.

(b) Suppose we repeat this experiment n times independently to obtain samples X1, X2, ..., Xn. Let Sn be the sample mean defined as Sn = (X1 + X2 + ... + Xn) / n. Find the exact value of the expectation E[Sn].

(c) Express the variance of the sample mean, var(Sn), as a function of n and p.
