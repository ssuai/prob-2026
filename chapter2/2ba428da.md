## 1. [난이도: 하] 이산 확률 변수의 기댓값
**주제: Probability Mass Functions & Expectation**

### 문제
이산 확률 변수 $X$의 확률 질량 함수(PMF)가 다음과 같이 주어졌을 때, 기댓값 $E[X]$를 구하세요.
- $P(X=0) = 0.3$
- $P(X=1) = 0.4$
- $P(X=2) = 0.3$

### 풀이
1. **기댓값의 정의**: $E[X] = \sum_{x} x \cdot p_X(x)$
2. **계산**:
   $$E[X] = (0 \times 0.3) + (1 \times 0.4) + (2 \times 0.3)$$
   $$E[X] = 0 + 0.4 + 0.6 = 1.0$$

**정답: 1.0**
