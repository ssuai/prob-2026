## Problem
A server transmits a data packet to a client over an unreliable network. Each transmission is independent, and the probability of a successful transmission is $p = 0.2$. If a transmission fails, the server immediately retransmits the packet until it succeeds. Let $X$ be the random variable representing the total number of transmissions needed for a successful delivery.
1) What type of random variable is $X$, and what is its probability mass function (PMF)?
2) Calculate the expected value $E[X]$ and the variance $Var[X]$.
