Problem 2: Query Success Rate in Distributed Databases
A single query is simultaneously sent to 10 independent nodes in a distributed system. Each node has an identical success probability of $p = 0.9$ for responding to the query.
1.Let $K$ be the random variable representing the number of successful nodes. State the name of the probability distribution of $K$ and write its Probability Mass Function (PMF).
2.The system is considered operational only if at least 9 nodes succeed. Formulate and calculate the probability that the system is operational (P(K >= 9)).
