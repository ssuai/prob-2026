### [Easy Level] Question 1: AI System Alert Costs (15 Points)

An artificial intelligence server automatically logs system alerts when it detects minor anomalies. Let the discrete random variable $X$ represent the number of system alerts generated per hour. Based on historical log data, the Probability Mass Function (PMF) of $X$, denoted as $p_X(x)$, is given in the following table:

| $x$ (Number of Alerts) | $0$ | $1$ | $2$ | $3$ |
| :--- | :---: | :---: | :---: | :---: |
| **$p_X(x)$** | $\frac{4}{10}$ | $\frac{3}{10}$ | $\frac{2}{10}$ | $\frac{1}{10}$ |

**(1-a)** [Discrete RV Basics] Using the provided PMF table, calculate the expected number of alerts per hour, $E[X]$, and the variance of the alerts, $Var(X)$. Show the step-by-step calculation for both.
