Topic: Conditioning and Total Expectation in Randomized Retry Algorithms

Problem:
A randomized algorithm for accessing a database succeeds with probability p. If it fails, it retries until it succeeds. Let X be the number of trials until the first success (a Geometric random variable). However, the system has a "Power-Save" mode (S=1) and a "High-Performance" mode (S=0).

In High-Performance mode (S=0), which occurs with probability 0.8, the success probability is p=0.5.

In Power-Save mode (S=1), which occurs with probability 0.2, the success probability is p=0.1.

Find the total expected number of trials E[X] the algorithm takes to succeed, given that the mode S is fixed before the trials begin. Show your derivation using the Total Expectation Theorem: E[X]=∑sP(S=s)E[X∣S=s].
