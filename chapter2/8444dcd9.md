# Problem 2: INTERMEDIATE

**Topic:** Discrete Random Variables  
**Style:** Scenario-based word problem

---

## Problem

A factory produces electronic chips. Each chip is tested and classified as either defective (D) or non-defective (N). Suppose a quality inspector tests 3 chips independently. Each chip has probability $p = 1/4$ of being defective and probability $1 - p = 3/4$ of being non-defective.

Define the random variable $X$ = number of defective chips out of the 3 tested.

1. Write out the full sample space $\Omega$ for this experiment, and identify which outcomes correspond to $X = 1$.

   **(a)** ________

2. Using the probability mass function of the binomial random variable, compute $P(X = 2)$.

   **(b)** ________

3. A second random variable $Y$ is defined as follows: $Y = 1$ if $X \geq 2$, and $Y = 0$ otherwise. What type of standard discrete random variable is $Y$? Write its PMF completely.

   **(c)** ________

---

## Answer

**(a)**  
$$\Omega = \{DDD,\ DDN,\ DND,\ DNN,\ NDD,\ NDN,\ NND,\ NNN\}$$

The outcomes where $X = 1$ are $\{DNN,\ NDN,\ NND\}$.

**(b)**  
$$P(X = 2) = \binom{3}{2} \left(\frac{1}{4}\right)^2 \left(\frac{3}{4}\right)^1 = 3 \cdot \frac{1}{16} \cdot \frac{3}{4} = \frac{9}{64}$$

**(c)**  
$Y$ is a **Bernoulli random variable**.

$$P(X = 3) = \binom{3}{3} \left(\frac{1}{4}\right)^3 \left(\frac{3}{4}\right)^0 = \frac{1}{64}$$

$$P(Y = 1) = P(X \geq 2) = \frac{9}{64} + \frac{1}{64} = \frac{10}{64} = \frac{5}{32}$$

$$P(Y = 0) = 1 - \frac{5}{32} = \frac{27}{32}$$

PMF:

$$p_Y(y) = \begin{cases} 5/32 & \text{if } y = 1 \\ 27/32 & \text{if } y = 0 \end{cases}$$

---

## Explanation

**Step 1.** The experiment is testing 3 independent chips, so the sample space consists of all $2^3 = 8$ sequences of D and N. $X$ counts the number of D's, making it a **binomial random variable** with $n = 3$ and $p = 1/4$, whose PMF is:

$$p_X(k) = \binom{n}{k} p^k (1-p)^{n-k}$$

**Step 2.** For $P(X = 2)$, apply the binomial PMF directly: $\binom{3}{2} = 3$ counts the number of ways to choose which 2 of 3 chips are defective, $(1/4)^2$ is the probability of those 2 being defective, and $(3/4)^1$ is the probability the remaining chip is non-defective.

**Step 3.** $Y$ takes only two values (0 or 1), so by definition it is a **Bernoulli random variable**. Its parameter equals $P(Y = 1)$, which we obtain by summing the relevant binomial probabilities for $X = 2$ and $X = 3$.
