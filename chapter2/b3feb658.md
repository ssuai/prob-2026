# Problem 2 (Intermediate) — 결합 PMF, 주변화, 조건부 PMF, 독립성

## 문제

어느 편의점에서 하루 동안 **도시락 주문 수** $X$와 **음료 주문 수** $Y$를 조사하였다. 두 확률변수의 **결합 PMF** $p_{X,Y}(x, y)$는 다음 표와 같다.

|  | $Y=0$ | $Y=1$ | $Y=2$ |
|:---:|:---:|:---:|:---:|
| $X=1$ | 0.10 | 0.10 | 0.00 |
| $X=2$ | 0.20 | 0.20 | 0.10 |
| $X=3$ | 0.10 | 0.10 | 0.10 |

**(a)** $X$와 $Y$의 주변 PMF $p_X(x)$와 $p_Y(y)$를 각각 구하시오.

**(b)** $X = 2$가 주어졌을 때의 조건부 PMF $p_{Y|X}(y \mid 2)$를 구하시오.

**(c)** $X$와 $Y$는 독립인가? 수치를 이용하여 판별하시오.

**(d)** $Z = X + 2Y$로 정의된 확률변수의 기댓값 $E[Z]$를 구하시오.
