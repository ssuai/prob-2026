Problem 2 — Intermediate

Topic: Function of a random variable, discrete random variable

A coffee shop records the number of pastries sold during the first hour after opening.
 Let X be the number of pastries sold, with probability distribution:

   P(X=0)=0.10,P(X=1)=0.20,P(X=2)=0.35,P(X=3)=0.25,P(X=4)=0.10

The manager defines a bonus score Y for the morning shift by the rule Y=2X+1

1. Find the probability mass function of Y
2. Find P(Y≥5)
3. Compute the expected value E(Y)
4. Compute the variance of Y
