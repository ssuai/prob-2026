# Problem 3 (Hard) — 연속 확률변수: PDF, 기댓값/분산, 선형변환, 표본평균

## 문제

품질 관리 부서에서 생산된 부품의 **직경** (단위: cm)을 측정한다. 직경 $X$는 구간 $[10, 16]$에서 **균일(uniform) 분포**를 따른다.

**(a)** $X$의 PDF $f_X(x)$를 쓰고, $E[X]$와 $\text{var}(X)$를 구하시오.

**(b)** 규격 기준값은 13 cm이다. 기준값으로부터의 오차를 나타내는 확률변수 $Y = X - 13$을 정의하자.

- $E[Y]$와 $\text{var}(Y)$를 구하시오.
- 확률 $P(-1.5 \leq Y \leq 1.5)$를 구하시오.

**(c)** 독립적으로 측정한 $n$개의 직경 $X_1, X_2, \ldots, X_n$ 으로 **표본평균**을 다음과 같이 정의한다.

$$S_n = \frac{1}{n} \sum_{i=1}^{n} X_i$$

- $E[S_n]$을 구하시오.
- $\text{var}(S_n)$을 구하시오. (독립 확률변수의 분산 합 성질을 이용하시오.)
- $\text{var}(S_n) \leq 0.05$가 되려면 $n$은 최소 얼마여야 하는가?

**(d)** 두 직경 $X_1$, $X_2$가 서로 독립이고 같은 균일분포를 따를 때, $W = X_1 + X_2$를 정의하자.

- $E[W]$와 $\text{var}(W)$를 구하시오.
- $W$의 범위는 어떻게 되는가? (가능한 최솟값과 최댓값)
