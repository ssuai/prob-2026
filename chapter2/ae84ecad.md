# Intermediate Problem

어떤 학생이 4문제로 구성된 시험을 본다. 각 문제를 맞힐 확률은 0.5이며,
서로 독립이다.  
확률변수 X를 맞힌 문제의 개수라고 하자.

또, 새로운 확률변수 Y = 3X + 2를 정의한다.

## 문제

1. X의 확률질량함수(PMF)를 구하시오.
2. P(X ≥ 2)를 구하시오.
3. E\[X]와 Var(X)를 구하시오.
4. E\[Y]와 Var(Y)를 구하시오.
5. 확률변수 Z = X²일 때, E\[Z]를 구하시오.
6. Var(X) = E\[X²] - (E\[X])² 공식을 이용하여 분산을 다시 구하시오.
