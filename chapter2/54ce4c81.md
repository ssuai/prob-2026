### Hard : Properties of Conditional Random Variables

Two random variables $X$ and $Y$ are defined as follows. $X$ takes values in $\{1, 2\}$ with $p_X(1) = 0.5$ and $p_X(2) = 0.5$. Given the value of $X$, the conditional PMF of $Y$ is:
- $p_{Y|X}(y|1)$: Uniformly distributed over $\{1, 2\}$ (i.e., $P(Y=1|X=1) = P(Y=2|X=1) = 0.5$)
- $p_{Y|X}(y|2)$: Uniformly distributed over $\{2, 3\}$ (i.e., $P(Y=2|X=2) = P(Y=3|X=2) = 0.5$)

1. Find the joint PMF $p_{X,Y}(x, y)$ for all possible pairs of $(x, y)$.
2. Find the marginal PMF $p_Y(y)$.
3. Given the condition $Y=2$, find the conditional PMF of $X$, $p_{X|Y}(x|2)$.
4. Calculate $E[XY]$ and check if the independence identity $E[XY] = E[X]E[Y]$ holds.
