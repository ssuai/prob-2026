Problem 3.1: The Bayesian Poisson Process
A server farm handles web traffic using one of two routing algorithms. Let L (Lambda) be a discrete random variable representing the active algorithm's parameter:

Prior Probability pi_1 = 1/3: Algorithm 1 is active, where X ~ Poisson(lambda_1 = 2).

Prior Probability pi_2 = 2/3: Algorithm 2 is active, where X ~ Poisson(lambda_2 = 5).

You observe the system for one millisecond and record X requests.

1.Marginalization: Formulate the marginal PMF of the observed requests, P_X(k).

2.Bayesian Updating: Given that you observe exactly X = 3 requests, derive the posterior probability PMF of the parameter L, specifically finding P(L = lambda_1 | X = 3) and P(L = lambda_2 | X = 3).

3.Sequential Prediction: After observing X = 3 in the first millisecond, you observe the system for a second millisecond, letting Y be the number of requests. Assuming X and Y are conditionally independent given L, derive the expected number of requests in the second millisecond, E[Y | X = 3].
