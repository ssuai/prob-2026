# Problem 1: IT Support Tickets

**Problem Statement:**
Let $X$ represent the number of daily support tickets assigned to an IT technician. The probability distribution for the discrete random variable $X$ is given in the table below:

| $x$ (Number of tickets) | $P(X=x)$ |
| :---: | :---: |
| 0 | 0.5 |
| 2 | 0.3 |
| 4 | 0.2 |

**Questions:**
1. Find the Expected Value, $E[X]$.
2. Calculate the Variance, $var[X]$.
