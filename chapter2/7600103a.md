# Intermediate Problem

A random experiment is conducted where a fair coin is tossed **three times**.

Define two random variables:
- \(X\): the number of heads obtained.
- \(Y\): the number of tails obtained in the first two tosses.

Answer the following:

1. List the sample space of this experiment.

2. Construct the **joint probability mass function** \(p_{X,Y}(x,y)\) in table form.

3. Find the marginal PMFs \(p_X(x)\) and \(p_Y(y)\).

4. Are \(X\) and \(Y\) independent? Justify your answer.
