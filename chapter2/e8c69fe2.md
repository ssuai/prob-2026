A control signal X in a digital system takes states in {−1, 0, 1} with the following probability 

mass function (PMF):

pX(−1) = 1/4, pX(0) = 1/2, pX(1) = 1/4

This signal passes through an amplifier, resulting in the final output Y=2X+3.



1. Find the expectation E\[X] and variance var(X) of the original signal X.
2. Using the properties of linearity, find the expectation E\[Y] and variance var(Y) of 

the final output.
