# 3. Hard — Two Random Variables, Joint PMF, Marginalization, and a Function of \((X,Y)\)

At a coffee shop, a customer orders one drink. The drink size is chosen first, and then the number of flavor syrup pumps is chosen according to the size.

Let \(X\) denote the number of espresso shots in the drink, and let \(Y\) denote the number of syrup pumps added.

The coffee shop uses the following model:

- With probability \(\tfrac{1}{2}\), the customer orders a **small** drink, which has \(X=1\) shot.
- With probability \(\tfrac{1}{3}\), the customer orders a **medium** drink, which has \(X=2\) shots.
- With probability \(\tfrac{1}{6}\), the customer orders a **large** drink, which has \(X=3\) shots.

The number of syrup pumps \(Y\) depends on the drink size:

- If \(X=1\), then
  \[
  P(Y=0\mid X=1)=\frac{3}{4}, \qquad P(Y=1\mid X=1)=\frac{1}{4}.
  \]

- If \(X=2\), then
  \[
  P(Y=0\mid X=2)=\frac{1}{2}, \qquad P(Y=1\mid X=2)=\frac{1}{3}, \qquad P(Y=2\mid X=2)=\frac{1}{6}.
  \]

- If \(X=3\), then
  \[
  P(Y=1\mid X=3)=\frac{2}{3}, \qquad P(Y=2\mid X=3)=\frac{1}{3}.
  \]

Now define a third random variable \(Z\) by

\[
Z = X(1+Y),
\]

which represents the “flavor-weighted espresso amount.”

Answer the following questions.

### (a)
Find the joint pmf

\[
p_{X,Y}(x,y)=P(X=x,\;Y=y).
\]

### (b)
Using marginalization, compute the marginal pmfs

\[
p_X(x) \qquad \text{and} \qquad p_Y(y).
\]

### (c)
Determine whether \(X\) and \(Y\) are independent. Justify your answer using the pmfs.

### (d)
Find the range of \(Z\), and then compute the pmf

\[
p_Z(z)=P(Z=z).
\]

Your solution should clearly show which pairs \((x,y)\) produce each value of \(z\).

### (e)
Compute the expectation \(E[Z]\).
