# 1. Easy — Random Variable, PMF, Normalization, Expectation, and Variance

A standard **double-six domino set** contains all unordered pairs \((a,b)\) with integers \(0 \le a \le b \le 6\). Thus, each tile is an unordered pair of numbers from 0 to 6, and all such tiles are equally likely to be selected.

One tile is chosen uniformly at random from the set.

Define the random variable \(X\) by

\[
X = |a-b|,
\]

the absolute difference between the two numbers on the selected domino tile \((a,b)\).

Answer the following questions.

### (a)
Identify the range of \(X\), that is, all possible values that \(X\) can take.

### (b)
For each possible value \(x\) in the range of \(X\), find the pmf

\[
p_X(x)=P(X=x)
\]

by counting the number of domino tiles for which \(X=x\).

### (c)
Verify the normalization axiom by showing that

\[
\sum_x p_X(x)=1.
\]

In your explanation, make clear why the events \(\{X=x\}\) form a partition of the sample space.

### (d)
Compute the expectation \(E[X]\) and the variance \(\mathrm{Var}(X)\).

You may use

\[
E[X]=\sum_x x\,p_X(x)
\]

and

\[
\mathrm{Var}(X)=\sum_x (x-E[X])^2 p_X(x),
\]

or equivalently,

\[
\mathrm{Var}(X)=E[X^2]-(E[X])^2.
\]
