## 2. [난이도: 중] 결합 PMF와 확률의 정규화
**주제: Joint PMFs of Multiple Random Variables**

### 문제
두 이산 확률 변수 $X$와 $Y$의 결합 PMF $p_{X,Y}(x,y)$가 다음과 같습니다.
$$p_{X,Y}(x,y) = c(x+y) \quad (\text{단, } x \in \{1, 2\}, y \in \{1, 2\})$$
이 모델이 유효한 확률 분포가 되기 위한 상수 $c$의 값을 구하세요.

### 풀이
1. **확률의 총합 성질**: 모든 가능한 $(x, y)$에 대해 결합 PMF의 합은 1이어야 합니다.
   $$\sum_{x=1}^{2} \sum_{y=1}^{2} c(x+y) = 1$$
2. **각 항 전개**:
   - $(x,y) = (1,1) \implies c(1+1) = 2c$
   - $(x,y) = (1,2) \implies c(1+2) = 3c$
   - $(x,y) = (2,1) \implies c(2+1) = 3c$
   - $(x,y) = (2,2) \implies c(2+2) = 4c$
3. **합산 및 방정식 풀이**:
   $$2c + 3c + 3c + 4c = 12c = 1$$
   $$c = \frac{1}{12}$$

**정답: $1/12$**
