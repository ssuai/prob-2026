# Hard Problem

A factory produces items using two independent machines, A and B.

- Machine A produces a defective item with probability \(0.2\).
- Machine B produces a defective item with probability \(0.3\).

Each machine produces one item independently.

Define the following random variables:
- \(X\): indicator of whether Machine A produces a defective item  
  (\(X=1\) if defective, \(0\) otherwise)
- \(Y\): indicator of whether Machine B produces a defective item  
  (\(Y=1\) if defective, \(0\) otherwise)

Answer the following:

1. Construct the joint PMF \(p_{X,Y}(x,y)\).

2. Let \(Z = X + Y\) be the total number of defective items.  
   Find the PMF of \(Z\).

3. Find the conditional probability \(P(X=1 \mid Z=1)\).

4. Compute the expectation \(E[Z]\) in two different ways:
   - (a) Using the PMF of \(Z\)
   - (b) Using properties of expectation and independence

5. Are \(X\) and \(Z\) independent? Justify your answer.
