# Problem 1 (Easy)

Assume that the number of data packets arriving at a network switch per second follows a Poisson distribution with an average rate of lambda = 4.

(a) Provide the Probability Mass Function (PMF), pX(k), for the number of packets arriving in exactly one second.

(b) Calculate the expected number of packets E[X] and the variance var(X) for this one-second interval.
