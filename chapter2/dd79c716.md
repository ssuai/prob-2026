2. Intermediate: The Linear TransformationLet $Y$ be a discrete random variable with the following probability distribution
y,−2,0,2,4
P(Y=y),0.1,0.4,0.3,0.2
A new random variable $Z$ is defined by the linear transformation $Z = 3Y - 5$.

Task A: Determine the expected value $E[Z]$ using the properties of expectation.
Task B: Calculate the variance $Var(Z)$.
Task C: Find the PMF of the transformation $W = Y^2$. Note how the mapping affects the probabilities of the resulting values.
