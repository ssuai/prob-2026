PMF and Expectation of a Discrete Random VariableProblem:
A student is working on a programming assignment.
The probability that the code passes the test on any given attempt is 0.6.
The student can submit the code at most 3 times. If the code passes, the student stops.
If it fails, the student modifies it and submits again, up to the 3rd attempt. (Assume each attempt is independent.)

1. Let X be the number of times the student submits the code. Find the probability mass function (PMF) p_X(x)
2. Calculate the expected number of submissions E[X]
