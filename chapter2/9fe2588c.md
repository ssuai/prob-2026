# Hard Problem

어떤 실험에서 두 개의 독립적인 동전을 동시에 던진다.  
각 동전은 앞면이 나올 확률이 각각 0.6, 0.5이다.

확률변수들을 다음과 같이 정의한다.

* X: 첫 번째 동전이 앞면이면 1, 아니면 0
* Y: 두 번째 동전이 앞면이면 1, 아니면 0
* Z = X + Y (앞면의 개수)
* W = XY (두 동전이 모두 앞면인지 여부)

## 문제

1. X, Y의 확률질량함수(PMF)를 각각 구하시오.
2. (X, Y)의 결합확률분포를 구하시오.
3. Z의 확률질량함수(PMF)를 구하시오.
4. P(Z = 1), P(Z = 2)를 구하시오.
5. E\[Z]와 Var(Z)를 구하시오.
6. E\[W]를 구하고 의미를 설명하시오.
7. E\[XY] = E\[X]E\[Y]가 성립함을 확인하시오.
