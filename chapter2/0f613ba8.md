\##3. 심화 난이도 (Hard)

\*\*주제: 결합 PMF를 이용한 주변 확률 계산 및 독립성 검정\*\*



\*\*\[문제]\*\*

두 이산 확률 변수 X와 Y의 결합 확률 질량 함수 p\_X,Y(x, y)가 아래 표와 같습니다. (X는 행, Y는 열)



X \\ Y | Y=1 | Y=2 | 주변 확률 p\_X(x)

\---|---|---|---

\*\*X=0\*\* | 0.2 | 0.1 | ( A )

\*\*X=1\*\* | 0.4 | 0.3 | ( B )

\*\*주변 확률 p\_Y(y)\*\* | ( C ) | ( D ) | 합계: 1.0



1\. 빈칸 ( A ), ( B ), ( C ), ( D )에 들어갈 확률값을 구하세요.

2\. X와 Y는 서로 독립입니까? 그 이유를 p\_X(0), p\_Y(1), p\_X,Y(0, 1)의 관계를 이용해 설명하세요.
