## [Problem 3] Inference of PMF and Function Transformation (Hard)

[cite_start]A discrete random variable $X$ is defined on the set $\{-2, -1, 0, 1, 2\}$[cite: 101, 296]. The probability mass function (PMF) of $X$ is symmetric, such that $p_X(x) = p_X(-x)$ for all $x$. [cite_start]It is given that $P(X=0) = 0.2$ and the second moment of $X$ is $E[X^2] = 2$[cite: 342, 344].

[cite_start]Now, let a new random variable $Y$ be defined by the function $Y = |X - 1|$[cite: 245, 297].

1. [cite_start]Determine the complete **Probability Mass Function (PMF)** of $X$[cite: 117, 321].
2. [cite_start]Derive the **PMF** of the transformed random variable $Y$[cite: 281, 291].
3. [cite_start]Calculate the **Mean** $E[Y]$ and the **Variance** $var(Y)$[cite: 335, 358].
