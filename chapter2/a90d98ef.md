# Probability Problem Set

## Problem 3 — Hard

The Poisson random variable has PMF given by:

$$p_X(k) = e^{-\lambda} \frac{\lambda^k}{k!}, \quad k = 0, 1, 2, \ldots$$

Recall the Taylor series expansion:

$$e^{\lambda} = \sum_{k=0}^{\infty} \frac{\lambda^k}{k!}$$

**(a)** Show that $p_X(k)$ is a valid PMF, i.e., verify that $\displaystyle\sum_{k=0}^{\infty} p_X(k) = 1$.

**(b)** Using the definition $\displaystyle E[X] = \sum_{k=0}^{\infty} k \cdot p_X(k)$, derive $E[X]$.
