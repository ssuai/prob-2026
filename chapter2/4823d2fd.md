# Problem 3: True Success Rate (Difficulty: Hard)

We want to find the true success rate of a new program. We call this unknown rate p. 
We run the program n times. The results are independent random variables: X1, X2, ..., Xn. 
(X = 1 for success, 0 for failure). All of them have the same expected value: E[X] = p.

You need to use the sample mean (Sn) to estimate p. 
Show the math steps to prove that the sample mean Sn is an unbiased estimator for p. 
(Hint: Write down the formula for Sn first, then find its expected value.)
