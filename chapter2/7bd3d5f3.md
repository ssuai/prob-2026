Consider n independent Bernoulli random variables X1, X2, ... ,Xn, each representing a response from a survey with a success probability p.

1. Define the sample mean as Sn = \frac{1}{n} \sum_{i=1}^n X_i. Show that E[Sn] = p, proving it is an unbiased estimator.
2. Derive the variance of the sample mean, var(Sn), using the property of the sum of independent random variables. Explain how the variance behaves as the sample size $n$ increases.
3. If n=100 and p=0.2, calculate the specific numerical value of var(Sn).
