# Intermediate Problem: Cloud Server Routing

### Situation
A cloud computing network routes incoming data packets to one of two servers, represented by the discrete random variable $Y \in \{1, 2\}$. The marginal Probability Mass Function (PMF) that a packet is routed to server $y$ is defined algebraically as:

$$p_Y(y) = \frac{3y + 2}{13} \quad (\text{for } y \in \{1, 2\})$$

Once a packet reaches a server, the processing time is evaluated as a random variable $X$, which can be either $1$ (Fast) or $2$ (Standard). The conditional PMF of the processing time $X$, given the server $Y$, is defined by the following equation:

$$p_{X|Y}(x|y) = \frac{xy + 1}{3y + 2} \quad (\text{for } x \in \{1, 2\} \text{ and } y \in \{1, 2\})$$

---

### Question
Using the multiplication rule and the Law of Total Probability, calculate the **marginal PMF of the processing time, $p_X(x)$**, for all possible values of $X$.
