Problem 3 — Hard

Topic: Multiple random variables, conditioning, independence, function of random variables

A small health clinic uses two rapid screening tests, Test A and Test B, to check whether a patient may need further examination.

For a randomly selected patient:

- The probability Test A is positive is P(A)=0.30
- The probability Test B is positive is P(B)=0.40
- The probability both tests are positive is P(A∩B)=0.18

Let: 
- 𝑋= 1 if Test A is positive, and  𝑋 = 0 otherwise
- 𝑌= 1 if Test B is positive, and 𝑌 = 0 otherwise

Define a new random variable:       W=X+Y

  So W represents the total number of positive test results for the patient.

1. Construct the joint probability table of X and 𝑌
2. Determine whetherX and Y are independent. Justify your answer using probabilities
3. Find the probability mass function of W
4. Find the conditional probability that Test A is positive given that exactly one test is positive
5. Find the conditional probability that both tests are positive given that at least one test is positive
