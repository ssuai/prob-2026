# Hard Problem — Multi-Stage Conditional System

## Scenario
Let Y be message size and Z be processing mode.

Let X be transmission time.


P(Y=1)=0.7, P(Y=2)=0.3

P(Z=A|Y=1)=0.6, P(Z=B|Y=1)=0.4  
P(Z=A|Y=2)=0.2, P(Z=B|Y=2)=0.8  

Conditional PMF of X:

(Y=1,Z=A): X=1(0.5), X=2(0.5)  
(Y=1,Z=B): X=2(1)  
(Y=2,Z=A): X=2(0.3), X=3(0.7)  
(Y=2,Z=B): X=3(0.6), X=4(0.4)

---

## Problem

1. Find P(Y,Z).
2. Find P(X).
3. Compute P(Y=1|X=2) and determine independence.
4. Define W = X − Y. Find the PMF of W.
