# Problem 2: Server Tasks (Difficulty: Intermediate)

Server A finishes X tasks, and Server B finishes Y tasks.
Here is their Joint PMF table:

| X \ Y | Y = 0 | Y = 1 |
| :--- | :--- | :--- |
| X = 0 | 0.2 | 0.1 |
| X = 1 | 0.3 | 0.4 |

**Part A**
Use marginalization to find the Marginal PMF of X and the Marginal PMF of Y.

**Part B**
Are X and Y independent random variables? Show your math to prove whether they are independent or not.
