### [Medium Level] Question 2: Server Failures & Malicious Traffic (20 Points)

**(2-a)** [Binomial Distribution] A distributed database is supported by a cluster of $5$ identical nodes. During a peak load test, each node has a probability $p = \frac{1}{4}$ of failing. The failure of any single node is strictly independent of the others. Calculate the probability that **exactly $2$** out of the $5$ nodes fail during the test.

**(2-b)** [Poisson Distribution] The server's firewall monitors malicious login attempts. These attempts are rare but occur at an average rate of $\lambda = 2$ attempts per minute. Assuming the number of attempts follows a Poisson distribution, calculate the probability that the firewall logs **exactly $3$** malicious attempts in a given minute. *(Note: Since you do not have a calculator, leave the mathematical constant $e$ in your final answer, e.g., in terms of $e^{-\lambda}$.)*

**(2-c)** [Unbiased Estimator] The engineering team is evaluating a new type of server. They want to estimate the true, unknown failure probability $p$ of this server type. They deploy $n$ independent servers and define a random variable $X_i$ for each server $i$ such that:
* $X_i = 1$ if the server fails.
* $X_i = 0$ if the server does not fail.

They use the sample mean $S_n$ as their estimator, defined as:
$$S_n = \frac{1}{n} \sum_{i=1}^{n} X_i$$
Mathematically prove that the expected value of the sample mean, $E[S_n]$, is exactly equal to the true probability $p$, thereby proving that $S_n$ is an unbiased estimator.
