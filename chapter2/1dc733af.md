# 2. Intermediate — Function of a Random Variable

Three distinct numbers are chosen uniformly at random from the set

\[
\{1,2,3,4,5,6\}.
\]

Since order does not matter, each 3-element subset is equally likely.

Let

\[
X = \max\{\text{chosen numbers}\} - \min\{\text{chosen numbers}\},
\]

the difference between the largest and smallest selected numbers.

Now define a new random variable \(Y\) by

\[
Y = |X-3|.
\]

Answer the following questions.

### (a)
Describe the sample space and determine its size. Then find the range of \(X\), and from that determine the range of \(Y\).

### (b)
Find the pmf \(p_X(x)=P(X=x)\) by counting how many outcomes produce each possible value of \(X\).

### (c)
Using the mapping \(Y=|X-3|\), find the pmf \(p_Y(y)=P(Y=y)\). Your solution should clearly show which values of \(x\) map to each value of \(y\), and should make explicit use of

\[
p_Y(y)=\sum_{x:\,g(x)=y} p_X(x).
\]

### (d)
Compute \(E[Y]\) and \(\mathrm{Var}(Y)\).
