# **Problem 3 (Hard): Joint PMFs, Conditioning \& Independence**

## **Theme: Independence Proof of Parallel Thread Executions**

* Two concurrent threads are executing in a highly nondeterministic runtime environment. Thread A's current execution state is modeled by random variable $X$, where 0 means "Idle" and 1 means "Executing". Thread B's state is modeled by random variable $Y$, also utilizing states 0 and 1. A memory analysis bounded-model-checking tool captures their Joint Probability Mass Function $p\_{X,Y}(x,y)$ as a matrix grid:
* $P(X=0, Y=0) = 0.25$
* $P(X=0, Y=1) = 0.25$
* $P(X=1, Y=0) = 0.25$
* $P(X=1, Y=1) = 0.25$
* Using principles derived from Probabilistic Concurrent Outcome Logic, mathematically prove whether the execution state of Thread A is strictly independent of the execution state of Thread B.

##### **Extra Credit Features**

* **Professor's Insight:** Proving strict statistical independence is absolutely critical in concurrent separation logic. If thread variables are independent, the thread operations do not suffer from latent race conditions, false sharing, or hidden resource locking over shared memory states. This joint PMF marginalization calculation fundamentally validates that the operating system's thread scheduler is treating the two concurrent threads without any entangled or blocking dependencies.
