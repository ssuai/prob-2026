# **Problem 1 (Easy): Discrete Random Variables \& PMF**

## **Theme:** API Error Rates \& Packet Success in Wi-Fi 7 Networks

* A high-density Wi-Fi 7 network router utilizes Multi-Link Operation (MLO) to transmit a continuous data stream to an edge computing server. To avoid network congestion and optimize the contention window, a background microservice groups the payload data into exactly 2 independent API packets per cycle. Historical traffic models indicate that due to dynamic channel adjustments, the absolute probability of any single packet successfully reaching the server without being dropped by collision is exactly $p = 0.5$.
* Let the random variable $X$ represent the total number of successfully transmitted packets in the group.
* Calculate the exact Probability Mass Function (PMF) for all possible values of $X$.

##### **Extra Credit Features**

* **Professor's Insight:**
* This introductory problem demonstrates the fundamental Binomial distribution within a highly relevant modern networking context. While Wi-Fi 7 dramatically improves throughput via 320 MHz channels, packet transmission at the physical layer remains a series of independent Bernoulli trials. Grasping this specific distribution is the mandatory first step toward understanding how transport protocols mathematically handle packet loss and trigger retransmission cascades.
