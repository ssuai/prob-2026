# Probability Mass Function (PMF) Practice Problem: Easy

## Problem: The Backpack
A returning student has a pencil case containing **4 blue pens** and **2 red pens**. If they randomly select exactly 1 pen for their first class, let $X = 1$ if the pen is red and $X = 0$ if it is blue.

* **Task:** Write the complete PMF of $X$.
