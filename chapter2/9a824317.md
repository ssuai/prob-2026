Hard Problem:
A hybrid signal processing unit receives two types of data from independent sensors.
Sensor A (Discrete): Produces a discrete signal X. The signal is equally likely to be any integer value in the set {1,2,3,4}.
Sensor B (Continuous): Produces a continuous noise interference Y. The interference level Y is uniformly distributed over the interval .
The unit processes these signals into a single output U defined by the non-linear relationship: 
U=4X+Y^2
 
Calculate the following:
The expected value of the total output, E[U].
The variance of the total output, var(U).
