Functions of Random Variables and Variance Problem:
A discrete random variable X has the following PMF:

p_X(x) =  1/5,   x in {-2, -1, 0, 1, 2}
              0,     otherwise

Let a new random variable be defined as Y = X^2 + 1
1. Find the PMF of Y, p_Y(y).
2. Calculate the expected value E[Y]
3. Calculate the variance of X, var(X)
