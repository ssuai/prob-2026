# Problem 2 — Functions of Random Variables & Conditioning

## Problem Statement

Let \( X \) be a discrete random variable taking values in:

\[
\{0, 1, 2, \dots\}
\]

with probability mass function:

\[
P(X = k) = (1 - p)^k p, \quad k = 0, 1, 2, \dots
\]

where \( 0 < p < 1 \).

---

## Definition of New Random Variable

Define a new random variable:

\[
Y = \mathbf{1}_{\{X \text{ is even}\}}
\]

i.e.,

\[
Y =
\begin{cases}
1, & \text{if } X \text{ is even} \\
0, & \text{otherwise}
\end{cases}
\]

---

## Questions

1. Compute:

\[
P(Y = 1)
\]

2. Compute:

\[
E[X \mid Y = 1]
\]
