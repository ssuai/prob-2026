**Problem Statement: Poisson Distribution and Linear Transformations**

Consider a Poisson random variable $X$ used to model the number of heads appearing in a sequence of $n$ independent coin tosses where $n$ is very large and the probability of a head $p$ is very small. The distribution is characterized by the parameter $\lambda = np$.

1. Given that the mean of $X$ is $\lambda = 4$, write the probability mass function $p_X(k)$.
2. Let a new random variable be defined by the linear function $Y = 3X + 5$. Using the properties of expectation for linear functions, calculate $E[Y]$.
3. Using the properties of variance for linear functions, calculate $var(Y)$.
4. Calculate the probability $P(X \le 1)$ using the Poisson PMF.
