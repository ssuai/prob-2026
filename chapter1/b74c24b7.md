# Intermediate Problem — Multipath Packet Transmission

## Scenario
A data packet is transmitted through two independent network paths.

- Path 1 succeeds with probability 0.9  
- Path 2 succeeds with probability 0.8  

The packet is considered successfully delivered if at least one path succeeds.

There are two transmission modes:
- High-priority mode (probability 0.6): Path 1 success probability becomes 0.95  
- Normal mode (probability 0.4): Probabilities remain unchanged  

## Problem

1. Using the total probability theorem, compute the overall probability that a packet is successfully delivered.  

2. Given that the packet was successfully delivered, compute the probability that it was sent in high-priority mode using Bayes’ rule.
