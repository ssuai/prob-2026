Problem 2 (Intermediate)

Topic: Total Probability Theorem and Bayes' Rule

A tech company uses three different cloud server clusters (Cluster A, Cluster B, and Cluster C) to process its user requests.

- Cluster A processes 40% of the total requests.
- Cluster B processes 35% of the total requests.
- Cluster C processes 25% of the total requests.

Based on historical data, the probability that a request experiences a processing timeout depends on the cluster:

- 2% for Cluster A
- 4% for Cluster B
- 6% for Cluster C

If a randomly selected user request resulted in a timeout, what is the probability that it was processed by Cluster B?
