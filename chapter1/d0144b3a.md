3. Hard
A factory uses three machines A_1, A_2, and A_3 to produce components. The prior probabilities of a component being produced by each machine are P(A_1)=0.5, P(A_2)=0.25, and P(A_3)=0.25. The probabilities of a defective component (Event B) given each machine are: P(B|A_1) = 0.02, P(B|A_2) = 0.05, P(B|A_3) = 0.10
(1) Find the total probability P(B) that a randomly chosen component is defective. 
(2) If a component is found to be defective, what is the posterior probability that it was produced by machine A_1?
