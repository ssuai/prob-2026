An urn contains exactly 9 identical-looking marbles: 5 Red marbles and 4 Blue marbles.

You draw 3 marbles uniformly at random without replacement.



Let us define the following events: 



* Event A: All 3 drawn marbles are of the same color.
* Event B: At least 2 drawn marbles are Red.
* Event C: All 3 drawn marbles are Red.



1. Using the principles of counting (combinations), derive the probability P(A) that 
all 3 drawn marbles share the same color.
2. Given that event B has occurred, find the conditional probability P(C∣B) that all 

3 drawn marbles are Red. Formulate your answer using the formal mathematical 

definition of conditional probability and combinations.
