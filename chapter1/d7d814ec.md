# Practice Problem 1: Basics of Conditional Probability

### [Problem]
An analysis of a university student's weekly schedule provided the following probabilities:
* Probability that the student goes to the library: $P(A) = 0.6$
* Probability that the student studies: $P(B) = 0.5$
* Probability that the student both goes to the library and studies: $P(A \cap B) = 0.4$

Find the **conditional probability $P(B|A)$** that the student actually studies ($B$), given that they went to the library ($A$).
