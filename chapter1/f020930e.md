 Event A is rolling an even number: {2, 4, 6}
 Event B is rolling a number strictly greater than 4: {5, 6}

Question: Using the Additivity Rule for non-disjoint events (P(A union B) = P(A) + P(B) - P(A intersection B)), what is the probability of rolling a number that is either even or greater than 4?
