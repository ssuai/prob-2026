# Problem 3 (Hard)

A university email system uses two spam filters.

- 40% of incoming emails pass through Filter A
- 60% of incoming emails pass through Filter B

Suppose:
- If an email is actually spam, Filter A correctly flags it with probability 0.92
- If an email is actually spam, Filter B correctly flags it with probability 0.85
- If an email is not spam, Filter A falsely flags it with probability 0.04
- If an email is not spam, Filter B falsely flags it with probability 0.06
- 8% of all incoming emails are actually spam

An incoming email is selected at random and is found to be flagged as spam.

1. What is the probability that a randomly selected email is flagged as spam?
2. Given that the email was flagged as spam, what is the probability that it was actually spam and processed by Filter A?

Assume each email is processed by exactly one of the two filters, according to the given proportions.
