# Problem 2 (Intermediate) — 전확률정리 + 베이즈 정리

## 문제

한 대학교의 프로그래밍 수업에서 학생들은 세 가지 프로그래밍 언어 중 하나를 선택하여 과제를 제출한다.

- 학생의 **40%** 는 **Python**을 선택한다.
- 학생의 **35%** 는 **C++**을 선택한다.
- 학생의 **25%** 는 **Java**를 선택한다.

각 언어별로 과제를 **기한 내에 제출**할 확률은 다음과 같다:

| 언어 | 기한 내 제출 확률 |
|------|-------------------|
| Python | 0.85 |
| C++ | 0.70 |
| Java | 0.75 |

**(a)** 임의로 선택한 학생이 과제를 **기한 내에 제출**할 확률을 구하시오.

**(b)** 한 학생이 과제를 **기한 내에 제출**했다고 알려졌다. 이 학생이 **Python**을 선택했을 확률을 구하시오.

**(c)** 한 학생이 과제를 **기한 내에 제출하지 못했다**고 알려졌다. 이 학생이 **C++**을 선택했을 확률을 구하시오.
