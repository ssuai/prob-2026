# H9. Conditional Independence vs. Marginal Independence

## Setup

A medical test is used to detect a disease. Let:

- $D$ = the patient has the disease
- $T_1$ = Test 1 returns positive
- $T_2$ = Test 2 returns positive

The following probabilities are known:

- $P(D) = 0.01$
- $P(T_1 \mid D) = 0.90$, $\quad P(T_1 \mid D^c) = 0.05$
- $P(T_2 \mid D) = 0.85$, $\quad P(T_2 \mid D^c) = 0.10$

**Assume** $T_1$ and $T_2$ are **conditionally independent** given $D$ (and given $D^c$):

$$P(T_1 \cap T_2 \mid D) = P(T_1 \mid D)\,P(T_2 \mid D)$$

$$P(T_1 \cap T_2 \mid D^c) = P(T_1 \mid D^c)\,P(T_2 \mid D^c)$$

---

## Questions

### Q1.
Find $P(T_1 \cap T_2)$ — the probability that **both** tests return positive.

### Q2.
Find $P(D \mid T_1 \cap T_2)$ — the probability the patient has the disease given **both** tests are positive.

### Q3.
Are $T_1$ and $T_2$ **marginally independent**? That is, does $P(T_1 \cap T_2) = P(T_1)\,P(T_2)$?

---

## Notes

- Conditional independence **does not** imply marginal independence.
- This is one of the most commonly misunderstood concepts in probability.
- Q3 is a conceptual trap — think carefully before answering.
