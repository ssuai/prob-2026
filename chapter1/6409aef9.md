# Practice Problem 2: Total Probability Theorem and Independence

### [Problem]
You randomly choose one of two servers (Server 1, Server 2) to process a task.
* The probability of choosing Server 1 is $0.6$, and the probability of choosing Server 2 is $0.4$.
* The probability of a task error occurring on Server 1 is $0.05$.
* The probability of a task error occurring on Server 2 is $0.10$.

1. Find the **total probability of an error occurring, $P(Error)$**, for a randomly performed task.
2. Prove whether the **event of choosing Server 1** and the **event of an error occurring** are **independent** using the formula $P(A \cap B) = P(A)P(B)$.
