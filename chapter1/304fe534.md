## 📌 Hard Problem

In the board game *The Great Dalmuti*, there are cards numbered from \(1\) to \(12\), and there are exactly \(k\) cards of number \(k\). Thus, the total number of cards is \(78\).

The deck is well shuffled, and a player randomly draws \(5\) cards.

Define the following events:

- \( A \): at least two of the selected cards have the same number  
- \( B \): the selected cards include the card numbered \(1\)

Answer the following:

1) Find the total number of possible outcomes \( |\Omega| \) using combinations.  
2) Find the number of outcomes in which event \( B \) occurs.  
3) Find the number of outcomes in which both \( A \) and \( B \) occur.  
4) Find the conditional probability \( P(A \mid B) \).
