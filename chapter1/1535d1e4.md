## Problem 2 (Intermediate)

A student creates a 5-character access code using the letters $A,B,C,D,E,F,G,H$, with no repeated letters.

The code is chosen uniformly at random from all such 5-letter sequences, subject to the rule that the first character cannot be $A$.

Let $E$ be the event that the code contains exactly two letters from the set $\{A,E\}$.

**Task:** Determine the probability of event $E$.
