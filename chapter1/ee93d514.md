# Hard Problem — Cloud Resource Allocation with Conditional Independence

## Scenario

A cloud system consists of two data centers: DC1 and DC2.

### DC1 state (N1)
- Good (probability 0.8): server success probability = 0.9  
- Bad (probability 0.2): server success probability = 0.5  

### DC2 state (N2)
- Good (probability 0.7): server success probability = 0.95  
- Bad (probability 0.3): server success probability = 0.4  

N1 and N2 are independent.

---

### Task Processing

Each job uses:
- 2 servers from DC1  
- 1 server from DC2  

A job succeeds if at least 2 out of 3 servers succeed.

Three jobs are processed, all sharing the same N1 and N2 environment.

---

## Problem

1. Compute the probability that a single job succeeds.  

2. Given that a job succeeded, compute the probability that DC1 was in the Good state:
   
   P(N1 = G | S)

3. Compute the probability that exactly 2 out of 3 jobs succeed.
