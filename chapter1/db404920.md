# Problem 3 (Hard)

어떤 공장에는 두 개의 기계 A와 B가 있다.

- 기계 A는 전체 생산량의 60%를 만들고, 불량률은 3%이다.
- 기계 B는 전체 생산량의 40%를 만들고, 불량률은 8%이다.

각 제품은 검사 시스템을 거친다.

- 제품이 불량이면, 검사에서 불량으로 판정될 확률은 0.9이다.
- 제품이 정상인데도 불량으로 잘못 판정될 확률은 0.1이다.

다음 사건을 정의하자:
- D: 제품이 불량이다
- T: 검사 결과가 불량(양성)이다

1. P(T)를 구하시오.
2. 검사 결과가 불량일 때, 실제로 불량일 확률 P(D | T)를 구하시오.
3. 검사 결과가 불량일 때, 해당 제품이 기계 B에서 나왔을 확률을 구하시오.
