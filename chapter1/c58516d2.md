### [Easy] 

A software quality assurance (QA) team analyzed a recent build and found that the probability of a **user interface (UI) bug** (Event $U$) is **0.15**, and the probability of a **database connectivity error** (Event $D$) is **0.20**. 

The team also discovered that the probability of both a UI bug and a database error occurring in the same test session ($U \cap D$) is **0.05**. 

Using the appropriate probability property derived from the axioms, calculate the probability that a randomly selected test session contains **at least one** of these two issues ($U \cup D$).
