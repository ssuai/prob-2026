# Problem 3 (Hard) — 독립성, 조건부독립, 베르누이 시행, 베이즈 정리

## 문제

통신 시스템에서 송신기(transmitter)가 **비트 0** 또는 **비트 1**을 전송한다. 각 비트가 전송될 사전 확률(prior probability)은 다음과 같다:

$$P(\text{비트 0 전송}) = 0.6, \quad P(\text{비트 1 전송}) = 0.4$$

채널에 잡음(noise)이 있어서, 수신기(receiver)가 받는 비트가 뒤집힐 수 있다. 각 비트가 **독립적으로** 오류가 발생하며, 비트 오류 확률(bit error probability)은 **p = 0.1**이다.

송신기가 **같은 비트를 3번 반복 전송**하고, 수신기는 3번의 수신 결과 중 **다수결(majority vote)**로 원래 비트를 판정한다.

**(a)** 비트 0이 전송되었을 때, 수신기가 올바르게 비트 0으로 판정할 확률을 구하시오.
(힌트: 3번의 수신 중 2번 이상 올바르게 수신되어야 한다. 베르누이 시행을 이용하시오.)

**(b)** 수신기가 **비트 1로 판정**했을 때, 실제로 **비트 1이 전송되었을** 확률을 구하시오.
(힌트: 베이즈 정리와 전확률정리를 이용하시오.)

**(c)** 두 개의 독립적인 채널(Channel A, Channel B)을 통해 같은 비트를 각각 1번씩 전송한다고 하자. Channel A의 비트 오류 확률은 **0.1**, Channel B의 비트 오류 확률은 **0.2**이다. 두 채널의 오류는 서로 독립이다.

   - (c-1) 두 채널 모두 올바르게 수신할 확률을 구하시오.
   - (c-2) 적어도 하나의 채널이 오류를 발생시킬 확률을 구하시오.
   - (c-3) 비트 0이 전송되었다는 조건 하에서, Channel A와 Channel B의 수신 결과는 조건부 독립인지 논증하시오.
