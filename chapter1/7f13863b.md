# Problem 3 (Hard)

You are reviewing 10 new algorithm codes to be deployed to a server. Among these, **3 codes contain critical bugs, and 7 codes are perfectly bug-free**. Due to time constraints, you decide to randomly **select 3 codes simultaneously** to test.

Define the following two events:
* Event C: **At least 1** bug is found among the 3 selected codes.
* Event D: **Exactly 1** bug is found among the 3 selected codes.

During the test, you find that a bug occurred in the selected batch (meaning at least one bug exists). Given this information, exactly calculate the **conditional probability P(D|C)** that exactly 1 of the selected codes has a bug, using the combination notation (nCk).
