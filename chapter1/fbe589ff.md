### Probability & Statistics | Section 1.1 ~ 1.6

**Problem**: A school conducts a flu test. 10% of all students actually have the flu. The test has the following accuracy:
- If a student **has** the flu, the test returns **positive** with probability **0.9**
- If a student does **not** have the flu, the test still returns **positive** (false alarm) with probability **0.2**

Given that a student tests **positive**, what is the probability that they **actually have the flu**?
