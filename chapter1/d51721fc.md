# Problem 3 (Hard)

A fair 4-sided dice is rolled three times.

A = { the first roll results in an even number },  
B = { the second roll results in a number greater than 2 },  
C = { the sum of the three rolls is even }.

1) Compute P(A), P(B), and P(C)

2) Compute P(A ∩ B)

3) Are A and B independent?

4) Are A and C independent?

5) Are A, B, and C mutually independent?
