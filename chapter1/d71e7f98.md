PART 1 — Binomial Probability

A student takes a quiz with 4 true/false questions. The student
guesses each answer independently, with probability p = 3/4 of
getting each question correct.

Let X = number of correct answers. The binomial probability formula
from the chapter states:

  P(X = k) = C(n, k) · p^k · (1-p)^(n-k)

where C(n, k) = n! / (k!(n-k)!).

1. Find the probability of getting exactly 3 questions correct,
   P(X = 3):
   (a) ________

2. Find the probability of getting all 4 questions correct,
   P(X = 4):
   (b) ________

3. Find the probability of getting at least 3 questions correct,
   P(X ≥ 3):
   (c) ________

4. Given that the student got at least 3 correct, find the
   probability that they got all 4 correct, P(X = 4 | X ≥ 3):
   (d) ________

PART 2 — Pairwise vs. Mutual Independence

Consider three events A, B, and C defined on the same sample space,
with the following probabilities:

  P(A) = 1/2,  P(B) = 1/2,  P(C) = 1/2
  P(A ∩ B) = 1/4,  P(A ∩ C) = 1/4,  P(B ∩ C) = 1/4
  P(A ∩ B ∩ C) = 1/4

5. Check whether A and B are pairwise independent by verifying
   P(A ∩ B) = P(A)·P(B):
   (e) ________

6. Check whether A, B, and C are mutually independent by verifying
   P(A ∩ B ∩ C) = P(A)·P(B)·P(C):
   (f) ________

7. Based on (e) and (f), what can you conclude about the
   relationship between pairwise independence and mutual
   independence?
   (g) ________

---

Answer:
   (a) P(X = 3) = 108/256 = 27/64
   (b) P(X = 4) = 81/256
   (c) P(X ≥ 3) = 189/256
   (d) P(X = 4 | X ≥ 3) = 81/189 = 3/7
   (e) P(A ∩ B) = 1/4 = P(A)·P(B) = 1/2 × 1/2  → Pairwise independent ✓
   (f) P(A ∩ B ∩ C) = 1/4 ≠ P(A)·P(B)·P(C) = 1/8  → NOT mutually independent ✗
   (g) Pairwise independence does NOT imply mutual independence.
