# Problem 1 (Easy)

Assume a computer program runs an experiment that fairly generates a random integer from 1 to 6. The sample space is Ω = {1, 2, 3, 4, 5, 6}.

Define the following two events:
* Event A: The generated number is even.
* Event B: The generated number is a multiple of 3.

**(a)** Are events A and B **mutually exclusive**? Explain mathematically by checking if their intersection exists.

**(b)** Are events A and B **independent**? Explain mathematically by comparing the product of their individual probabilities with the probability of their intersection.
