3. A network security system uses a sensor to detect whether an attack is occurring (A: attack, Aᶜ: no attack).
The prior probability that an attack actually exists is 2%.

The sensor operates as follows:
• Probability of raising an alarm when an attack is present: 0.92
• Probability of raising a false alarm when no attack is present: 0.08

a. Compute the overall probability P(Alarm).
b. Compute the probability that an attack is actually occurring given that an alarm has been raised, P(A | Alarm).
