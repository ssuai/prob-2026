## Intermediate (Independence)

**Problem Title**

* Numerical Independence Proof in Packet Header Validation

**Problem Statement**

* Consider a 3-bit network packet header ($b\_1 b\_2 b\_3$) where each bit is generated independently and is equally likely to be 0 or 1. This creates a sample space of 8 equally likely outcomes ($\\Omega = {0, 1}^3$). Define the following two events:
* **Event A**: The first bit is 1 ($b\_1 = 1$).
* **Event B**: The total number of 1s in the 3-bit header is even.
* Using the formal definition of independence ($P(A \\cap B) = P(A) \\cdot P(B)$), prove whether Event A and Event B are statistically independent.
* Furthermore, based on these results, demonstrate that probabilistic independence is distinct from the mere existence of an intersection (overlap) in a Venn diagram.

**Mandatory Solution Protocol**

* To receive full credit, **students must draw a Sample Space Venn Diagram**. \[cite\_start]List all 8 outcomes ${000, 001, 010, 011, 100, 101, 110, 111}$ inside a rectangle.
* **Students are required to** circle the outcomes for Event A (starts with 1) and Event B (sum of bits is even).
* Observe that the intersection contains exactly 2 outcomes ($101$ and $110$), which is $2/8 = 1/4$ of the total. This manual verification is mandatory to confirm the numerical results.

**Professor's Insight**

* The goal here is for students to realize that independence is a mathematical property, not just a physical one. Even though both events are defined on the same 3-bit string, knowing the first bit provides no information about the parity of the entire string. This is a crucial concept in error detection and parity bit generation in communications.
* Furthermore, the primary objective is to clarify that probabilistic independence is a distinct concept from the mere existence of an intersection (overlap) in a Venn diagram.
