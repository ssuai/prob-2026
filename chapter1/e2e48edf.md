# Topic: Multiplication Rule and Conditional Probability

**Problem:**
A standard deck of 52 playing cards contains 13 hearts. You draw two cards from the deck successively without replacement. Let A1 be the event that the first card is a heart, and A2 be the event that the second card is also a heart. 

1. Find the probability P(A1). 
2. Find the conditional probability P(A2 | A1). 
3. Using the multiplication rule, calculate the probability that both cards drawn are hearts (P(A1∩A2)).
