# Exercise 3: Bernoulli Trials and Reliability

An Internet Service Provider (ISP) has installed $c = 2$ modems to serve a group of $n = 4$ customers. Based on usage statistics, each customer requires a connection at any given time with a probability $p = 0.1$, independent of the other customers. An "outage" occurs if the number of customers requesting a connection exceeds the number of available modems.

**Questions:**
1. Model this scenario as a sequence of Bernoulli trials and clearly state the parameters ($n, p$).
2. Calculate the probability that exactly 3 customers require a connection simultaneously.
3. Calculate the exact probability that an outage occurs (i.e., the probability that more than 2 customers need a connection).
