### [Hard Level] Question 3: Distributed Cloud Server Failure (25 Points)

A micro-data center contains exactly 5 servers in total. This set of servers is partitioned based on their operating systems: 3 are Linux servers and 2 are Windows servers. 
To deploy a new distributed application, a sub-network of exactly **2 servers** is randomly selected from the 5 available servers. 

Let the sample space of all possible sub-networks be partitioned into three disjoint events:
* Event $A_1$: The selected sub-network consists of 2 Linux servers.
* Event $A_2$: The selected sub-network consists of exactly 1 Linux server and 1 Windows server.
* Event $A_3$: The selected sub-network consists of 2 Windows servers.

A sudden targeted network attack occurs (e.g., a ransomware infection). Based on threat intelligence:
* Linux servers are immune to this attack (Probability of survival = $1$).
* Windows servers are vulnerable and have a $\frac{1}{2}$ probability of surviving the attack.
* The survival of any server is strictly **independent** of the survival of any other server.

The deployed application is considered to have "Survived" (Let this be Event $S$) **ONLY IF both** servers in the selected sub-network survive the attack. 

**(3-a)** [Partitions & Combinations] Calculate $P(A_2)$, the probability that the randomly selected sub-network consists of exactly 1 Linux server and 1 Windows server.

**(3-b)** [Conditional Prob & Independence] Given that Event $A_2$ has occurred, calculate the conditional probability that the application survives, $P(S|A_2)$.

**(3-c)** [Total Probability Theorem] Regardless of which 2 servers were initially chosen to form the sub-network, calculate the total probability $P(S)$ that the application survives. (Hint: You will need to determine the probabilities of all partitions and their respective conditional survival probabilities).

**(3-d)** [Bayes' Rule] After the attack, the monitoring system indicates that the application has successfully **Survived** (Event $S$). Given this new evidence, what is the posterior probability that the chosen sub-network was originally composed of exactly 1 Linux and 1 Windows server ($P(A_2|S)$)?
