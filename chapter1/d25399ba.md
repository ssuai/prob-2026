Topic: Independence vs. Mutually Exclusive Events in System Redundancy
CS Context: Fault-tolerant System Design

In a distributed database system, you have two independent nodes: Node A 
and Node B. Let E_A be the event that Node A fails, and E_B be the event 
that Node B fails. 

Given: P(E_A) = 0.05 and P(E_B) = 0.10.

1. Since the events are independent, the probability that both nodes 
   fail simultaneously, P(E_A AND E_B), is:
   (a) ________

2. If a student mistakenly assumes that E_A and E_B are mutually 
   exclusive (disjoint), they would calculate the probability that 
   *at least one* node fails, P(E_A OR E_B), to be:
   (b) ________

3. Using the correct axioms of probability for independent events, 
   the actual probability that at least one node fails is:
   (c) ________
