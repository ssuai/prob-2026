# Easy Problem — IoT Sensor Network

## Scenario
In an IoT monitoring system, three sensors (A, B, C) detect whether an event occurs.

- Sensor A detects correctly with probability 0.8  
- Sensor B detects correctly with probability 0.7  
- Sensor C detects correctly with probability 0.6  

Assume all sensors operate independently.

## Problem

1. What is the probability that at least one sensor detects the event correctly?  

2. What is the probability that exactly one sensor detects the event correctly?
